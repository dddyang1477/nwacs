## 核心能力

### 1. 题材创新

#### 题材融合
```
融合公式：
题材A + 题材B = 新题材

示例：
- 玄幻 + 赛博 = 《没钱修什么仙?》
- 修仙 + 家族 = 《玄鉴仙族》
- 末日 + 搞笑 = 《第一序列》
- 都市 + 悬疑 = 《隐秘工作者》
```

#### 元素重组
```
重组公式：
核心元素A + 新元素B + 差异化设定C = 创新题材

示例：
- 废柴逆袭 + 系统流 + 搞笑 = 创新题材
- 重生复仇 + 美食 + 种田 = 创新题材
```

### 2. 世界观创新

#### 世界观模板
```
基础模板：
- 世界类型：现实/异世界/未来
- 核心规则：物理法则/魔法法则/混合
- 独特设定：独特的力量体系/社会结构/文化

创新方向：
- 物理规则改变（重力反转/时间异常）
- 社会结构创新（母系社会/阶级颠倒）
- 力量体系独特（情绪为力/梦境为实）
```

#### 设定生成
```
设定类型：
- 修炼体系设定
- 社会制度设定
- 文化习俗设定
- 地理环境设定
- 历史背景设定
```

### 3. 人物创新

#### 角色模板
```
基础设定：
- 身份：普通人/特殊身份/隐藏身份
- 性格：1个核心+2个次要

创新方向：
- 性格反转（冷酷角色有柔软面）
- 身份反差（底层身份隐藏大佬）
```

```
创新点：
```

## 六、经典开篇模板库（v8.0新增）

### 1. 废柴逆袭型
**模板**：
```
曾经的天才，如今的废物。
[曾经辉煌] → [遭遇变故] → [最低谷] → [奇遇/觉醒]
```

**示例开头**：
```
青云宗大殿，十年一度的资质测试正在进行。

"林逸，金属性，资质...下下品！"

话音落下，全场哗然。曾经的天才，十二岁就突破练气期的天才，如今却连最差的资质都评不上。

修炼之道，资质为根。没有资质，就等于与修仙无缘。

角落里，未婚妻苏雪儿的眼神中闪过一丝失望。

"林逸哥哥..."她轻声呢喃，"我..."

就在这时，林逸的脑海中，一道苍老的声音响起：

"小子，想变强吗？"
```

### 2. 总裁豪门型
**模板**：
```
[误解/契约] → [相处] → [感情] → [危机] → [大团圆]
```

**示例开头**：
```
"这是离婚协议，签了吧。"

顾寒深将一份文件扔在苏念面前，眼神淡漠得像是看一个陌生人。

三年的隐婚，他连正眼都没看过她一次。

苏念低头看着协议上的字：三套房产，一千万现金，公司股份5%。

呵，三年的青春，就值这些？

"我不要这些。"她拿起笔，签下自己的名字，"我只要——自由。"

顾寒深皱眉，心中莫名一紧。

"苏念——"

话没说完，她已经转身离开。

他不知道的是，她肚子里，还揣着他的孩子。
```

### 3. 穿越重生型
**模板**：
```
[死亡/意外] → [穿越/重生] → [适应] → [逆袭] → [登顶]
```

**示例开头**：
```
"滴滴滴——"

心电监护仪的声音在耳边响起。

林尘缓缓睁开眼睛，入目的是一片刺眼的白。

"我...还活着？"

记忆中，他是为了掩护队友撤退，被敌人的炮弹淹没。

可现在，他躺在一张陌生的床上，身边是穿着古装的人。

"少爷，您终于醒了！"一个丫鬟模样的女孩哭着说，"您昏迷了三天三夜，老爷都急坏了！"

林尘愣住了。

他低头看着自己的手——年轻、白皙、纤细。

这不是他的手。

这是...另一个人的人生。

而他，要在这个世界，重新活一次。
```

### 4. 悬疑推理型
**模板**：
```
[诡异事件] → [调查] → [发现线索] → [接近真相] → [反转]
```

### 5. 仙侠修真型
**模板**：
```
[平凡出身] → [获得传承] → [修炼成长] → [遭遇危机] → [飞升/称王]
```

## 八、剧情创意生成器（v8.0新增）

### 核心创意公式
```
黄金三秒开篇公式：
冲突前置 + 人设反差 + 金句钩子 = 吸引人的开头
```

### 创意生成方法
```
创意三问：
1. 如果...会怎样？（反常规假设）
2. 换个角度呢？（视角转换）
3. 反过来呢？（反转思维）
```

### 创意生成示例
```
创意1：如果修仙需要氪金？
→ 金融修仙流：灵石=货币，修炼=投资，门派=公司

创意2：如果时间倒流，但只有主角记得？
→ 重生复仇流：带着未来记忆，改变过去遗憾

创意3：如果系统变成反派？
→ 反套路系统流：主角对抗系统，掌握自己命运
```

### 卡文诊断工具
```
卡文类型诊断：
- 情节卡文：不知道接下来写什么
- 人物卡文：不知道人物会怎么做
- 情感卡文：写不出想要的情感
```

### 瓶颈突破方案
```
方案1：换个视角写

→ 先写下一个爽点，然后回来补中间

方案3：提问法

方案4：灵感激发
```
