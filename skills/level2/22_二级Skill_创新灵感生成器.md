## 核心能力

### 1. 题材创新

#### 题材融合
```
融合公式：
题材A + 题材B = 新题材

示例：
- 玄幻 + 赛博 = 《没钱修什么仙?》
- 修仙 + 家族 = 《玄鉴仙族》
- 末日 + 搞笑 = 《第一序列》
- 都市 + 悬疑 = 《隐秘工作者》
```

#### 元素重组
```
重组公式：
核心元素A + 新元素B + 差异化设定C = 创新题材

示例：
- 废柴逆袭 + 系统流 + 搞笑 = 创新题材
- 重生复仇 + 美食 + 种田 = 创新题材
```

### 2. 世界观创新

#### 世界观模板
```
基础模板：
- 世界类型：现实/异世界/未来
- 核心规则：物理法则/魔法法则/混合
- 独特设定：独特的力量体系/社会结构/文化

创新方向：
- 物理规则改变（重力反转/时间异常）
- 社会结构创新（母系社会/阶级颠倒）
- 力量体系独特（情绪为力/梦境为实）
```

#### 设定生成
```
设定类型：
- 修炼体系设定
- 社会制度设定
- 文化习俗设定
- 地理环境设定
- 历史背景设定
```

### 3. 人物创新

#### 角色模板
```
基础设定：
- 身份：普通人/特殊身份/隐藏身份
- 性格：1个核心+2个次要
- 目标：短期+中期+长期
- 缺陷：1个致命+2个次要

创新方向：
- 性格反转（冷酷角色有柔软面）
- 身份反差（底层身份隐藏大佬）
- 能力异化（超能力+致命代价）
```

#### 关系网络
```
关系类型：
- 血缘关系：家族/门派/师徒
- 利益关系：合作/竞争/背叛
- 情感关系：爱情/友情/仇恨
- 命运关系：宿命/因果/羁绊

创新点：
- 多重身份关系
- 身份反转关系
- 隐藏关系揭示
```

### 4. 情节创新

#### 情节模板
```
经典模式：
- 升级流：打怪升级换地图
- 逆袭流：废柴崛起打脸
- 探秘流：解开谜题揭示真相
- 情感流：情感纠葛发展

创新方向：
- 多线叙事交织
- 非线性时间线
- 视角创新（多人视角/器灵视角）
- 反套路设计
```

#### 冲突设计
```
冲突类型：
- 外部冲突：敌人/环境/社会
- 内部冲突：自我/欲望/信念
- 人际冲突：家庭/朋友/爱人

创新点：
- 冲突反转（敌人变朋友）
- 冲突升级（个人→家族→天下）
- 冲突解决（非常规方式）
```

### 5. 灵感触发器

#### 随机组合
```
灵感公式：
随机元素A + 随机元素B + 随机元素C = 创新灵感

触发器：
- 随机题材触发
- 随机设定触发
- 随机人物触发
- 随机情节触发
```

#### 联想扩展
```
联想路径：
核心词 → 相关词 → 延伸词 → 创新点

示例：
核心词：剑
相关词：剑客、剑法、剑意
延伸词：剑冢、剑魂、剑灵
创新点：剑修合一、人剑共生
```
