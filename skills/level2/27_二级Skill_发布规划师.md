## 核心能力

### 1. 平台选择策略

#### 平台特性分析
| 平台 | 特点 | 适合题材 | 收益模式 |
|------|------|---------|---------|
| 起点中文网 | 老牌、大神多、付费 | 玄幻、都市 | VIP订阅+全勤 |
| 番茄小说网 | 免费流量大、年轻 | 爽文、系统 | 广告分成+全勤 |
| 晋江文学城 | 女频强、付费意愿高 | 言情、耽美 | VIP订阅 |
| 纵横中文网 | 买断制、保底 | 各类题材 | 买断+订阅 |
| 七猫小说网 | 免费、西红柿 | 赘婿、总裁 | 广告分成 |

#### 平台选择矩阵
```
选择因素：
- 题材匹配度
- 收益潜力
- 门槛难度
- 全勤奖励
- 成长空间

推荐策略：
- 新人首发的首选：番茄小说网（流量大、门槛低）
- 女频首选：晋江文学城（女频强、付费高）
- 精品路线：起点中文网（大神多、订阅高）
```

### 2. 更新策略

#### 更新节奏
| 类型 | 更新频率 | 字数 | 适用场景 |
|------|---------|------|---------|
| 日更 | 每天 | 3000-5000字 | 冲榜期 |
| 双更 | 每天2更 | 6000-10000字 | 上架初期 |
| 稳定更 | 每2天1更 | 3000字 | 稳定期 |
| 周更 | 每周5更 | 15000字 | 兼职写作 |

#### 更新时机
```
最佳更新时间：
- 早7-8点：通勤阅读
- 午12-13点：午休阅读
- 晚20-22点：黄金时段

推荐时段：
- 起点：晚20-22点
- 番茄：早7-8点 + 晚20点
- 晋江：晚21-23点
```

### 3. 发布计划

#### 开篇策略
```
开篇法则：
- 前3章必须抓住读者
- 黄金3章原则
- 冲突前置
- 期待感建立

开篇优化：
- 前3000字：核心冲突+金手指
- 前10000字：主要人物登场+世界观
- 前30000字：第一个小高潮
```

#### 冲榜计划
```
冲榜时间表：
- 新书期（入库前7天）：日更6000字+
- 强推期（上架前）：日更10000字
- 上架期（第一个月）：稳定日更
- 稳定期（后续）：保持更新

冲榜技巧：
- 保持更新节奏
- 引导读者投票
- 利用评论区互动
```

### 4. 数据优化

#### 关键指标
| 指标 | 说明 | 优化方向 |
|------|------|---------|
| 点击率 | 封面点击/展示 | 优化书名+封面 |
| 追读率 | 继续阅读比例 | 优化内容质量 |
| 收藏率 | 收藏/点击 | 优化开篇 |
| 付费率 | 付费/免费 | 优化VIP章节 |
| 追更率 | 追更/总阅读 | 优化更新节奏 |

#### 优化策略
```
数据问题诊断：
- 点击低：书名、封面、简介问题
- 追读低：前10章质量问题
- 收藏低：前30章吸引力问题
- 付费低：VIP章节节奏问题

优化方案：
- A/B测试书名
- 优化封面设计
- 调整简介内容
- 改进更新节奏
```

### 5. 长期规划

#### 完本策略
```
完本时机：
- 质量下滑时及时完本
- 剧情完整时优雅完本
- 不要强行续写

完本收益：
- 完本推：平台奖励
- 完本感言：读者告别
- 完本活动：最后一波打赏
```

#### 作品线规划
```
作品规划：
- 第一本：试水+积累经验
- 第二本：复制成功+优化
- 第三本：突破创新
- 长期：形成风格+粉丝积累

系列化策略：
- 同一世界观多本书
- 主角相关人物独立故事
- 类型化写作（专业户）
```
