## 一、基础修辞手法

### 1. 比喻（明喻）

**定义**：用某一事物与另一事物的相似点来描写。

**格式**：像、如、似、仿佛、如同、像...一样

**类型**：
- **明喻**：A像B（直接说明）
  - 例：她的笑容像春天的阳光，温暖而明亮。
- **暗喻**：A是B（暗示）
  - 例：她的笑容是冬日里的一把火。
- **借喻**：直接用B代替A（不出现本体）
  - 例：她的笑容是黑暗中的一点星光。

**应用场景**：
- 开篇描写环境
- 人物外貌描写
- 情感表达
- 抽象概念具象化

**高阶训练**：
摒弃“脸红得像苹果”此类烂俗比喻。练习“陌生化三问”：本体最特异之处是什么？→什么事物有类似特质？→合成新比喻。
- 例：她的眼泪像老水龙头关不住的滴漏，静默地，一滴，一滴，腐蚀着室内的死寂。

### 2. 拟人

**定义**：把事物当作人来写，赋予人的情感、动作。

**例句**：
- 风轻轻地吹过，像母亲的手抚摸着大地。
- 夜色像一个巨大的幕布，缓缓降下。
- 时间像一个吝啬的老人，从不肯多给一秒。

**应用技巧**：
- 适用于自然环境描写
- 增强画面感
- 抒发情感

### 3. 排比

**定义**：三个或以上结构相同、内容相关的句子。

**例句**：
- 他跑得很快，像一阵风，像一支箭，像一道闪电。
- 我看到了，看到了高山，看到了大海，看到了星空。

**作用**：
- 增强气势
- 强调情感
- 节奏感强

### 4. 夸张

**定义**：故意夸大或缩小事物特征。

**类型**：
- **扩大夸张**：飞流直下三千尺
- **缩小夸张**：他的脑袋小得像一颗豌豆

**例句**：
- 教室里安静得连针掉在地上的声音都能听见。
- 他的眼睛瞪得像铜铃一样大。

### 5. 对偶

**定义**：结构相同、字数相等、意义相关的句子。

**例句**：
- 山不在高，有仙则名；水不在深，有龙则灵。
- 春风又绿江南岸，明月何时照我还。

**作用**：
- 形式整齐
- 音韵和谐
- 便于记忆

### 6. 反复

**定义**：同一词语或句子重复出现。

**例句**：
- 等待着，等待着，等待着那个人的到来。
- 长江，长江，你是我的母亲河。

**作用**：
- 强调情感
- 增强节奏
- 深化主题

## 二、高阶写作手法

### 1. 意识流

**定义**：模仿人脑思维活动，打破时空顺序。

**技巧**：
- 自由联想
- 内心独白
- 蒙太奇手法
- 时空交错

**例句**：
```
她的思绪飘回了十年前的那个夏天。那时候，他还年轻，她还美丽，一切都还是原来的模样。可是现在...时间都去哪儿了？
```

### 2. 伏笔与照应

**定义**：前文埋下线索，后文给出呼应。

**伏笔技巧**：
- 人物细节
- 环境描写
- 物品出现
- 言语暗示

**例句**：
```
开头：他摸了摸口袋里那枚旧钥匙，心中有种说不出的感觉。
结尾：那枚钥匙打开了尘封十年的往事。
```

### 3. 悬念设置

**类型**：
- **疑问式**：他为什么要这样做？
- **预测式**：接下来会发生什么？
- **冲突式**：两难的选择

**技巧**：
- 开头抛出问题
- 答案延迟揭晓
- 层层递进
- 高潮揭秘

### 4. 衬托与对比

**定义**：用其他事物来突出主要事物。

**正衬**：用相似的事物衬托
```
阳光照耀下，她的笑容更加灿烂。
```

**反衬**：用相反的事物衬托
```
他的成功，是在无数失败之后取得的。
```

### 5. 讽刺与幽默

**讽刺技巧**：
- 反语
- 夸张
- 比喻
- 对比

**幽默技巧**：
- 夸张变形
- 意外反转
- 自嘲
- 双关

## 四、叙事视角

- **第一人称**（“我”）：沉浸感强，适合情感描写，但受限于“我”的所知。
- **第三人称限制**（跟拍某角色）：兼顾代入与客观，是当下网文主流。
- **第三人称全知**（上帝视角）：信息量最大，但易失悬念。
- **第二人称**（“你”）：极强指令感或介入感，常用于实验性文本或互动小说。
- **多视角切换**：展现不同人物内心，但切换信号必须明确。
- **器灵视角**：创新视角，赋予器物以观察和叙述能力。

**视角选择技巧**：
- 视角即偏见：利用视角限制制造信息差。
- 视角转换信号：用明确的分节符，新视角第一句就确立“观察者身份”。
- 距离感调控：内心独白拉近读者，外部动作推远读者。

## 五、节奏控制

- **快慢结合**：紧张场景快速推进，抒情场景放慢节奏。
- **信息密度**：控制每段信息量，保持读者兴趣。
- **情绪曲线**：引入“情绪曲线”概念，替代单纯的快慢节奏。
- **信息释放曲线**：控制读者认知节奏，避免信息过载。

**快节奏例句**：
```
他冲出门，奔向医院。一路上，他的心狂跳不已。
```

**慢节奏例句**：
```
夕阳的余晖洒在街道上，将一切都染成了金色。他慢慢地走着，每一步都重若千钧。
```

## 六、黄金三秒开篇引擎

### 开篇手法

**黄金三秒法则**：
- 前三秒必须抓住读者
- 制造悬念或冲突
- 营造氛围

**开篇类型**：
- **悬念开场**：“那天之后，一切都变了...”
- **场景开场**：“深夜，寂静的小镇上传来一声尖叫...”
- **对话开场**：“”站住！“他大喊道。”
- **内心开场**：“我不明白，为什么会变成这样...”

### 开篇生成提示词模板
```

2. 语言风格匹配[风格定位]

```

## 七、去AI化写作方法

### 核心概念
“去AI化”不是反对技术，而是**剔除刻板的机器感，重现属于人类作家的“不完美之美”**。AI文本常表现为：平滑但无锋芒、逻辑但无意料、通顺但无腔调。去AI化即找回文字的**肌理、冲动、偏执和意外**。

### 实用技巧
- **刻意留“瑕疵”**：适当使用非流畅断句，模仿真实思维的中断与跳跃。
- **注入感官偏差**：AI常忽略细节的感受偏移。人类描述下雨不会只写“雨很大”，而会写“雨打在遮雨棚上，声音像有人往油锅里泼水”。
- **偏爱具象，避开抽象总结**：AI爱升华：“这让他明白了生命的可贵。”人类作家应写：“他把被打翻的蚂蚁窝重新堆好，看着蚁群慌乱一阵后恢复秩序，心里头什么大道理也没想，就是爽。”
- **制造“无效”细节**：增加与主线无关但刻画人物的闲笔。

### 去AI化检查清单
| 检查项 | AI表现 | 人类表现 |
|--------|--------|----------|
| 情感表达 | 直接使用情绪词汇（悲伤、愤怒） | 侧面描写（生理反应、环境投射） |
| 细节描写 | 平滑流畅，无意外 | 刻意留瑕疵，有真实感 |
| 语言风格 | 通顺但无腔调 | 有个人口头禅和偏好 |
| 节奏控制 | 紧张-舒缓-紧张平滑波形 | 有突兀的情绪碎片 |

## 八、画面感构建技巧

### 镜头语言法
- **推拉镜头**：从宏大环境（全景）聚焦到具体微小物体（特写）
  - 普通：战场很惨烈，到处都是尸体
  - 进阶：硝烟遮蔽了天空（全景），一只断箭插在焦土中，箭羽还在微微颤动（特写）
- **光影切割**：利用光与影的对比来突出主体
  - 技巧：不要写“天黑了”，要写“月光像冰冷的刀锋，切开了黑暗的帷幔，正好照在他苍白的侧脸上”

### 五感全息法
视觉只占感官的一部分，加入听觉、嗅觉、触觉、味觉，画面瞬间立体。
- 技巧：在描写场景时，强制加入至少一种非视觉元素
  - 听觉：死寂中，只有烛火爆裂的“噼啪”声
  - 嗅觉：空气中弥漫着一股铁锈般的血腥味，混杂着潮湿的泥土气息
  - 触觉：剑柄冰冷刺骨，握在手里像握住了一块万年玄冰

### 动词名词化
- 弱：他很生气地关上了门
- 强：他踹上门，门框上的灰尘簌簌落下

## 九、情感共鸣技巧

### 移情与物化
- 技巧：将情绪投射到环境或物体上
- 直白：他非常痛苦，看着她的遗物流泪
- 共鸣：那件未织完的毛衣还搭在椅背上，针脚有些乱，仿佛她只是去厨房热了一杯牛奶，随时都会回来坐下继续织

### 乐景写哀（反差法）
- 在最悲伤的时候写快乐，在最绝望的时候写希望，杀伤力加倍
- 技巧：不要写乌云密布，要写“今天的阳光真好，街角的卖花女笑得真甜，可惜这朵花，我再也闻不到了”

### 具体的“痛点”
- 宏大的苦难（如“天下大乱”）很难让人共情，具体的、微小的失去才最痛人
- 技巧：聚焦于一个具体的承诺或习惯的破碎
- 不是：战友死了
- 而是：再也没有人会在熄灯后，偷偷塞给他半块压缩饼干了

## 十、风格定位与统一

### 核心概念
**风格**是作品的声音指纹，它由**词汇选择、句式结构、节奏韵律、修辞偏好、情感温度**五要素共同构成。风格定位解决“这篇文章听起来像谁在说话”，而风格统一确保“这个人从头到尾说话一致”。

### 实用技巧
- **定调三步法**：
  ① 确定叙述者身份（古代书生？现代宅男？冷艳杀手？）
  ② 列出匹配的常用词汇库（如古文风多用“须臾、罢了、不意”，都市风多用“搞定、离谱、内卷”）
  ③ 规定禁用词汇（如古风文中切忌出现“绝对”“超级”“OK”）。
- **句式硬控**：若定位为简约硬朗风，刻意多用短句；若定位为华丽铺陈风，长短句交错如交响乐。
- **情感温度标尺**：设定叙述语气的冷热度。全文保持此距离的连贯性。

### 风格混搭的黄金配比
主风格占80%，调剂风格占20%。如《琅琊榜》整体深沉凝重，但飞流与梅长苏的互动带来偶尔的稚趣。

### 风格演变与角色弧光绑定
主角的叙述视角语言随其成长而变化。如主角初入江湖时心直口快，文风跳脱；历经沧桑后，同一视角的文风渐趋含蓄内敛。

## 十一、修辞手法综合运用

### 示例：完整段落

```
夕阳像一团燃烧的火焰，缓缓沉入地平线。

她站在窗前，看着那片绚烂的晚霞，心中泛起层层涟漪。回忆像潮水一般涌来，冲击着她脆弱的心堤。

十年前的那个夏天，也是这样的晚霞。那时候，他还是个意气风发的少年，而她还是个不谙世事的少女。他们在夕阳下许下誓言，要一起走完这一生。

可是，誓言终究抵不过时光的侵蚀。

如今，夕阳依旧绚烂，而他，却已不在身边。

“你还好吗？”她在心里轻声问道。

只有风，回应着她的沉默。
```

**运用手法**：
1. 比喻：夕阳像火焰
2. 拟人：晚霞心中泛起涟漪
3. 排比：回忆、潮水、心堤
4. 反复：夕阳、誓言
5. 借景抒情：夕阳、晚霞
6. 对比：十年前vs现在
7. 问句：制造悬念

### 修辞的隐蔽叙事
用修辞暗示情节而不用直接陈述。如不说“王朝衰败”，而写：“宫墙的朱漆剥落如岁月之痂，匾额上的金字也瘦成了枯骨。”

### 反修辞
在需要极强爆发力的段落，突然全部撤回修辞，纯用白描。如生死关头：“他站起来。子弹穿过胸口。他低头看了一眼。倒下。”前文的华丽修辞在此刻的朴素下，反衬出山崩般的冲击力。

## 十二、经典作品写作技巧学习

### 《剑来》风格学习
- **陈平安的说话方式**：朴实真诚，言出必行，有底线有原则
- **三观较正的语句**：“善有善报，恶有恶报，不是不报，时候未到。”
- **哲理对话风格**：引经据典但不晦涩，言简意赅，意味深长
- **细腻心理描写**：从细微处入手，展现内心矛盾和挣扎
- **行文节奏**：快慢有度，张弛结合，留白得当

## 十三、实用写作模板

### 画面感增强提示词
```
请重写以下段落，要求：
1. 运用电影镜头感，从远景缓慢推进到特写
2. 增加环境描写，重点描写光影变化和声音细节
3. 减少形容词，多用精准的动词来体现动作的力度
4. 风格参考：[赛博朋克/古风武侠/克苏鲁恐怖]
```

### 情绪共鸣增强提示词
```
请修改这段心理描写，要求：
1. 禁止直接使用情绪词汇（如悲伤、愤怒），改用侧面描写（如生理反应、环境投射）
2. 运用物化手法，通过描写一个具体的旧物来寄托情感
3. 营造此时无声胜有声的压抑氛围，节奏要慢
4. 目标是让读者感到无力感/遗憾/绝望
```

### 开篇生成提示词模板
```
请为[小说类型]生成一个[开篇类型]的开头，要求：

2. 语言风格匹配[风格定位]
3. 字数控制在[100-200字]
```

### 爽点密度控制模板
```
请设计[小说名称]的爽点节奏：

```
