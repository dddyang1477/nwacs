## 核心能力

### 1. 五感描写增强系统

#### 视觉描写
- **基础版→增强版**
  - "天空很蓝" → "澄澈的碧空如一块刚打磨过的蓝宝石，几缕云丝像被随手扯碎的棉絮，慵懒地漂浮着"
  - "他很英俊" → "他的鼻梁高挺如山峰，剑眉斜飞入鬓，那双桃花眼微微弯起时，仿佛有星光从里面溢出来"

- **动态描写技巧**
  - 光影变化：阳光穿过树叶的光斑移动
  - 物体运动：衣袂飘飘、发丝飞舞
  - 颜色层次：从深到浅的渐变过渡

#### 听觉描写
- **声音层次构建**
  - 远景声音：虫鸣、鸟叫、风声
  - 中景声音：脚步声、说话声、水流声
  - 近景声音：呼吸声、心跳声、衣物摩擦声

- **拟声词运用**
  - 清脆：叮咚、噼啪、咔嚓
  - 沉闷：轰隆、咚咚、咕嘟
  - 柔和：沙沙、潺潺、嘤嘤

#### 嗅觉描写
- **气味类型**
  - 自然气息：花香、草香、泥土气息
  - 生活气息：饭菜香、茶香、墨香
  - 特殊气息：血腥味、硝烟味、腐臭味

- **气味传递**
  - 随风飘来
  - 逐渐弥漫
  - 突然袭来

#### 味觉描写
- **味道层次**
  - 基础味：酸、甜、苦、辣、咸
  - 复合味：酸甜、麻辣、苦涩
  - 口感：脆、软、糯、滑

- **味觉联想**
  - "像含了一块冰"
  - "像喝了蜜一样甜"
  - "像吃了黄连一样苦"

#### 触觉描写
- **触感类型**
  - 温度：冰凉、滚烫、温暖、清凉
  - 质感：光滑、粗糙、柔软、坚硬
  - 力度：轻柔、沉重、有力、无力

- **触觉细节**
  - 皮肤接触的感觉
  - 衣物摩擦的感觉
  - 物体握持的感觉

### 2. 场景描写增强

#### 环境描写分层
- **远景**：天空、山脉、建筑轮廓
- **中景**：树木、道路、人群
- **近景**：人物、物品、细节

#### 天气描写模板
- **晴天**：
  - "阳光像碎金一样洒在地面，连风里都带着温暖的气息"
- **雨天**：
  - "豆大的雨点噼里啪啦砸下来，地面很快积起了水洼，倒映着灰蒙蒙的天空"
- **雪天**：
  - "鹅毛大雪纷纷扬扬飘落，整个世界都变成了银白色，静得能听到雪落下的声音"

#### 战斗场景描写
- **战斗前**：紧张气氛、环境铺垫
- **战斗中**：动作、声音、感官刺激
- **战斗后**：战场状况、人物状态

### 3. 人物描写增强

#### 外貌描写
- **整体印象**：身材、气质
- **面部细节**：五官、表情、眼神
- **衣着打扮**：服饰、配饰、整体风格

#### 动作描写
- **连贯动作**：拆解成多个小动作
- **细节动作**：手指的动作、脚尖的动作
- **情绪动作**：紧张时的小动作、开心时的动作

#### 心理描写
- **直接心理**：内心独白
- **间接心理**：通过动作、表情、环境暗示
- **感官心理**：通过五感描写体现心理状态

### 4. 描写增强工具

#### 快速增强模式
- **输入**：简单句子
- **输出**：五感增强后的段落

**示例**：
- 输入："他很渴"
- 输出："他的喉咙干得像冒了烟，嘴唇干裂得起了皮，每一次吞咽都牵扯着疼痛，他渴望着能有一口水，哪怕只是一滴也好。"

#### 选择性增强
- **指定增强**：只增强视觉/听觉/嗅觉等某一感
- **强度选择**：轻度增强/中度增强/深度增强
- **风格选择**：古风/现代/玄幻等

### 5. 描写质量检查

#### 检查维度
- **五感平衡**：是否过度依赖某一感
- **细节密度**：是否过于啰嗦或过于简略
- **服务剧情**：描写是否推动剧情或塑造人物
- **风格统一**：描写风格是否与整体文风一致

#### 优化建议
- 指出可以增强的地方
- 提供具体的改写建议
- 推荐更精准的用词

## 使用示例

### 示例1：场景描写增强
**输入**：
```
原文：房间很暗，气氛很紧张
要求：五感全开，深度增强
```

**输出**：
```
房间里没有开灯，只有窗外透进来的微弱月光，在地板上投下长长的影子。空气中弥漫着一股陈旧的灰尘味，还混合着淡淡的血腥味。寂静得能听到自己的心跳声，扑通、扑通，每一声都像是在敲打着什么。空气仿佛凝固了，让人喘不过气来，皮肤上起了一层细密的鸡皮疙瘩。
```

### 示例2：人物描写增强
**输入**：
```
原文：她很漂亮，很伤心
要求：古风风格，中度增强
```

**输出**：
```
她一袭素白长裙，站在那里如月下梨花般清雅。眉眼如画，只是那双眸子此刻蒙着一层水雾，泪珠儿在眼眶里打转，终于忍不住滚落下来，在白皙的脸颊上留下两道泪痕。她紧紧咬着下唇，仿佛在极力压抑着什么，纤弱的肩膀微微颤抖着，让人看了心生怜惜。
```
