## 2. 全球各国地理地貌

### 2.1 美国

| 地理特征 | 描述 |
|----------|------|
| **地形地貌** | 本土位于温带地区，耕地面积占全球十分之一，东西两侧被太平洋和大西洋环抱，北方与加拿大接壤，南方与墨西哥相邻 |
| **战略价值** | 两次世界大战都未波及本土，全球五分之一的海运贸易需经过美国港口，五大湖储藏着丰富淡水，内河运输成本只有卡车的三分之一 |
| **主要地形区** | 东部平原、西部落基山脉、中部大平原 |

### 2.2 中国

| 地理特征 | 描述 |
|----------|------|
| **地形地貌** | 青藏高原成为亚洲的水塔，西部高山挡住陆上进攻路线，东部海岸线长达1.8万公里，宁波舟山港是全球最大港口群之一 |
| **战略价值** | 长江黄河两条大河养活全国六成人口，气候从冷到热全都有，海陆兼顾的位置，"一带一路"陆路和海路运量又创新高 |
| **主要地形区** | 青藏高原、黄土高原、云贵高原、华北平原、长江中下游平原 |

### 2.3 俄罗斯

| 地理特征 | 描述 |
|----------|------|
| **地形地貌** | 横跨11个时区，从飞地加里宁格勒到远东勘察加，西伯利亚地区气候寒冷，但蕴藏着海量未开发的资源 |
| **战略价值** | 战略纵深极大，世界面积最大的国家，西部欧洲部分人口集中，东部开发难度大 |
| **主要地形区** | 东欧平原、西西伯利亚平原、中西伯利亚高原、东西伯利亚山地 |

### 2.4 印度

| **主要地形区** | 恒河平原、印度河平原、德干高原 |

### 2.5 日本

## 6. 军事地理与战术应用

### 6.1 地形与战术

| 地形类型 | 战术应用 | 代表地区 |
|----------|----------|
| **平原地形** | 装甲突击、坦克集群作战、大纵深作战，利于机械化兵团运动 |
| **山地地形** | 游击战、阻击战、伏击战，利于地形防御，机械化机动困难 |
| **沙漠地形** | 装甲机动、空降作战，装甲作战受沙漠风暴作战，后勤补给要求高 |
| **雨林地形** | 特种作战、游击战、渗透作战，隐蔽性强，装甲受限 |
| **城市地形** | 城市作战、巷战、CQB战术，逐屋争夺 |
| **岛屿地形** | 两栖作战、登陆战、反登陆作战，需要海空协同 |
| **极地地形** | 特种作战、生存作战，严寒挑战大 |

### 6.2 海峡与水道

| 水道名称 | 位置 | 战略价值 |
|----------|------|----------|
| **马六甲海峡** | 东南亚 | 全球四分之一的贸易和一半的原油运输经过这里 |
| **霍尔木兹海峡** | 波斯湾 | 全球四成原油运往亚洲经过这里 |
| **博斯普鲁斯海峡** | 土耳其 | 黑海舰队必须经过这里才能出海 |
| **英吉利海峡** | 英法之间 | 历史上重要的海战发生地点 |
| **台湾海峡** | 中国东南 | 中国近海防御与区域拒止的重要节点 |
