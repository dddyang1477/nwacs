# 📚 NWACS 知识库共享中心 v2.0

---

## 🎯 核心理念

**所有知识库对所有Skill共享！**
- 写都市小说可以用风水/军事/玄幻知识
- 写军事小说可以用总裁/风水/格斗知识
- 写玄幻小说可以用命理/军事/都市知识
- 写总裁文可以用全球地理/人文知识

## 📖 跨题材创作示例

### 示例1：都市+玄学+总裁

**题材**：都市玄学总裁文
**可用知识库**：
- 霸道总裁文与都市情感文知识库 → 男主设定、甜宠套路
- 风水玄学小说知识库 → 女主身份（风水师）、玄学技能
- 小说人物起名知识库 → 顾北辰、苏晚棠
- 军事小说知识库 → 男主特种兵背景

### 示例2：玄幻+军事+总裁

**题材**：玄幻军事文
**可用知识库**：
- 风水玄学小说知识库 → 修炼体系（五行相生相克）
- 军事小说知识库 → 战斗场面、武器装备
- 霸道总裁文与都市情感文知识库 → 女主身份设定
- 国外经典故事知识库 → 世界观结构参考

### 示例3：都市+总裁+军事

**题材**：都市婚恋+商战
**可用知识库**：
- 霸道总裁文与都市情感文知识库 → 核心套路
- 军事小说知识库 → 男主军人身份、特种背景
- 小说人物起名知识库 → 高级人名
- 风水玄学小说知识库 → 合婚、八字契合

### 示例4：西方奇幻+军事

**题材**：西幻战争文
**可用知识库**：
- 国外经典故事知识库 → 希腊神话、凯尔特神话
- 军事小说知识库 → 冷兵器战术、战略战术
- 风水玄学小说知识库 → 八卦阵法（魔法阵）
- 小说人物起名知识库 → 西方风格人名

## 🚀 使用指南

### 创作前准备
1. 确定小说题材类型
2. 打开相关核心知识库文件
3. 根据需要调用其他知识库内容
4. 整合设定后开始创作

### 创作中调用
- 遇到人名问题 → 查看起名知识库
- 遇到战斗场面 → 查看军事知识库
- 遇到玄学设定 → 查看风水知识库
- 遇到情感套路 → 查看总裁文知识库
- 遇到世界观 → 查看国外经典知识库
