## 一、经典开篇模板

### 1. 废柴逆袭型

**模板**:
```
曾经的天才，如今的废物。
[曾经辉煌] → [遭遇变故] → [最低谷] → [奇遇/觉醒]
```

**示例开头**:
```
青云宗大殿，十年一度的资质测试正在进行。

"林逸，金属性，资质...下下品！"

话音落下，全场哗然。曾经的天才，十二岁就突破练气期的天才，如今却连最差的资质都评不上。

修炼之道，资质为根。没有资质，就等于与修仙无缘。

角落里，未婚妻苏雪儿的眼神中闪过一丝失望。

"林逸哥哥..."她轻声呢喃，"我..."

就在这时，林逸的脑海中，一道苍老的声音响起：

"小子，想变强吗？"
```

### 2. 总裁豪门型

**模板**:
```
[误解/契约] → [相处] → [感情] → [危机] → [大团圆]
```

**示例开头**:
```
"这是离婚协议，签了吧。"

顾寒深将一份文件扔在苏念面前，眼神淡漠得像是看一个陌生人。

三年的隐婚，他连正眼都没看过她一次。

苏念低头看着协议上的字：三套房产，一千万现金，公司股份5%。

呵，三年的青春，就值这些？

"我不要这些。"她拿起笔，签下自己的名字，"我只要——自由。"

顾寒深皱眉，心中莫名一紧。

"苏念——"

话没说完，她已经转身离开。

他不知道的是，她肚子里，还揣着他的孩子。
```

### 3. 穿越重生型

**模板**:
```
[死亡/意外] → [穿越/重生] → [适应] → [逆袭] → [登顶]
```

**示例开头**:
```
"滴滴滴——"

心电监护仪的声音在耳边响起。

林尘缓缓睁开眼睛，入目的是一片刺眼的白。

"我...还活着？"

记忆中，他是为了掩护队友撤退，被敌人的炮弹淹没。

可现在，他躺在一张陌生的床上，身边是穿着古装的人。

"少爷，您终于醒了！"一个丫鬟模样的女孩哭着说，"您昏迷了三天三夜，老爷都急坏了！"

林尘愣住了。

他低头看着自己的手——年轻、白皙、纤细。

这不是他的手。

这是...另一个人的人生。

而他，要在这个世界，重新活一次。
```

### 4. 悬疑推理型

**模板**:
```
[命案/异常] → [调查] → [线索] → [反转] → [真相]
```

**示例开头**:
```
"死者，男，三十五岁，某上市公司董事长。"

"死亡时间：昨晚十点到十二点之间。"

"死因：心脏被锐器刺穿，当场死亡。"

我蹲在尸体旁边，目光扫过整个房间。

门窗紧锁，没有撬动痕迹。

密室杀人。

"监控呢？"我站起身。

"坏了，"警官苦笑，"正好在死者死亡时间段。"

我点点头，目光落在死者的右手上。

他的右手，紧紧攥着一样东西。

那是一张扑克牌。

黑桃A。

"有意思。"我嘴角微微上扬。

凶手故意留下了线索。

他在挑衅。
```

## 六、写作瓶颈突破

### 1. 卡文时的处理

| 问题 | 解决方法 | 示例 |
|------|----------|------|
| 不知道写什么 | 翻灵感库 | 查看人设、剧情、金句 |
| 情节太顺 | 加入意外 | 突然打断、突发状况 |
| 节奏太慢 | 增加冲突 | 制造误会、加入敌人 |
| 节奏太快 | 增加细节 | 心理描写、环境描写 |
| 人物扁平 | 增加缺点 | 完美人设加弱点 |

### 2. 灵感激发方法

1. **随机组合**: 随机选择2-3个元素强制组合
   - 例：霸道总裁 + 修仙 + 星际 + 猫咪

2. **反差设定**: 加入完全相反的元素
   - 例：最强大的神，却是社恐

3. **经典颠覆**: 反转固有设定
   - 例：灰姑娘是反派，白雪公主是心机女

4. **职业混搭**: 不同职业组合
   - 例：杀手+医生、律师+和尚

### 3. 突破瓶颈的技巧

- 换个场景写：先写后面的情节
- 写日记法：代入角色写日记
- 聊天法：角色之间聊天推进剧情
- 清单法：列出一堆情节一个个试
- 休息法：放下笔，出去走走

## 七、灵感练习

### 练习1：随机组合

**随机元素**:
- 主角身份：_____
- 金手指：_____
- 敌人：_____
- 感情线：_____
- 世界观：_____

**强制组合**: 尝试把完全不搭的元素组合在一起

### 练习2：反转练习

**选择一个经典设定，然后反转**:
- 好人 → 隐藏很深的坏人
- 坏人 → 被误解的好人
- 废柴 → 故意藏拙的天才
- 天才 → 只会理论的战五渣

### 练习3：扩写练习

**选择一个金句，扩写成一段情节**:
- "这一拳，我等了十年。"
  → 扩写成：为什么等？打的是谁？结果如何？
