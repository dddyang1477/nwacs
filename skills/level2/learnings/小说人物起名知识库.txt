## 1. 起名核心原则

### 四大原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **好读好记** | 别用生僻字、难读音 | ❌ 翀、彧、龑 ✅ 琛、烬、屿 |
| **符合人设** | 名字=气质，听起来就有感觉 | 温柔→温阮 强势→厉爵风 |
| **避免撞姓** | 别满篇都是顾、陆、沈 | 适当使用常见姓 |
| **时代统一** | 古代/现代/民国风格不能乱 | 古代不用"梓涵" |

## 7. 起名技巧

### 7.1 姓 + 气质字公式

**男姓公式**：
```
姓氏 + 清冷字：知、清、屿、晏、辞
姓氏 + 温柔字：温、阮、景、然、宁
姓氏 + 强势字：烬、骁、烈、凛
姓氏 + 霸气字：爵、珩、砚、驰
```

**女姓公式**：
```
姓氏 + 温柔字：柔、婉、卿、阮、晚
姓氏 + 清冷字：清、浅、知、舒
姓氏 + 甜美字：栀、糯、糖、软、柠
姓氏 + 诗意字：棠、禾、烟、禾
```

### 7.2 八大起名方法

| 方法 | 说明 | 示例 |
|------|------|------|
| **寓意象征法** | 名字暗示性格/命运 | 黛玉→柔弱 |
| **音韵声调法** | 读音营造感觉 | 萧峰→大气 |
| **时代地域法** | 符合时代背景 | 民国→方鸿渐 |
| **谐音双关法** | 谐音暗示命运 | 甄士隐→真事隐 |
| **职业特征法** | 暗示职业身份 | 叶修→散人 |
| **对比反讽法** | 名字与性格相反 | 虎妞→彪悍 |
| **自然元素法** | 自然意象入名 | 月、霜、雪 |
| **诗词典故法** | 出自诗词经典 | 出自诗经、楚辞 |

### 7.3 古风/仙侠起名

**仙气型**：
- 云清晏、灵汐、月辞、星眠
- 凌玄天、墨临渊、夜寒枭

**帝王型**：

## 9. 禁忌与注意事项

### ❌ 避免使用

| 类型 | 示例 | 原因 |
|------|------|------|
| **生僻字** | 翀、彧、龑、顼 | 读者不会读 |
| **太长绕口** | 夣蒳醴繺、嘂嘂 | 记不住 |
| **玛丽苏** | 上官殇寒、勗攜鹽 | 太尴尬 |
| **接地气过度** | 张伟、王强、李狗蛋 | 没有代入感 |
| **同音不同字** | 张章争、周舟州 | 容易混淆 |

### ✅ 推荐做法

| 类型 | 说明 |
|------|------|
| **2-3字最佳** | 最好记的名字长度 |
| **声调搭配** | 避免全平或全仄 |
| **男女有别** | 男名用珩、砚、烬；女名用绾、栀、糯 |
| **避免撞名** | 不要和名著/热剧主角撞名 |
| **可读可写** | 常见字，易读易写 |
