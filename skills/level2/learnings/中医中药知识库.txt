## 3. 鬼门十三针与祝由术

### 3.1 鬼门十三针

**历史渊源**：
- 最早记载于唐代孙思邈《千金要方》
- 治疗"百邪所病"、癫狂、癔症
- 十三鬼穴沿用至今

**十三鬼穴**：
| 序号 | 穴位 | 代号 | 定位 |
|------|------|------|------|
| 1 | 人中 | 鬼宫 | 鼻唇沟正中 |
| 2 | 少商 | 鬼信 | 拇指末节 |
| 3 | 隐白 | 鬼垒 | 足大趾内侧 |
| 4 | 大陵 | 鬼心 | 腕横纹中央 |
| 5 | 申脉 | 鬼路 | 外踝下方 |
| 6 | 风府 | 鬼枕 | 后发际正中 |
| 7 | 颊车 | 鬼床 | 下颌角前上方 |
| 8 | 承浆 | 鬼市 | 颏唇沟中央 |
| 9 | 劳宫 | 鬼窟 | 掌心 |
| 10 | 上星 | 鬼堂 | 前发际正中 |
| 11 | 会阴 | 鬼藏 | 阴囊根部 |
| 12 | 曲池 | 鬼臣 | 肘横纹端 |
| 13 | 舌下中缝 | 鬼封 | 舌系带中央 |

**操作要点**：
```
顺序：从鬼宫到鬼封
方向：男左女右
时机：单阳日、阳时针右转
口诀：刺入十三穴尽之时，医师即当口问病人
```

### 3.2 祝由术

**历史**：
- 古代中医十三科之一
- 通过符咒、祝祷治病
- 基于"信则灵"的心理暗示

**经典十三科**：
| 科名 | 治疗范围 |
|------|----------|
| 祝由 | 精神类疾病 |
| 禁科 | 禁制疾病 |
| 禁咒 | 咒语法术 |

### 3.3 小说中的中医秘术

**《九流术》设定**：
- 主角传承祝由十三科
- 鬼门十三针治疗疯癫
- 驱蛇咒、腐肉咒等符咒

**《大医永言》设定**：
- 润脂秘方
- 凤凰展翅
- 青龙摆尾
- 霹雳火龙弹

| 草药 | 识别特征 | 功效 | 毒性 |

| 党参 | 藤蔓、铃铛花、肉质根 | 补气健脾 | 无 |
| 桔梗 | 紫色五角星花、根可入药 | 宣肺利咽 | 无 |
| 柴胡 | 细叶、伞状黄花 | 解表退热 | 无 |
| 何首乌 | 心形叶、块根有云锦花纹 | 补肝肾 | 无 |
| 甘草 | 叶圆、根茎甜 | 补脾益气 | 无 |
| 丹参 | 紫色花、根茎紫红 | 活血祛瘀 | 无 |

| 草药 | 识别特征 | 毒性 |
|------|----------|------|
| 商陆 | 红茎串果、成熟后紫黑 | 全株有毒 |
| 乌头 | 深蓝紫花 | 全株剧毒 |
| 白屈菜 | 黄色浆液 | 有毒 |
| 细辛 | 根细如须、辛味极浓 | 小毒 |

### 5.3 草药采集与保存

**采集时间**：
```
花类：初开时采
根类：秋季采
叶类：茂盛时采
```

**保存方法**：
- 阴干或低温烘干
- 防潮、防虫、防霉变
- 密封保存

## 8. 中医急救知识 【新增】

### 8.1 中医急救基本原则

**核心理念**：
- 中医急救不是取代现代急救，而是在第一时间争取"黄金救治时间"
- 简便易行、立竿见影，关键时刻能救命

**急救黄金顺序**：
```
1. 呼叫专业帮助（120）
2. 确保环境安全
3. 进行基础生命支持
4. 运用已知的中医辅助急救法
```

### 8.3 常见急症急救

#### 8.3.1 晕厥急救

**症状识别**：
- 突然意识丧失
- 面色苍白
- 四肢厥冷
- 大汗淋漓

**急救步骤**：
```
1. 立即取平卧位
2. 解开衣领
3. 保持呼吸道通畅
4. 针刺人中穴强刺激
5. 配合合谷穴按压
6. 艾灸百会穴至苏醒
7. 十宣穴放血（每指1-2滴）
```

**操作要点**：
- 人中穴：用拇指屈曲90°，指尖垂直切压
- 频率：一紧一松，每秒3-4次
- 合谷穴：拇指用力掐捏，持续2-3分钟

#### 8.3.2 心绞痛急救

**症状识别**：
- 胸骨后压榨性疼痛
- 放射至左肩、后背
- 伴胸闷、窒息感

**急救步骤**：
```
1. 立即停止活动
2. 采取坐位或半卧位
3. 舌下含服硝酸甘油/速效救心丸
4. 按压内关穴（腕横纹上2寸）
5. 艾灸至阳穴（第7胸椎棘突下）
```

**注意**：
- 疼痛持续15分钟不缓解，立即就医
- 血压低于100/60mmHg慎用硝酸甘油

#### 8.3.3 中风急救

**症状识别**：
- 突然昏倒、不省人事
- 牙关紧闭
- 肢体强直
- 面红气粗（闭证）

**急救步骤**：
```
1. 立即拨打120！
2. 准确记录发病时间
3. 患者平卧，头偏一侧
4. 十宣穴放血（闭证适用）
5. 每指挤出3-5滴血
```

**重要警示**：
| 证型 | 表现 | 处理 |
|------|------|------|
| 闭证 | 牙关紧闭、面红气粗 | 十宣放血 |
| 脱证 | 手撒口开、大汗淋漓 | 绝对禁止放血！ |

#### 8.3.4 中暑急救

**症状识别**：
- 高热、头痛
- 恶心呕吐
- 皮肤干热
- 意识障碍

**急救步骤**：
```
1. 迅速转移至阴凉处
2. 脱去外衣
3. 刮痧：沿脊柱两侧自上而下
4. 十宣穴放血
5. 耳尖放血（快速点刺）
6. 藿香正气水口服
```

#### 8.3.5 癫痫发作

**症状识别**：
- 抽搐
- 口吐白沫
- 意识丧失

**急救步骤**：
```
1. 保持呼吸道通畅
2. 头偏向一侧
3. 防止咬伤舌头
4. 十宣穴放血
```

**注意**：掐人中和强行按压可能加重抽搐！

### 8.4 十宣穴放血详解

**位置**：十个手指尖的末端正中

**适应症**：
- 高热惊厥
- 癫痫发作
- 中风闭证
- 中暑昏厥

**操作方法**：
```
1. 患者取平卧位
2. 头部偏向一侧（防呕吐物堵塞）
3. 75%酒精消毒穴位皮肤
4. 消毒毫针快速刺入
5. 每指挤出3-5滴血
6. 总量不宜超过20滴
```

**禁忌人群**：
| 人群 | 处理 |
|------|------|
| 体质虚弱者 | 慎用 |
| 孕妇 | 慎用 |
| 小儿 | 慎用 |
| 凝血功能障碍者 | 严禁使用 |

**重要提醒**：
- 只适用于闭证（牙关紧闭、面红气粗）
- 脱证（面色苍白、四肢厥冷）禁止放血
- 不能替代拨打120和送医

| 钥匙 | 穴位 | 功效 |
|------|------|------|
| 第一钥匙 | 人中 | 晕倒急救，唤醒开关 |
| 第二钥匙 | 劳宫 | 心梗急救，心脏休息宫殿 |
| 第三钥匙 | 委中 | 急性腰痛，腰背万能解药 |
| 第四钥匙 | 太阳 | 头痛急救，阳气生发之源 |
| 第五钥匙 | 脚后跟 | 流鼻血，天然止血阀 |
| 第六钥匙 | 内关 | 呕吐急救，止呕要穴 |
| 第七钥匙 | 少商 | 咳嗽急救，清肺止咳 |

### 8.6 中医急救注意事项

| 原则 | 说明 |
|------|------|
| 准确辨证 | 分清寒热虚实，避免误治 |
| 操作规范 | 针灸需专业，非专业可改用点穴 |
| 及时就医 | 中医急救为临时措施，需送医 |
| 特殊人群 | 孕妇、儿童、老人注意用药剂量 |
| 禁忌症 | 出血性疾病患者慎用放血疗法 |
