## 1. 经典风水书籍核心精解

### 1.1 《葬书》/《葬经》- 郭璞

**地位**：中国风水文化之宗，风水经典之首

**核心概念**：
```
葬者，乘生气也。
气乘风则散，界水则止。
古人聚之使不散，行之使有止，故谓之风水。
```

**五大理论**：
| 理论 | 说明 |
|------|------|
| 乘生气论 | 葬者藏也，乘生气也 |
| 藏风得水论 | 风水之法，得水为上，藏风次之 |
| 形势论 | 形与势的观察 |
| 四神砂论 | 青龙、白虎、朱雀、玄武 |
| 土质标准论 | 土色判断吉凶 |

**小说素材**：
- 龙脉寻找：寻找真龙穴
- 乘气葬法：如何乘生气
- 四神砂：左青龙右白虎前朱雀后玄武
- 土色鉴定：五色土吉凶

### 1.2 《青囊经》

青囊奥语：坤壬乙文曲从头出，艮丙辛位位武曲同

**小说素材**：

## 3. 经典易术书籍核心精解

### 3.1 《梅花易数》- 邵雍（托名）

**地位**：易学象数派代表作，随时随地可占

**名称由来**：观梅占
```
辰年十二月十七日申时
邵雍观梅，看见两只麻雀争枝坠地
起卦：泽火革之泽山咸
断：次日有少女折梅摔伤膝盖
应验！
```

**核心原理**：
```
以象取卦、以数推演、触机而断
```

**起卦方法**：

| 方法 | 操作 |
|------|------|
| 时间起卦 | 年+月+日=上卦，年+月+日+时=下卦 |
| 数字起卦 | 第一个数为上卦，第二个数为下卦 |
| 方位起卦 | 物为上卦，方位为下卦 |
| 外应起卦 | 心动时所见所闻 |

**八卦万物属类（小说素材）**：

| 卦 | 人物 | 身体 | 动物 | 物品 | 颜色 |
|----|------|------|------|------|------|
| 乾 | 父、老人、官贵 | 头、骨 | 马 | 金宝、圆物、镜 | 赤色 |
| 坤 | 母、老妇 | 腹、皮肤 | 牛 | 布帛、方物、瓦器 | 黄黑 |
| 震 | 长男 | 足、发 | 龙、虫 | 乐器、竹木 | 青绿 |
| 巽 | 长女 | 股、肱 | 鸡 | 绳、扇、香料 | 青绿 |
| 坎 | 中男 | 耳、血 | 猪、狐 | 弓箭、轮子、有核物 | 黑 |
| 离 | 中女 | 目 | 野鸡、龟 | 文书、炉、甲胄 | 红紫 |
| 艮 | 少男 | 手指、鼻 | 狗、鼠、虎 | 门、果子、岩石 | 黄 |
| 兑 | 少女 | 口、舌 | 羊 | 开口物、金属、有缺口物 | 白 |

**体用生克**：

| 关系 | 吉凶 | 说明 |
|------|------|------|
| 用生体 | 大吉 | 得财、得助 |
| 体生用 | 耗损 | 付出、消耗 |
| 体克用 | 小吉 | 费力但成 |
| 用克体 | 大凶 | 破财、受伤 |
| 体用比和 | 中平 | 和谐顺利 |

**实战案例**（观梅占）：
```
辰年（5）+十二月（12）+十七日（17）= 34
34 ÷ 8 = 4 余 2 → 兑卦（上卦）
34 + 申时（9）= 43
43 ÷ 8 = 5 余 3 → 离卦（下卦）
43 ÷ 6 = 7 余 1 → 初爻动

本卦：泽火革（☱☲）
互卦：乾、巽
变卦：泽山咸（☱☶）
```

**小说素材**：
- 观梅占：邵雍成名卦
- 少年亡齿：老人有忧
- 今日动静：物我感应
- 鸡叫雨降：万物类象

### 3.2 《皇极经世》- 邵雍

**地位**：北宋理学大家邵雍的宇宙哲学体系

**核心概念**：
```
元、会、运、世：时间周期
日、月、星、辰：空间体系
声音起卦：万物数术
```

**时间体系**：
```
一元 = 12会
一会 = 30运
一运 = 30世
一世 = 30年
```
