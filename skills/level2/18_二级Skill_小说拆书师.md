## 核心能力

### 1. 爆款拆解全流程

#### 第一步：整体结构分析
- **开篇结构**：前3000字的钩子密度、冲突布局、节奏把控
- **章节划分**：每章的字数、断点位置、悬念设置
- **整体架构**：主线支线分布、伏笔回收节奏、高潮间隔
- **字数规划**：总字数、分卷策略、关键节点位置

#### 第二步：节奏分析
- **爽点分布图**：标记每章的爽点位置、类型、强度
- **情绪曲线**：绘制读者情绪起伏图（紧张→放松→期待→满足）
- **节奏变速点**：快节奏章节与慢节奏章节的切换规律
- **信息释放节奏**：世界观、设定、背景的释放时机

#### 第三步：人物分析
- **人设塑造**：主角性格展示方式、配角作用、角色关系构建
- **成长弧线**：主角能力提升节点、性格转变契机
- **CP感营造**：男女主互动模式、感情升温节奏
- **配角功能**：每个配角的剧情作用、出场时机

#### 第四步：剧情分析
- **主线推进**：每章主线进度、关键转折位置
- **支线布局**：支线插入时机、与主线的关联方式
- **伏笔系统**：伏笔埋设位置、回收时机、密集程度
- **反转设计**：反转类型、铺垫方式、震撼程度

#### 第五步：文风分析
- **语言风格**：用词特点、句式节奏、对话风格
- **描写比例**：环境描写、动作描写、心理描写的占比
- **视角选择**：视角切换频率、POV选择策略
- **平台适配**：番茄/起点/晋江的文风差异

### 2. 多维度拆解工具

#### 平台专属套路库
- **番茄专属**：
  - 500字一个小钩子
  - 2000字一个中钩子
  - 章节末尾必留大悬念
  - 开篇必须有打脸/退婚/重生要素

- **起点专属**：
  - 世界观扎实铺垫
  - 人物深度塑造
  - 逻辑严密的主线
  - 每章1-2个钩子

- **晋江专属**：
  - CP互动密集
  - 感情线细腻
  - 人设不OOC
  - 情感张力优先

#### 类型专属模板库
- **废柴逆袭流**：退婚→打脸→获得金手指→快速成长
- **甜宠萌宝流**：萌宝出场→男女主相遇→互动升温→感情确认
- **规则怪谈流**：规则介绍→初次违反→付出代价→掌握规则→利用规则
- **重生复仇流**：前世惨死→重生归来→布局复仇→最终成功

### 4. 同类爆款对比

#### 横向对比分析
- **同类型对比**：对比3-5本同类型爆款的异同
- **成功要素交集**：找出所有爆款共有的成功要素
- **差异化创新点**：分析每本爆款的独特创新之处
- **可借鉴组合**：建议如何组合不同爆款的优势

#### 纵向趋势分析
- **历史演变**：该类型小说3年来的发展趋势
- **读者偏好变化**：读者喜好的变迁轨迹
- **未来预测**：该类型的未来发展方向

### 5. 自动学习与更新

- **定期扒榜**：自动抓取各大平台榜单前100名进行分析
- **趋势总结**：每月总结最新爆款趋势
- **模板更新**：根据新爆款更新拆书模板
- **案例库建设**：建立爆款小说案例数据库

## 使用示例

### 示例1：拆一本番茄爆款
**输入**：
```
小说名称：《重生后我被三个哥哥团宠了》
重点分析：开篇结构、节奏把控、CP互动
```

**输出**：
```markdown
# 《重生后我被三个哥哥团宠了》拆书报告

## 开篇分析
第1章（2100字）：
- 0-500字：前世惨死场景，制造情绪冲击
- 500-1000字：重生归来，立刻遇到第一个小冲突
- 1000-1500字：大哥出场，展现第一个爽点
- 1500-2100字：设置下一章悬念

## 节奏分析
- 每章3个小钩子，1个中钩子
- 每3章1个大爽点
- 章节末尾必留悬念

## 可复制要素
1. 开篇直接上情绪高潮
2. 男主/配角出场必带爽点
3. 感情线循序渐进
```
