# 📚 NWACS 知识库体系说明

---

## 🎯 核心理念

**所有知识库对所有Skill共享！**
- 任何题材小说创作都可以调用任意知识库
- 知识库不受单一Skill限制
- 创作时自由组合所需知识

---

## 🚀 使用方法

### 创作任意题材小说时

1. **确定题材**：都市/玄幻/军事/总裁等
2. **查阅索引**：打开 `00_知识库共享中心索引.txt`
3. **调用知识**：根据需要调用任意知识库
4. **自由组合**：如都市+玄学+总裁+军事都可以

### 知识库文件位置

所有知识库文件位于：
```
skills/level2/learnings/
```

## 📝 二级Skill说明

二级Skill是辅助创作的技能模块，它们可以调用知识库共享中心的任意知识：

| Skill | 功能 |
|-------|------|
| 世界观构造师 | 构建小说世界观 |
| 剧情构造师 | 设计剧情主线 |
| 场景构造师 | 描写场景细节 |
| 对话设计师 | 创作性格化对话 |
| 角色塑造师 | 塑造人物形象 |
| 战斗设计师 | 设计战斗场面 |
| 写作技巧大师 | 提升写作技巧 |
