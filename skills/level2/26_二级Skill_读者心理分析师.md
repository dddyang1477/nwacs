## 核心能力

### 1. 读者画像分析

#### 基本画像
| 维度 | 内容 |
|------|------|
| 年龄段 | 18-25/26-35/36-45/45+ |
| 性别 | 男频/女频/通用 |
| 职业 | 学生/白领/自由职业/其他 |
| 阅读场景 | 通勤/睡前/休闲/碎片时间 |

#### 阅读动机
```
动机类型：
- 娱乐消遣：打发时间、放松
- 情感共鸣：寻求代入感、情感宣泄
- 知识获取：学习技能、了解世界
- 社交需求：讨论分享、圈子认同
- 成就满足：看主角成长、获得成就感

优先级分析：
主要动机 + 次要动机 = 读者核心需求
```

### 2. 阅读体验设计

#### 爽点设计
```
爽点类型：
- 升级爽：实力提升
- 打脸爽：反击嘲讽
- 获得爽：获得宝物/技能
- 守护爽：保护重要的人
- 复仇爽：大仇得报
- 认知爽：揭开真相

爽点密度：
每章至少1个小爽点
每10章1个中爽点
每50章1个大爽点
```

#### 虐点设计
```
虐点类型：
- 情感虐：生离死别、爱而不得
- 成长虐：付出代价、艰难困苦
- 命运虐：被误解、被背叛
- 认知虐：发现残酷真相

虐点原则：
- 虐后必有甜
- 虐是为爽服务
- 控制虐的频率和程度
```

### 3. 代入感设计

#### 代入方式
```
视觉代入：
- 第一人称视角
- 限制性第三人称
- 读者即是主角

情感代入：
- 主角有明确目标
- 主角有成长空间
- 主角有鲜明性格

情境代入：
- 熟悉的场景
- 能理解的冲突
- 能感受的情绪
```

#### 移情设计
```
移情路径：
熟悉感 → 认同感 → 代入感 → 情感共鸣

设计要点：
- 主角设定接地气
- 困境具有普遍性
- 成长具有可复制性
```

### 4. 期待感管理

#### 期待建立
```
期待类型：
- 结果期待：想知道结果
- 过程期待：想看发展
- 认知期待：想了解真相
- 情感期待：想看感情发展

建立方式：
- 埋下伏笔
- 设定目标
- 制造悬念
- 承诺冲突
```

#### 期待满足
```
满足节奏：
- 即时满足（小期待）
- 短期满足（中期待）
- 长期满足（大期待）

满足方式：
- 正向满足（达成目标）
- 反向满足（意外惊喜）
- 超越满足（超出预期）
```

### 5. 情感曲线设计

#### 情绪光谱
```
情绪类型：
正面情绪：
- 快乐、兴奋、自豪、满足
- 温暖、感动、治愈、甜蜜

负面情绪：
- 悲伤、愤怒、恐惧、焦虑
- 压抑、绝望、孤独、痛苦

中性情绪：
- 好奇、期待、惊讶、疑惑
```

#### 曲线设计
```
曲线公式：
情绪曲线 = 高潮(20%) + 低谷(20%) + 平缓(60%)

节奏设计：
- 每3章一个小波动
- 每10章一个完整起伏
- 每50章一个重大转折

极端情绪处理：
- 大悲后必有大喜
- 大喜后必有隐患
- 绝境逢生是最好的安排
```
