# NWACS Skill协作总览 v2.0

## ⚠️ 重要理念：技能是工具，不是教条

**所有Skill都应该灵活运用，不要死板遵循规则。**

技能是创作工具，不是枷锁。要根据具体情况、创作需求、读者群体灵活调整。

## 🎯 灵活运用原则

### 1. 组合自由
```
技能可以自由组合，而不是固定流程：
- 可以只用2-3个核心技能完成短篇
- 可以跳过某些技能直接创作
- 可以根据需要重复使用某个技能
```

### 2. 灵活顺序
```
传统顺序：市场→选题→大纲→世界观→情节→人物→场景...
灵活顺序：根据需要调整：
- 先有灵感？→直接进入剧情构造
- 先有人物？→先做角色塑造
- 先有世界观？→先做世界观构造
```

### 3. 按需调用
```
不是每次创作都要调用所有技能：
- 短篇小说：剧情+人物+技巧（3个技能）
- 中篇小说：+场景+对话（5个技能）
- 长篇小说：全流程（10+个技能）
```

### 4. 创新组合
```
鼓励跨技能组合创新：
- 苟道流 + 智斗流 = 新类型
- 单元剧 + 递进式 = 混合结构
- 痛点 + 爽点 = 痛爽文
```

## 📋 创作流程（灵活版）

**根据创作需求选择：**

### 短篇小说（1-3个技能）
1. **剧情构造**（必须有）→ 设计核心情节
2. **角色塑造**（建议有）→ 人物立起来
3. **写作技巧**（可选）→ 润色语言

### 中篇小说（3-5个技能）
1. **选题/题材选择**（可选）→ 确定方向
2. **剧情构造**（必须有）→ 核心情节
3. **角色塑造**（建议有）→ 人物群像
4. **场景构造**（可选）→ 关键场景
5. **技巧优化**（可选）→ 润色提升

### 长篇小说（5-10+个技能）
1. **市场分析**（可选）→ 了解趋势
2. **选题策划**（建议有）→ 确定赛道
3. **大纲架构**（建议有）→ 长期规划
4. **世界观构造**（建议有）→ 世界设定
5. **剧情构造**（必须有）→ 完整情节
6. **角色塑造**（必须有）→ 人物弧线
7. **场景构造**（可选）→ 关键场景
8. **战斗设计**（可选）→ 战斗场面
9. **对话设计**（可选）→ 对话打磨
10. **技巧+去AI+审计**（可选）→ 最终优化

## 🎨 创新组合案例

### 案例1：苟道流创新
```
基础组合：剧情构造 + 角色塑造
苟道特色：+ 智斗设计 + 稳健风格
灵活调整：减少战斗设计，增加心理描写
```

### 案例2：痛爽文创新
```
基础组合：剧情 + 人物 + 技巧
痛爽特色：+ 痛点设计 + 情感共鸣
灵活调整：在爽点章节加入痛点元素
```

### 案例3：混搭类型
```
仙侠 + 苟道 + 悬疑 = 《道诡异仙》风格
都市 + 系统 + 治愈 = 《我只想安静的做个苟道中人》风格
历史 + 推理 + 穿越 = 《如果可以再见》风格
```

## ⚡ 灵活运用检查清单

在创作时问自己：

- [ ] 这次创作需要哪些技能？全部还是部分？
- [ ] 技能的顺序可以调整吗？
- [ ] 可以混合不同类型的元素吗？
- [ ] 有没有创新的组合方式？
- [ ] 规则是帮助创作还是限制创作？

**记住：规则是死的，创作是活的。灵活运用才是王道！**
