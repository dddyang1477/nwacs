## 🎯 灵活运用原则

### 1. 组合自由
```
技能可以自由组合，而不是固定流程：
- 可以只用2-3个核心技能完成短篇
- 可以跳过某些技能直接创作
- 可以根据需要重复使用某个技能
```

### 2. 灵活顺序
```
传统顺序：市场→选题→大纲→世界观→情节→人物→场景...
灵活顺序：根据需要调整：
- 先有灵感？→直接进入剧情构造
- 先有人物？→先做角色塑造
- 先有世界观？→先做世界观构造
```

### 3. 知识库共享
```
不是每次创作只用自己题材的知识库：
- 写都市？可以加风水/军事/总裁元素
- 写玄幻？可以加命理/军事/国外神话
- 写军事？可以加玄学/总裁/历史元素
```

### 4. 创新组合
```
鼓励跨技能、跨知识库组合创新：
- 苟道流 + 智斗流 = 新类型
- 都市 + 玄学 + 军事 = 复合题材
- 总裁 + 风水 = 豪门玄学
- 玄幻 + 军事 + 总裁 = 超级混合
```

### 5. 2026年创作口诀
```
开篇三秒要出事，第一章人设反差露
每章结尾金句钩，节奏紧凑不可逆
平台规则要吃透，短剧改编是趋势
内求发疯双赛道，避开雷区是正道
```

## 📋 创作流程（灵活版）

**根据创作需求选择：**

### 短篇小说（1-3个技能）
1. **剧情构造**（必须有）→ 设计核心情节（冲突前置）
2. **角色塑造**（建议有）→ 人物立起来（人设反差）
3. **技巧优化**（可选）→ 润色语言（金句钩子）

### 中篇小说（3-5个技能）
1. **选题/题材选择**（可选）→ 确定方向（2026年顶流赛道）
2. **剧情构造**（必须有）→ 核心情节（节奏紧凑不可逆）
3. **角色塑造**（建议有）→ 人物群像
4. **场景构造**（可选）→ 关键场景
5. **技巧优化**（可选）→ 润色提升

### 长篇小说（5-10+个技能）
1. **市场分析**（可选）→ 了解趋势（2026年平台动态）
2. **选题策划**（建议有）→ 确定赛道
3. **大纲架构**（建议有）→ 长期规划
4. **世界观构造**（建议有）→ 世界设定
5. **剧情构造**（必须有）→ 完整情节
6. **角色塑造**（必须有）→ 人物弧线
7. **场景构造**（可选）→ 关键场景
8. **战斗设计**（可选）→ 战斗场面
9. **对话设计**（可选）→ 对话打磨
10. **技巧+去AI+审计**（可选）→ 最终优化

## ⚡ 灵活运用检查清单

在创作时问自己：

- [ ] 这次创作需要哪些技能？全部还是部分？
- [ ] 技能的顺序可以调整吗？
- [ ] 可以混合不同题材的知识库吗？
- [ ] 有没有创新的组合方式？
- [ ] 规则是帮助创作还是限制创作？
- [ ] 开篇是否做到了"冲突前置+人设反差+金句钩子"？
- [ ] 节奏是否符合"不可逆改变"原则？
- [ ] 是否适配目标平台（番茄/起点）？

**记住：规则是死的，创作是活的，知识库是共享的！**
