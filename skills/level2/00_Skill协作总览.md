#### 2. 状态追踪系统

```
状态定义：
- PENDING：等待分发
- RUNNING：执行中
- WAITING：等待上游输出（依赖未完成）
- PASSED：验收通过
- FAILED：验收失败，需重做
- BLOCKED：被其他任务阻塞
```

#### 3. 质量验收标准（三级门禁）

```
第一级：完整性检查 → 第二级：质量检查 → 第三级：综合终审

综合评分权重：
- 剧情设计（30%）
- 人物塑造（25%）
- 文笔质量（20%）
- 世界观（15%）
- 商业价值（10%）

评分等级：S(90-100) / A(80-89) / B(70-79) / C(60-69) / D(<60)
```

#### 4. 知识库路由

```
根据小说类型自动匹配知识库：
- 玄幻修仙：风水玄学 + 修炼体系 + 战斗技巧 + 起名知识库
- 都市异能：现代都市 + 格斗技巧 + 总裁知识 + 起名知识库
- 军事小说：军事战术 + 武器装备 + 地理环境 + 雇佣兵指南
- 悬疑推理：逻辑推理 + 心理学 + 经典案例 + 氛围营造
- 言情纯爱：情感描写 + 心理学 + 总裁/甜宠 + 起名知识库
```

#### 5. 冲突解决机制

```
仲裁优先级（从高到低）：
1. 用户明确需求 > Skill建议
2. 核心人设 > 辅助设定
3. 剧情完整性 > 单一元素完美度
4. 市场可行性 > 理想化设计
5. 效率 > 过度优化
```

## 🎯 灵活运用原则

### 1. 组合自由
```
技能可以自由组合，而不是固定流程：
- 可以只用2-3个核心技能完成短篇
- 可以跳过某些技能直接创作
- 可以根据需要重复使用某个技能
```

### 2. 灵活顺序
```
传统顺序：市场→选题→大纲→世界观→情节→人物→场景...
灵活顺序：根据需要调整：
- 先有灵感？→直接进入剧情构造
- 先有人物？→先做角色塑造
- 先有世界观？→先做世界观构造
```

### 3. 知识库共享
```
不是每次创作只用自己题材的知识库：
- 写都市？可以加风水/军事/总裁元素
- 写玄幻？可以加命理/军事/国外神话
- 写军事？可以加玄学/总裁/历史元素
```

### 4. 创新组合
```
鼓励跨技能、跨知识库组合创新：
- 苟道流 + 智斗流 = 新类型
- 都市 + 玄学 + 军事 = 复合题材
- 总裁 + 风水 = 豪门玄学
- 玄幻 + 军事 + 总裁 = 超级混合
```

## 📋 创作流程（灵活版）

**根据创作需求选择：**

### 短篇小说（1-3个技能）
1. **剧情构造**（必须有）→ 设计核心情节（冲突前置）
2. **角色塑造**（建议有）→ 人物立起来（人设反差）
3. **技巧优化**（可选）→ 润色语言（金句钩子）

### 中篇小说（3-5个技能）
1. **选题/题材选择**（可选）→ 确定方向（2026年顶流赛道）
2. **剧情构造**（必须有）→ 核心情节（节奏紧凑不可逆）
3. **角色塑造**（建议有）→ 人物群像
4. **场景构造**（可选）→ 关键场景
5. **技巧优化**（可选）→ 润色提升

### 长篇小说（5-10+个技能）
1. **市场分析**（可选）→ 了解趋势（2026年平台动态）
2. **选题策划**（建议有）→ 确定赛道
3. **大纲架构**（建议有）→ 长期规划
4. **世界观构造**（建议有）→ 世界设定
5. **剧情构造**（必须有）→ 完整情节
6. **角色塑造**（必须有）→ 人物弧线
7. **场景构造**（可选）→ 关键场景
8. **战斗设计**（可选）→ 战斗场面
9. **对话设计**（可选）→ 对话打磨
10. **技巧+去AI+审计**（可选）→ 最终优化
