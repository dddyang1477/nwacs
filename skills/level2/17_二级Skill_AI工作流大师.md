## 工作流程

### 短篇创作模式（一键跑完）

**适用场景**：短篇小说、短剧本、单元剧故事

**流程**：
1. 输入核心创意（题材 + 一句话简介）
2. 选择目标平台（番茄/起点/晋江/其他）
3. 点击「一键生成」
4. 等待 3-5 分钟，获得完整作品
5. （可选）人工微调润色

**示例流程**：
输入：都市爽文，主角隐忍三年觉醒，打脸伪善堂兄
自动：总纲 → 人设 → 分章大纲 → 正文 1-3 章 → 润色 → 番茄风格消痕
输出：完整短篇作品

### 长篇创作模式（分步确认）

**适用场景**：长篇小说、系列作品、精品创作

**流程**：
1. 输入核心设定
2. 生成世界观设定 → 人工确认修改
3. 生成人物设定 → 人工确认修改
4. 生成大纲 → 人工确认调整
5. 生成第 1 章 → 人工确认
6. 继续生成后续章节 → 每 3-5 章人工确认一次
7. 全部完成后批量润色消痕

**优点**：
- 精准把控质量
- 避免走偏
- 随时调整方向

## 预设工作流模板

### 番茄爽文工作流

```
1. 选题策划（300字爆款开头创意）
2. 人物塑造（3个核心人物三维建模）
3. 剧情构造（黄金三章 + 前30章爽点排布）
4. 正文生成（第1-3章，每章2000字）
5. 润色优化（强化爽点、快节奏调整）
6. AI消痕（番茄专属口语化）
```

### 起点玄幻工作流

```
1. 世界观构建（修炼体系 + 势力格局）
2. 人物塑造（主角成长弧线）
3. 剧情构造（完整大纲 + 前10章）
4. 战斗设计（关键战斗场景）
5. 正文生成（第1-10章）
6. 润色优化
```

### 晋江情感工作流

```
1. 情感线设计
2. 人物塑造（CP 搭配）
3. 剧情构造（情感起伏曲线）
4. 正文生成
5. 细腻润色
6. 消痕处理
```

## 分步工作流详解

### 第一步：设定确认

**系统输出**：完整世界观 + 人物档案 + 总纲
**用户操作**：
- [ ] 确认通过 → 进入下一步
- [ ] 提出修改意见 → 重新生成
- [ ] 自行修改 → 手动编辑

### 第二步：大纲生成

**系统输出**：分章大纲 + 伏笔清单 + 节奏规划
**用户操作**：
- [ ] 确认通过
- [ ] 调整某几章设定
- [ ] 修改伏笔位置

### 第三步：正文生成

**系统输出**：每章 2000-3000 字正文
**用户操作**：
- [ ] 每章单独确认
- [ ] 批量生成后统一确认
- [ ] 指定章节要求调整

### 第四步：润色消痕

**系统输出**：润色后的终稿
**用户操作**：
- [ ] 一键消痕处理
- [ ] 平台风格适配
- [ ] 违规词检查
