```
一级Skill (总调度官)
  ├── 接收用户指令

  ├── 调度二级Skill
  │   ├── 世界观构造师 → 三级: 地理/种族/规则
  │   ├── 剧情构造师   → 三级: 主线/支线/伏笔
  │   ├── 角色塑造师   → 三级: 背景/性格/关系
  │   ├── 战斗设计师   → 三级: 场景/招式/节奏
  │   └── ...
  └── 整合结果输出
```
