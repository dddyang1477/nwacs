## 一、系统快速开始

### 1.1 新手入门三步曲
```
第1步：选择类型
- 玄幻修仙
- 都市言情
- 悬疑推理
- 科幻未来
- 历史穿越
- 恐怖惊悚
- 游戏竞技

第2步：选择平台
- 起点中文网
- 番茄小说
- 晋江文学城
- 纵横中文网
- 七猫免费小说

第3步：一键生成
- 输入想法（100字以内）
- 点击"开始创作"
- 系统自动生成完整大纲+第1-3章正文
```

### 1.2 三种使用模式
| 模式 | 适用场景 | 自动化程度 | 效率 |
|------|---------|-----------|------|
| **全自动模式** | 新手，想快速写完 | 100%自动 | 10000-20000字/周 |
| **半自动模式** | 有基础，想优化节奏 | 70%自动+30%人工 | 5000-10000字/周 |
| **自定义模式** | 老手，想用特定Skill | 手动控制Skill选择 | 3000-5000字/周 |

## 二、详细使用流程

### 2.1 灵感激发阶段
**功能：**
- 选题策划大师分析市场
- 创新灵感生成器提供创意
- 市场分析师分析趋势

**操作：**
```
用户：我想写个小说，但不知道写什么
系统：自动分析中……
- 2026年最热题材：系统+马甲、年代文、恶毒女配
- 推荐方向：穿成恶毒女配，全家都是反派
- 创新点：结合空间+年代+直播
- 平台推荐：番茄小说
```

### 2.2 世界观构建阶段
**功能：**
- 世界观构造师+世界观大师共同设计
- 规则体系设计（修仙/现代/超能力）
- 地理环境设计
- 种族文明设计

**操作：**
```
用户：帮我设计个世界观
系统：自动生成……
- 力量体系：系统+空间+修仙
- 时代背景：80年代重生
- 地理：农村→城市→国际
- 核心冲突：改变历史+打脸+直播
```

### 2.3 大纲设计阶段
**功能：**
- 大纲架构师+剧情大师共同设计
- 主线剧情+支线剧情
- 伏笔布局
- 节奏控制

**操作：**
```
用户：帮我设计100章大纲
系统：自动生成……
- 1-10章：开篇钩子（重生+空间觉醒）
- 11-30章：第一个大高潮（打脸全家+直播赚钱）
- 31-60章：第二个大高潮（离开农村+系统升级）
- 61-90章：第三个大高潮（国际冲突+打脸）
- 91-100章：完美结局
```

### 2.4 人物设计阶段
**功能：**
- 角色塑造师+角色大师共同设计
- 主角5人+配角20人+反派5人
- 人物关系网络
- 人物成长弧光

**操作：**
```
用户：帮我设计主角
系统：自动生成……
- 女主：林晚，重生+系统+马甲
- 男主：顾晏城，总裁+偏执+宠妻
- 女配：恶毒原主+重生后的逆袭
- 男配：炮灰+追求者
```

### 2.5 正文写作阶段
**功能：**
- 写作技巧大师+金句大师+场景大师
- 战斗设计师+对话设计师
- 质量检测+AI消痕

**操作：**
```
用户：帮我写第1章
系统：自动生成……
- 场景：80年代破旧仓库
- 人物：林晚（重生）+李虎（反派）
- 情节：林晚醒来，李虎要伤害她，系统觉醒
- 结尾：下集预告，林晚逆袭！
```

### 2.6 质量检测阶段
**功能：**
- 四层质量门禁检测
- AI痕迹自动消除
- 质量优化建议

**操作：**
```
用户：帮我检查一下这一章
系统：自动检测中……
- 基础质量：✅通过（语法标点）
- AI痕迹：⚠️需优化（模板句式偏多）
- 节奏爽点：⚠️需优化（爽点不够密集）
- 平台合规：✅通过
- 自动修复：已修复3个问题，建议再看
```

### 2.7 发布规划阶段
**功能：**
- 发布规划师+市场分析师
- 平台选择建议
- 更新节奏规划
- 读者运营建议

**操作：**
```
用户：帮我规划发布
系统：自动生成……
- 平台推荐：番茄小说（最适合这个类型）
- 更新节奏：日更2章，每章2000字
- 黄金时间：早7-8点或晚20-22点
- 标题建议：每章标题要有钩子
```
