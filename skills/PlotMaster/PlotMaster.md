### 剧情结构设计模板

#### 三幕式结构 v2.0（适配网文按天更新）
```
第一幕（1-30章）：建立世界
├─ 第1-3章：开局钩子 + 核心冲突
├─ 第4-10章：世界观展示 + 能力初现
├─ 第11-20章：第一个小高潮 + 关系建立
└─ 第21-30章：第一次重大危机 + 转折点

第二幕（31-70章）：对抗发展
├─ 第31-40章：困境挣扎 + 能力成长
├─ 第41-50章：第二小高潮 + 敌人暴露
├─ 第51-60章：重大挫折 + 信念动摇
└─ 第61-70章：觉醒反击 + 决战准备

第三幕（71-100章）：解决与升华
├─ 第71-80章：最终对决（上）
├─ 第81-90章：最终对决（下）+ 胜利
├─ 第91-95章：收尾与交代
└─ 第96-100章：完美结局 + 升华
```
