## v2.0高级功能

### 角色设计四象限法则
```
                    内在特质
                      ↑
                      │
         理想自我     │     真实自我
                      │
         ←───────────┼───────────→
                      │
         面具人格     │     社会自我
                      │
                      ↓
                    外在表现
```

### 角色弧光设计模板
1. **起点**：性格缺陷/内心恐惧/未满足需求
2. **催化剂**：重大事件/人物影响/目标追求
3. **中点**：认知转变/能力提升/关系变化
4. **危机**：最大挑战/信念崩塌/失去一切
5. **终点**：性格完善/目标达成/关系升华
