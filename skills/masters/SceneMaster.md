## v1.0高级功能

### 五感描写矩阵
| 感官 | 描写要素 | 应用场景 | 示例 |
|------|---------|---------|------|
| 视觉 | 色彩/光影/形状 | 所有场景 | "金色的阳光..." |
| 听觉 | 声音/静默/回响 | 战斗/恐怖/温馨 | "风声鹤唳..." |
| 嗅觉 | 气味/香气/腐臭 | 特定场景 | "桂花香..." |
| 味觉 | 味道/口感/余味 | 美食/情感 | "苦涩的..." |
| 触觉 | 温度/材质/触感 | 亲密/危险 | "冰冷的..." |

### 场景类型设计模板
```
【玄幻场景模板】
├─ 灵气浓度：高/中/低
├─ 场景特色：仙气/魔气/妖气
├─ 光影效果：七彩祥云/紫气东来
├─ 环境音：仙鹤/风声/泉水声
└─ 氛围基调：神秘/威严/幽静

【都市场景模板】
├─ 时代感：现代/近未来
├─ 场景特色：繁华/破败/小资
├─ 光影效果：霓虹/日光/月光
├─ 环境音：车流/人声/音乐
└─ 氛围基调：紧张/轻松/浪漫

【恐怖场景模板】
├─ 空间感：封闭/开阔/扭曲
├─ 场景特色：血迹/黑暗/诡异
├─ 光影效果：无光/闪烁/阴影
├─ 环境音：寂静/异响/低语
└─ 氛围基调：恐惧/紧张/不安
```
