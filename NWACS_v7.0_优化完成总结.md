## 📝 快速使用指南

### 🚀 开始创作（推荐）

双击运行 `启动创作引擎.bat`
1. 选择小说类型
2. 输入创作需求
3. 设定章节标题和目标字数
4. 自动生成章节
5. 可选保存到文件

### 🔍 项目诊断

双击运行 `启动诊断工具.bat`
- 检查项目健康状态
- 检测编码问题
- 验证Python语法

### 📱 飞书集成（可选）

1. 在飞书群添加机器人，获取Webhook
2. 填入 `config/feishu_config.json`
3. 双击 `启动飞书集成.bat`
4. DeepSeek学习后自动推送报告
