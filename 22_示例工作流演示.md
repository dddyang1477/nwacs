# 📖 示例工作流：从需求到成品
## 完整演示：创作一本玄幻仙侠小说

---

## 用户输入

```
我想写一本玄幻仙侠小说，主角是一个被宗门抛弃的废柴弟子，
意外获得上古剑仙传承，最终成为一代剑宗。
风格要热血激昂，有兄弟情义，有红颜知己，预计100万字。
参考《凡人修仙传》的严谨，但节奏更快。
```

---

## Phase 1：需求解析（一级Skill执行）

### 输出
```
【需求解析报告】

小说类型：玄幻仙侠（东方玄幻）
子类型：废柴逆袭 + 剑道传承
风格：热血激昂
基调：兄弟情义 + 红颜知己
参考作品：《凡人修仙传》（节奏更快）
预计字数：100万字
目标读者：男性向，喜欢升级爽文

【任务拆解】
1. 世界观构建（激活：玄幻仙侠专家）
2. 角色设计（主角 + 3-5位关键配角）
3. 剧情架构（三幕式，预计800-1000章）
4. 正文创作（按卷推进）

【协作计划】
Phase 2: 世界观构造师 → 输出世界观
Phase 3: 角色塑造师 → 输出角色档案
Phase 4: 剧情构造师 → 输出大纲
Phase 5: 场景+对话+战斗 → 按需填充
Phase 6: 正文创作 → 逐卷输出
Phase 7: 审查优化 → 去AI化 + 质量审计
```

---

## Phase 2：世界观构建

### 输入上下文
```
类型：玄幻仙侠
核心需求：剑道传承、废柴逆袭、节奏快
参考：《凡人修仙传》的严谨
```

### 输出（世界观构造师）
```
# 《剑道独尊》世界观设定

## 一、核心规则（5条）
1. 【灵气守恒】世间灵气总量有限，越强者消耗越多，
   高阶修士必须争夺灵气资源 → 冲突源
2. 【剑意共鸣】剑修与剑产生共鸣，人剑合一可越级挑战，
   但共鸣失败会反噬 → 风险与机遇
3. 【天劫必至】每突破大境界必遭天劫，天劫强度与因果相关，
   杀戮越多天劫越强 → 道德约束
4. 【传承限制】上古传承有适配条件，强行接受会神魂撕裂，
   主角因特殊体质恰好适配 → 主角光环的合理性
5. 【因果轮回】修士的每个选择都会产生因果，
   大因果可影响天道 → 深层主题

## 二、世界概览
### 地理版图
- 九州大陆：人界核心，分东荒、南岭、西漠、北原、中州
- 天外天：仙界入口，需渡劫飞升
- 幽冥界：死者归宿，与主角身世相关

### 势力格局
| 势力 | 特点 | 与主角关系 |
|------|------|-----------|
| 青云宗 | 主角原宗门，伪君子聚集地 | 敌对 |
| 万剑阁 | 剑道圣地，主角最终归属 | 盟友 |
| 魔道六宗 | 不择手段，但非纯恶 | 复杂 |
| 天机阁 | 情报组织，亦正亦邪 | 交易 |

## 三、修炼体系
| 境界 | 名称 | 特征 | 战力标志 |
|------|------|------|----------|
| 1 | 炼气 | 引气入体 | 真气外放 |
| 2 | 筑基 | 丹田化海 | 御剑飞行 |
| 3 | 金丹 | 凝结金丹 | 剑气化形 |
| 4 | 元婴 | 元婴初成 | 剑域展开 |
| 5 | 化神 | 神识通天 | 一剑断山 |
| 6 | 合体 | 人剑合一 | 剑意化形 |
| 7 | 大乘 | 半步仙人 | 破碎虚空 |
| 8 | 渡劫 | 直面天道 | 九死一生 |
| 9 | 真仙 | 超脱凡俗 | 剑道永恒 |

## 四、待填充区域
- 上古剑仙的具体来历
- 主角被抛弃的深层原因
- 红颜知己的身世背景
```

## Phase 3：角色设计

### 输入上下文
```
世界观：已确定（见Phase 2输出）
需求：废柴弟子、剑仙传承、兄弟情义、红颜知己
```

### 输出（角色塑造师）
```
# 角色档案

## 主角：林凡
- **年龄**：16岁（开局）
- **外貌**：清秀但略显苍白，眼神倔强，
  右手有一道旧伤疤（被宗门驱逐时留下）
- **身份**：青云宗外门弟子 → 被驱逐 → 散修
- **核心欲望**：证明自己，让青云宗后悔
- **核心恐惧**：再次被抛弃，成为无用之人
- **错误信念**："只有强大才不会被抛弃"
- **秘密**：母亲是上代剑仙之女，因爱上凡人被家族除名
- **体质**：【空灵剑体】（罕见剑道体质，与上古传承完美适配）

### 角色弧线
- **起点**：被宗门驱逐，丹田破碎，沦为笑柄
- **催化剂**：坠崖获得上古剑仙"太虚子"传承
- **挣扎**：力量与道德的冲突，复仇与宽恕的选择
- **终点**：成为剑宗，但选择宽恕而非毁灭青云宗

## 兄弟：萧烈

- **特点**：狂放不羁，重情重义，表面冷酷内心柔软
- **与林凡关系**：从对手→战友→生死兄弟

## 红颜：苏婉儿
- **身份**：万剑阁圣女，天生剑心通明
- **特点**：外冷内热，有主见，不依附男主
- **与林凡关系**：从竞争→欣赏→并肩作战
- **功能**：不是花瓶，有自己的成长线

## 反派：赵天行
- **身份**：青云宗少宗主，驱逐林凡的罪魁祸首
- **动机**：嫉妒林凡的剑道天赋，害怕被超越
- **特点**：不是纯恶，有自己的恐惧和野心
- **结局**：被林凡击败后自我了断，留下反思空间
```

## Phase 4：剧情架构

### 输入上下文
```
世界观：Phase 2输出
角色：Phase 3输出
风格：热血激昂，节奏快
```

### 输出（剧情构造师）
```
# 《剑道独尊》剧情架构

## 核心设定
- **一句话梗概**：被宗门抛弃的废柴，获上古传承，
  以剑证道，最终成为一代剑宗。
- **核心冲突**：个人复仇 vs 大道无情 vs 兄弟情义
- **主题**：真正的强大不是力量，而是选择宽恕的勇气

## 三幕式结构

### 第一幕：觉醒（1-200章，20%）
- **钩子**：林凡被宗门驱逐，丹田破碎，众叛亲离
- **触发事件**：坠崖获太虚子传承，空灵剑体觉醒
- **第一转折点**：重返青云宗外门大比，一剑击败内门弟子，
  震惊全场（第50章小高潮）

### 第二幕：崛起（201-700章，60%）
- **进入新世界**：离开青云宗，游历九州
- **中点反转**：发现母亲身世，得知青云宗与母亲之死有关（第400章）
- **至暗时刻**：萧烈为救林凡堕入魔道深渊，
  苏婉儿被天机阁绑架（第600章）
- **盟友**：万剑阁、散修联盟、魔道中的正义派

### 第三幕：证道（701-900章，20%）
- **最终准备**：领悟"无我剑意"，突破大乘
- **高潮对决**：独闯青云宗，一人一剑，连败七大长老
- **结局**：击败赵天行，但选择宽恕，解散青云宗而非毁灭
- **尾声**：与苏婉儿、萧烈共建新秩序，开创新的剑道时代

## 分卷规划
| 卷数 | 名称 | 章节 | 核心事件 |
|------|------|------|----------|
| 第一卷 | 废柴觉醒 | 1-100 | 获传承，初露锋芒 |
| 第二卷 | 剑出青云 | 101-200 | 外门大比，离开宗门 |
| 第三卷 | 九州风云 | 201-400 | 游历，结识萧烈和苏婉儿 |
| 第四卷 | 身世之谜 | 401-600 | 发现母亲真相，危机爆发 |
| 第五卷 | 魔道深渊 | 601-700 | 救萧烈，战魔道 |
| 第六卷 | 剑道独尊 | 701-900 | 最终决战，证道成仙 |

## 伏笔清单
| 伏笔 | 埋设 | 回收 | 重要性 |
|------|------|------|--------|
| 右手伤疤 | 第1章 | 第400章（母亲留下的剑印） | ★★★★★ |
| 太虚子的叹息 | 第5章 | 第800章（预见到林凡的选择） | ★★★★☆ |
| 赵天行的玉佩 | 第20章 | 第700章（与林凡母亲有关） | ★★★★☆ |
```

## Phase 5：正文创作（示例：第一章）

### 输入上下文
```
世界观：Phase 2
角色：Phase 3（林凡16岁，被驱逐）
剧情：第一卷第一章，开局钩子
风格：热血激昂，节奏快
```

### 输出（正文）
```markdown
# 第一章 驱逐

雨很大。

青云宗的山门在雨幕中像一头沉默的巨兽，
石阶上流淌的雨水泛着淡淡的血色——
那是林凡的血。

"林凡，外门弟子，修炼三年，仍在炼气三层。"
执法长老的声音像一块冰，"按宗门规矩，
废去修为，逐出师门。"

林凡跪在地上，右手的伤疤在雨中隐隐作痛。
那是三年前，他为救赵天行被妖兽所伤留下的。

"长老，"他抬起头，雨水顺着苍白的脸颊滑落，
"三年前外门试炼，是我救了赵师兄。"

执法长老眼皮都没抬："赵师侄乃天灵根，
未来宗门栋梁。你不过伪灵根，
能为他挡伤，是你的荣幸。"

林凡笑了。

那笑容里没有愤怒，只有一种让人心悸的平静。
他缓缓站起身，脊背挺得笔直。

"好一个荣幸。"

他转身，一步一步走下山门石阶。
没有人送，没有人看。
三年前他入门时，石阶两旁站满了同门。
今天，只有雨。

走到第三百级石阶时，他停住了。

身后传来脚步声，还有一声刻意压低的嘲笑：
"林废物，这就走了？"

赵天行。

青云宗少宗主，天灵根，十六岁筑基。
他撑着一把青竹伞，伞面上的青云纹在雨中格外刺眼。

"赵师兄有事？"林凡没有回头。

"没什么，"赵天行走到他身侧，
声音轻得像在分享一个秘密，
"只是想告诉你，你那个空灵剑体的传闻，是我放出去的。"

林凡的手指微微一颤。

空灵剑体——他从未告诉过任何人。
这是母亲临终前告诉他的秘密，
说等他十八岁再开启。

"你怎么知道？"

"你母亲？"赵天行笑了，那笑容温和得像在谈论天气，
"她死前，我父亲去看过她。"

雨声突然变得很响。

林凡转过身，第一次直视赵天行的眼睛。
那双眼睛很漂亮，像两潭深不见底的湖水，
湖底沉着某种让人发冷的东西。

"你父亲……"

"青云宗宗主，赵无极。"赵天行整了整衣袖，
"你母亲是上代剑仙之女，却因爱上凡人被家族除名。
她死前把空灵剑体的秘密告诉了宗主，
希望他能照顾你。"

他顿了顿，歪了歪头：
"宗主确实照顾你了——让你在外门待了三年，
每天看着我这个天灵根，
让你知道什么叫云泥之别。"

林凡没有说话。

他的右手在发抖，不是因为恐惧，
而是因为某种更原始的东西——
那是十六年来第一次，他真正想要杀一个人。

"对了，"赵天行转身，走了两步又停下，
"你丹田里的那道封印，也是宗主亲手下的。
没有这道封印，你三年前就该筑基了。"

他撑开伞，走入雨幕，声音远远传来：
"好好活着，林凡。活着看我能走多远。"

林凡站在雨中，很久很久。

然后，他笑了。

那笑容和之前不同，
像一把刚出鞘的剑，锋利得让人不敢直视。

"赵天行，"他对着空无一人的石阶说，
"我会让你后悔今天没有杀了我。"

他转身，走入雨幕深处。

身后，青云宗的山门在雨中渐渐模糊，
像一幅被水晕开的画。

而林凡不知道的是，
在他踏出青云宗地界的那一刻，
丹田中那道封印，出现了一道裂痕。

一道剑光，正从裂痕中，缓缓苏醒。

---

【第一章完】
```

## Phase 6：审查优化

### 输入
将第一章发给去AI痕迹监督官和质量审计师

### 输出（去AI痕迹监督官）
```
【去AI痕迹审查报告】

总体评分：AI痕迹指数 15/100 ✅（优秀）

亮点：
- 对话有个性，赵天行的温和残忍很立体
- 情绪描写用行为表现（手抖、笑），而非直接陈述
- 结尾悬念自然，有钩子

轻微问题：
1. 第3段"像一头沉默的巨兽"稍常见，可更个人化
2. "那笑容里没有愤怒"可更具体

修改建议：
- 第3段改为："山门在雨里蹲着，像一只淋湿的兽"
- "那笑容里没有愤怒"改为："他笑了，嘴角翘着，眼睛却没弯"
```

### 输出（质量审计师）
```
【质量审计报告】

综合得分：88/100 等级：A（优秀）

分项：
- 剧情逻辑：9/10 → 伏笔自然，冲突清晰
- 节奏把控：9/10 → 信息密度适中，钩子有力
- 冲突设计：9/10 → 多层冲突（个人/宗门/身世）
- 主题深度：8/10 → 复仇与宽恕的铺垫
- 角色立体度：9/10 → 林凡和赵天行都有层次
- 画面感：9/10 → 雨、血、石阶，氛围到位
- 语言风格：9/10 → 符合热血激昂基调
- 类型适配：8/10 → 废柴开局+秘密体质，经典但有效

建议：
- 可增加一个路人视角，强化林凡的孤独感
- 赵天行的动机可再隐晦一些，留到后期揭示
```

## ✅ 完整工作流总结

| 阶段 | 执行者 | 输入 | 输出 | 耗时 |
|------|--------|------|------|------|
| 1.需求解析 | 调度官 | 用户想法 | 任务拆解 | 1轮 |
| 2.世界观 | 世界观师 | 类型+需求 | 世界设定 | 1-2轮 |
| 3.角色 | 角色师 | 世界观+需求 | 角色档案 | 1-2轮 |
| 4.剧情 | 剧情师 | 世界+角色 | 大纲+分卷 | 2-3轮 |
| 5.正文 | 全模块 | 以上全部 | 章节正文 | N轮 |
| 6.审查 | 监督模块 | 正文 | 修改建议 | 每章1轮 |
| 7.交付 | 调度官 | 全部成果 | 最终作品 | 1轮 |

**预计总轮次：世界观2 + 角色2 + 剧情3 + 正文N×2（创作+审查）**
