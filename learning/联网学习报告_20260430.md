## 三、核心理念更新

### ⚠️ 重要理念：技能是工具，不是教条

所有Skill都应该灵活运用，不要死板遵循规则。

**灵活运用四原则：**

1. **组合自由**
   - 可以只用2-3个核心技能完成短篇
   - 可以跳过某些技能直接创作
   - 可以根据需要重复使用某个技能

2. **灵活顺序**
   - 先有灵感？→直接进入剧情构造
   - 先有人物？→先做角色塑造
   - 先有世界观？→先做世界观构造

3. **按需调用**
   - 短篇小说：1-3个技能
   - 中篇小说：3-5个技能
   - 长篇小说：10+个技能

4. **创新组合**
   - 苟道流 + 智斗流 = 新类型
   - 单元剧 + 递进式 = 混合结构
   - 痛点 + 爽点 = 痛爽文
