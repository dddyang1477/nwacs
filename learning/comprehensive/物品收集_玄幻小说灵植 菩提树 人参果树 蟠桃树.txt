### 一、菩提树系（佛道·智慧·悟道·净化）

| **1. 七叶菩提** | 菩提树的本源形态，每片叶子上天然生有七道脉络，对应七情或七种天地法则。传说在树下静坐，可斩断七情，心境通明，极大提升神念修为。 | 主角被困上古幻阵，心魔丛生。危急时刻掏出仅存的一片“七叶菩提”叶，叶片绽放清净佛光，七道脉络流转，将幻象一一瓦解，神念反获突破，窥破阵眼。 | 菩提子、菩提叶、清净琉璃光 |
| **2. 悟道茶** | 以菩提树嫩芽与千年灵泉培炼的仙茶。饮下后能短暂进入“天人合一”的悟道状态，参悟功法、瓶颈突破事半功倍。是顶级宗门招待贵客的无上珍品。 | 宗门大比在即，掌门特意取出珍藏的“悟道茶”赐予核心弟子。主角饮下后，眼前竟浮现出星辰运转，原本晦涩的剑诀瞬间融会贯通，剑意飙升。 | 灵茶、道韵、灌顶、顿悟 |
| **3. 菩提心灯** | 以菩提子为灯芯，大德高僧圆寂前的慈悲愿力为火种，炼成的防御至宝。灯光所及，万邪不侵，能净化冤魂、驱散域外心魔，是魔道功法的绝对克星。 | 魔道大举进攻佛宗圣地，漫天血云遮天蔽日。危急关头，一老僧点燃“菩提心灯”，一念清净，烈焰成海。灯光过处，血云如雪消融，魔道修士神念遭受重创，狼狈溃散。 | 琉璃盏、定风珠、佛光普照 |

| **6. 菩提问心路** | 菩提树对闯入者设下的终极考验，非杀阵，而是幻阵。一条看似无尽的青石小径，两旁菩提林立。每走一步，都会陷入一次“人生重来”的幻境拷问道心。 | 主角为求取菩提本源，踏上“问心路”。他在幻境中历经了富贵、权势、绝顶修为等不同人生，每一次都在最终关头守住本心，选择放弃，最终勘破虚妄，走到了真实的菩提树前。 | 心魔劫、红尘炼心、大道之问 |
| **7. 菩提化龙劫** | (变异设定) 一棵生长在阴阳交汇处的菩提树发生异变，树身如虬龙，不再散发佛光，而是引动九天雷劫。其结出的“雷音菩提子”蕴含毁灭与创生之力，是雷修至宝。 | 主角追寻天地异象，见一株金色巨树并非祥和，反而引下万丈雷暴。无数修士围观，主角凭借雷灵根冒险闯入，在雷龙般的电光中夺得一枚雷音菩提子，借此淬炼出前所未有的雷劫神通。 | 雷劫木、渡劫、阴阳神雷、变异灵植 |

### 二、人参果树系（造化·长生·地灵·木系）

| 词汇/名称 | 详细解释 | 小说运用示例 | 扩展词汇 |
| :--- | :--- | :--- | :--- |
| **8. 地脉灵根·草还丹** | 对人参果树的别称。强调其不仅是树，更是大地祖脉的灵气汇聚点。其果实“草还丹”状如三朝未满的婴儿，四肢俱全，五官皆备，闻一闻能活三百六十岁，吃一个活四万七千年。 | 万寿山五庄观，一场长生盛宴即将开始。主角作为散修受邀观礼，亲眼看到那颗混沌初分时的灵根，枝头挂着二十八个栩栩如生的“婴儿”，周身瑞气升腾，霞光万道，震撼得无以复加。 | 长生果、大地胎膜、乙木精华 |
| **9. 木灵遁法·参果树影** | 人参果树根须覆盖万里，在此范围内，修炼此法者可将自身融入树的影子进行瞬移。只要是有土壤和树木的地方，皆可化为己身，是修真界最顶级的逃命与刺杀神通。 | 追杀主角的敌人将其围得水泄不通，千钧一发之际，主角捏碎一枚人参果树皮符箓，身影融入地面树影，瞬间在原地消失，出现在百里之外的另一片森林中，轻松脱困。 | 缩地成寸、木遁、影遁 |
| **10. 乙木青帝长生体** | 相传为人参果树得道后化形的人族形态。天生亲近万木，能号令天下草木之灵，生命力近乎无限，青春永驻。是无数追求长生者仰望的传说。 | 众人在秘境中遇险，被一株食人魔藤困住。一位青衫少年从林中走出，只是轻抚魔藤，那凶残的植物竟温顺得如宠物般退开。他淡然一笑：“此乃家父所种，惊扰各位了。”全场皆惊，猜测其身份。 | 草木之灵、先天木灵体、青帝传承 |
| **11. 五行灵土·息壤精粹** | 培植人参果树必不可少的土壤，由先天息壤与戊土精华混合而成，具有自我生长和修复能力。拳头大一块，就重达万钧，价值堪比仙器。 | 主角发现一株奄奄一息的人参果树幼苗，拍卖会上，他倾尽所有，力压群雄拍下一小瓶“息壤精粹”。当散发微光的土壤覆盖根系，幼苗瞬间焕发生机，散发出的乙木灵气让他多年的暗伤都好了大半。 | 九天息壤、戊土之精、生命之泉 |
| **11. 金击子** | 专门用于摘取人参果实的法器。此果与五行相畏，遇金而落，遇木而枯，遇水而化，遇火而焦，遇土而入。故必须以金器击落，且用绸帕衬垫磁盘接住，方能确保药性不失。 | 主角窃取人参果，法力触摸无效，刀剑砍落即化为虚无。反复试探后才在古卷中得知五行相克之理，连夜炼化一枚赤金如意，成功“金击”下一枚果实，过程惊心动魄。 | 摘星手、五行克制、玉净瓶、锦斓袈裟 |
| **12. 草还丹婴变** | 人参果成熟落地后，若未被及时收取，会遁入大地，吸收地脉阴气，化为“地精婴灵”。它不再是长生圣物，而是一种精通土遁、表情似笑非笑的诡异精怪，以吸取修士生机为食。 | 探险队在古遗迹中发现一株死去的人参果树，树下散落着几枚灰白色的果实。有队员上前触碰，那果实竟突然睁开空洞的双眼，发出婴儿般笑声，瞬间钻入其体内。片刻后，队员化作干尸，一个更加凝实的“地精婴灵”破体而出。 | 精怪、尸解仙、寻宝鼠、摸金校尉 |
| **13. 大地胎膜·周天护壁** | 以人参果树的落叶和树皮为主材，辅以地心磁母炼制而成的一次性防御至宝。激发后，会形成一个浑圆的、刻满上古地脉图的护罩，可硬抗天劫或大能全力一击，号称大地之母的怀抱。 | 天劫的最后一道神雷带着毁天灭地之势劈下，主角法宝尽毁。他咬牙捏碎了最后的本命底牌——“大地胎膜”。一个厚重的土黄色光罩将他包裹，雷光轰击其上，纹丝不动，光罩上无数山脉虚影流转，将天雷之力导入大地。 | 护身符、玄武甲、结界 |
| **14. 万寿丹宴** | 顶级宗门以人参果为主材，辅以千种灵药，耗时百年炼就的“万寿金丹”的丹方。一枚丹药可增寿一元（十二万九千六百年）。丹方本身，就是足以引发修真界战争的至宝。 | 一场覆盖整个东荒的腥风血雨，起因仅仅是一张“万寿丹宴”残方在黒市出现。主角意外获得一份，却深知怀璧其罪，他巧妙地将残方公布于天下，引得各大势力互相牵制，自己则在乱局中寻找炼制所需的最后几味药材。 | 炼丹术、延寿丹、天材地宝、拍卖会 |

### 三、蟠桃树系（帝道·权柄·品阶·星域）

| 词汇/名称 | 详细解释 | 小说运用示例 | 扩展词汇 |
| :--- | :--- | :--- | :--- |
| **15. 蟠桃品阶体系** | 严格划分：**三千年一熟**，人吃了成仙了道，体健身轻；**六千年一熟**，人吃了霞举飞升，长生不老；**九千年一熟**，人吃了与天地齐寿，日月同庚。紫纹缃核是极品的标志。 | 主角初登天庭，参加蟠桃盛会。仙女手捧的玉盘从身旁走过，他清晰地分辨出：外层仙娥的是果形较小、红晕浅淡的三千年桃；内层捧向帝座、王母的，则是果大如斗、紫纹遍布、缃核透光的九千年极品。一根桃枝，便可见地位之别。 | 仙阶、修仙等级、资质划分、灵果品鉴 |
| **16. 蟠桃化形·西王母侍** | 九千年蟠桃有概率诞生灵智，化形为绝色仙女。她们天生精通木系仙法，气质雍容华贵，是西王母身边的近侍，地位崇高，代行部分王母权柄，被称为“蟠桃仙子”。 | 主角误入瑶池禁地，正欲摘桃，桃树上一团烟霞飘落，化为一位凤冠霞帔的冷艳仙子。她并未立即攻击，而是淡淡说道：“你身上有瑶池旧令的气息，给你三息解释清楚，否则，将你魂魄打入桃核，永做树肥。” | 草木精怪、剑灵、护法神、侍剑者 |
| **17. 蟠桃不死药** | 将一颗九千年蟠桃浓缩、精炼，融入了王母的帝流浆与一丝先天造化法则后，形成的一滴璨若星河的金色液体。可令真灵溃散、魂飞魄散之人重塑肉身与魂魄，从死亡的法则中挣脱，是真正的逆天不死药。 | 为救活道侣，主角冒死盗取天庭至宝。当那一滴“蟠桃不死药”滴入道侣冰冷的眉心，时光仿佛倒流，破碎的魂魄重新凝聚，死寂的肉身焕发生机。道侣悠悠醒来，主角却因触犯天条，被天兵天将团团围住。 | 九转还魂丹、涅槃、复生术、逆天改命 |
| **18. 瑶池封神令** | 蟠桃会是天庭拉拢、封赏、统御三界的重要政治手段。一张由蟠桃木炼制的请柬，代表了体制内的身份——被邀请，意味着你是天庭承认的正神或地仙；不被邀请，实力再强也只是“妖猴”、“散魔”。 | 妖帝寿宴，万妖来朝，气焰熏天。然而当天庭使者手捧蟠桃木炼制的鎏金敕令降临，宣读招安圣旨，并赐下数枚蟠桃时，殿中妖帝的脸色变了。接，则受天庭管制；不接，则可能招来天兵围剿。一枚仙桃，重过万钧。 | 天庭、神职、招安、封神榜、气运之争 |
| **19. 蟠桃核·镇魔钉** | 蟠桃之核坚不可摧，是经过三千年至九千年天地精华淬炼的木中精金。佛道两教常将其打磨成镇魔钉，钉入魔头丹田、四肢，可将其彻底封印，神力日日夜夜消磨其魔性。 | 古寺之下，镇压着一尊上古魔头。魔头被七枚巨大的“蟠桃核”状巨钉钉穿了脊椎、琵琶骨和四肢，悬空固定在镇魔壁上。无数年后，其中一枚镇魔钉已布满裂痕，滚滚魔气从中溢出，预示着一场浩劫。 | 封印术、法器材料、降妖除魔 |
| **20. 先天壬水蟠桃** | 变异品种。不是种在土里，而是生长在弱水之底、由水之精华凝聚的先天壬水蟠桃树上。此树无枝无叶，通体如水波凝结。果实呈幽蓝色，吃下可掌控万水，免疫一切火系法则，是水行道基的无上圣物。 | 主角为修炼《黑帝玄水经》，潜入万丈弱水河底，意外发现一株如梦似幻的蓝色光树。树上结着数颗水波荡漾的蟠桃。他吃下后，血脉中仿佛有一条大河在奔腾，从此水火不侵，更能号令江河，实力暴涨。 | 五行灵根、先天之精、变异灵果 |

### 四、通用设定与体系扩展（跨品类融合）

| 词汇/名称 | 详细解释 | 小说运用示例 | 扩展词汇 |
| :--- | :--- | :--- | :--- |
| **21. 执笔者** | 专门守护、培育天地灵植的神秘传承。他们不修战法，却能利用灵植的能力改变一方天地的气运。执笔者可以“笔”为媒，嫁接生命，让蟠桃枝在人参果树上存活，创造出前所未有的神物。 | 主角的底牌，不是绝世功法，而是一支来自“执笔阁”的判官笔。危急时刻，他以自身精血为墨，快速在空中画出“菩提·醒”三字，周围的菩提树瞬间活了过来，枝条如鞭将心魔贯穿，不仅救了同伴，更揭示了他隐藏的身份。 | 灵植夫、灵纹师、阵法师、神农后裔 |
| **22. 三生树** | 传说中的神话，并非真实存在，而是将菩提树（代表过去）、人参果树（代表现在）、蟠桃树（代表未来）三大神木的一缕本源融合，栽种于时间夹缝中。在此树下修炼，可一梦三生，同时历经三种道法，有几率明悟时间法则。 | 大结局，为了突破最后的时间壁垒，阻止纪元毁灭，主角寻找到天地间唯一的一株“三生树”。他在树下盘坐，左眼映现菩提金光，右眼流淌蟠桃仙华，周身缠绕人参果树的生命气息。最终，他看到了过去、现在、未来的所有可能，斩出了超越时间的一剑。 | 时空法则、轮回、宇宙树、世界之基 |
