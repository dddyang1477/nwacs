# 📚 示例项目：《逆命剑仙》
## 系统协作全流程演示

---

## 项目背景

**用户输入**：
> "我想写一部玄幻仙侠小说，主角是一个被宗门抛弃的废柴，意外获得上古剑仙传承，最终逆袭成为一代剑宗。风格要热血激昂，带点黑暗现实主义，预计80万字。"

---

## 阶段一：需求解析（一级Skill执行）

### 小说总调度官输出

```
【需求解析报告】

项目代号：逆命剑仙
类型：玄幻仙侠（废柴逆袭 + 剑道传承）
风格：热血激昂 + 黑暗现实主义
规模：80万字（约400章）

推荐三级Skill：@玄幻仙侠专家

任务拆解：
1. [P0] 世界观构造 → 修仙世界基础设定
2. [P0] 角色塑造 → 主角 + 核心配角档案
3. [P0] 剧情构造 → 主线大纲 + 分章细纲
4. [P1] 场景构造 → 关键场景设计
5. [P1] 对话设计 → 经典对话脚本
6. [P1] 战斗设计 → 剑道对决设计
7. [P2] 写作技巧 → 风格统一指导

世界观 → 角色（依赖世界观）→ 剧情（依赖世界观+角色）

```

## 阶段二：世界观构造（二级Skill执行）

### 世界观构造师输出

```markdown
# 《逆命剑仙》世界观设定

## 一、核心规则（5条）
1. **剑道唯一**：剑修是同阶最强，但进阶最难，百不存一
   - 影响：剑修稀少但受人敬畏
   - 代价：剑修需以身为剑，寿命短于同阶
   - 冲突：其他修士嫉妒/恐惧剑修

2. **传承断绝**：上古剑仙传承已失传万年
   - 影响：主角获得传承=颠覆现有秩序
   - 代价：传承有诅咒，每代传承者不得善终
   - 冲突：各方势力争夺传承

3. **灵根定命**：灵根品质决定修炼上限
   - 影响：主角被测出废灵根，遭宗门抛弃
   - 代价：灵根可后天改造，但九死一生
   - 冲突：打破阶级固化，触动既得利益

4. **天道有缺**：天道并非完美，存在漏洞
   - 影响：可利用漏洞逆天改命
   - 代价：利用漏洞会遭天谴
   - 冲突：主角选择逆天，与天道为敌

5. **因果轮回**：一切行为皆有因果，迟早要还
   - 影响：主角的逆袭会积累因果债
   - 代价：最终需偿还因果，可能牺牲一切
   - 冲突：力量与代价的永恒博弈

## 二、世界概览
### 2.1 地理版图
- **九域大陆**：分为九大域，每域有霸主宗门
- **主角起点**：青云域，最弱的一域
- **终极目标**：中州域，天道所在

### 2.2 势力格局
| 势力 | 立场 | 与主角关系 |
|------|------|-----------|
| 天剑宗（原宗门） | 中立偏敌 | 抛弃主角，后期后悔 |
| 血魔教 | 敌对 | 觊觎传承 |
| 万剑阁 | 中立 | 剑修圣地，后期盟友 |
| 天道盟 | 表面正义 | 实际维护既得利益 |

## 三、修炼体系
### 境界划分（9境）
| 境界 | 名称 | 特征 | 寿命 |
|------|------|------|------|
| 1 | 炼气 | 引气入体 | 百岁 |
| 2 | 筑基 | 道基初成 | 两百岁 |
| 3 | 金丹 | 金丹大道 | 五百岁 |
| 4 | 元婴 | 元婴出窍 | 千岁 |
| 5 | 化神 | 神识化形 | 三千岁 |
| 6 | 炼虚 | 虚实相生 | 万岁 |
| 7 | 合体 | 天人合一 | 数万岁 |
| 8 | 大乘 | 道法自然 | 近乎永生 |
| 9 | 渡劫 | 破碎虚空 | 飞升 |

### 剑道特殊
- 剑修每境需凝练一柄"本命剑"
- 九柄本命剑合一，方可渡劫
- 主角传承可跳过某些境界，但有代价
```

### 角色塑造师输出

```markdown
# 角色档案：林渊（主角）

## 基础信息
- 姓名：林渊（"林"为木，"渊"为深水，暗示深藏不露）
- 年龄：16岁（开局）→ 300岁（结局）
- 外貌：平凡面容，唯有一双眼睛深邃如渊
- 标志性：左臂有一道剑痕（传承印记）

## 社会身份
- 原身份：天剑宗外门弟子
- 现身份：被逐弟子 → 散修 → 逆命剑宗宗主
- 人际关系：
  - 苏晚晴（红颜）：万剑阁圣女，亦敌亦友
  - 莫老（导师）：上古剑仙残魂，传承者
  - 楚天行（宿敌）：天剑宗天才，嫉妒主角
  - 血魔老祖（大敌）：觊觎传承的魔头

## 心理画像
### 核心欲望：证明自己不是废物
### 核心恐惧：再次被抛弃，再次成为累赘
### 错误信念："只有变强才配被爱"
### 内心矛盾：渴望温暖 vs 习惯孤独
### 秘密：传承中藏着上古剑仙的诅咒
### 创伤：被宗门当众测出废灵根，逐出师门

## 性格特征
- 优点：坚韧、重情义、有原则、悟性极高
- 缺点：偏执、不信任他人、有时过于冷酷
- 口头禅："剑在人在" / "我命由我不由天"
- 决策风格：先谋后动，但一旦决定绝不回头
- 压力反应：越危险越冷静，但会独自承担

## 角色弧线
- 起点：被抛弃的废柴，自卑但倔强
- 催化剂：获得传承，发现废灵根是假的
- 内心挣扎：用力量报复 vs 放下仇恨
- 终点：放下执念，以剑证道，非以剑复仇

## 参考原型
- 性格参考：萧炎（坚韧）+ 李逍遥（情义）
- 弧光参考：令狐冲（从门派到自由）
```

## 阶段四：剧情构造（二级Skill执行）

### 剧情构造师输出

```markdown
# 《逆命剑仙》剧情架构

## 核心设定
- 一句话梗概：废柴少年获上古传承，以剑逆天，最终发现传承是诅咒，选择以己身补天道
- 核心冲突：个体命运 vs 天道规则
- 主题思想：真正的强大不是战胜他人，而是战胜自己
- 情绪基调：热血中带着悲凉，激昂中藏着孤独

## 结构蓝图

### 第一幕：逆命（1-80章，20%）
- **钩子**：开篇即被逐出师门，雨夜独行
- **日常世界**：天剑宗外门，被人欺凌但隐忍
- **触发事件**：被测出废灵根，当众逐出
- **第一转折点**：坠崖获传承，发现废灵根是人为封印

### 第二幕：崛起（81-280章，60%）
- **进入新世界**：散修生涯，独自修炼
- **盟友**：莫老残魂、苏晚晴（初遇为敌）
- **中点反转**：发现封印自己灵根的正是天剑宗掌门
- **至暗时刻**：为救苏晚晴，暴露传承，被全天下追杀

### 第三幕：证道（281-400章，20%）
- **最终准备**：集齐九柄本命剑，直面天道
- **高潮对决**：与天道化身一战，发现天道有缺
- **结局**：选择以己身补天道，非破碎虚空
- **尾声**：苏晚晴继承逆命剑宗，林渊化为天道一部分

## 关键伏笔
| 伏笔 | 埋设 | 回收 | 重要性 |
|------|------|------|--------|
| 废灵根是封印 | 第1章 | 第30章 | ★★★★★ |
| 传承有诅咒 | 第50章 | 第300章 | ★★★★★ |
| 莫老的真实身份 | 第100章 | 第250章 | ★★★★☆ |
| 苏晚晴的宿命 | 第80章 | 第350章 | ★★★★☆ |
```

## 阶段六：对话设计（二级Skill执行）

### 对话设计师输出（示例：莫老传功）

```markdown
# 对话设计：坠崖获传承 - 莫老传功

## 对话目标
- 表面：传授剑道
- 真实：试探主角心性
- 预期：主角展现坚韧，获得认可

## 对话脚本
```
莫老：（虚影从断剑中飘出，声音如金石相击）
"三千年了...终于有人能触发剑冢。小子，你为何而来？"

林渊：（握剑的手在抖，但眼神坚定）
"我为活命而来。"

莫老：（轻笑，带着三分嘲讽）
"活命？多少人求长生，你只求活命？"

林渊：（沉默三秒，雨水顺着下巴滴落）
"...因为我还没活够。有人欠我的，我得亲手讨回来。"

莫老：（虚影凝实一分，目光如剑）
"传承给你，你会成为天下公敌。你可愿？"

林渊：（没有犹豫，单膝跪地）
"公敌又如何？我已经是弃徒了。"

莫老：（仰天大笑，笑声中带着悲凉）
"好！好一个弃徒！三千年前的我，也是弃徒！"
[断剑出鞘，蓝光暴涨]
"从今日起，你便是——逆命剑仙的传人！"
```

## 潜台词分析
- 莫老问"为何而来"：实际在问"你的道心是什么"
- 林渊说"没活够"：表面贪生，实际是不甘
- 莫老的"悲凉"：暗示他自己也是悲剧
```

## 阶段八：写作技巧指导（二级Skill执行）

### 写作技巧大师输出

```markdown
# 《逆命剑仙》风格指南

## 整体文风
- 基调：热血激昂中带着苍凉
- 语言密度：中等（战斗紧凑，抒情舒展）
- 情感表达：外放（主角不藏情绪，但孤独时不说话）

## 句式规范
- 战斗场景：短句为主，每句不超过15字
- 情感场景：长短交错，长句用于内心独白
- 说明场景：克制，不超过3句连续说明

## 词汇偏好
- 高频意象：剑、血、雨、夜、渊、孤
- 禁用词汇："值得注意的是"、"众所周知"、"总而言之"
- 特色词汇：多用古意词汇（"执剑"而非"拿剑"）

## 去AI化重点
- 允许不完美：主角有时会冲动、会说错话
- 具体化：不说"很强的剑气"，说"剑气割裂了三丈青石"
- 个人偏见：叙述者对剑修有偏爱，对法修有微词
```

## 阶段九：去AI痕迹审查（二级Skill执行）

### 去AI痕迹监督官输出

```markdown
# 去AI痕迹审查报告

## 总体评分
- AI痕迹指数：12/100 ✅（优秀）
- 自然度评级：A

## 主要修改
### 原文（第1章）
"林渊被测出废灵根，因此他被逐出了天剑宗。"

### 修改后
"废灵根三个字像三座山，把林渊压出了天剑宗的大门。"

### 修改理由
- 消除因果句式（"因此"）
- 用比喻替代说明
- 加入感官细节

## 其他优化
- 删除3处"值得注意的是"
- 将2处完美对称的排比改为不对称
- 增加4处角色的口误和重复
```
