【风格定位】
┌─────────────────────────────────────────────────────────────┐
│ ✓ 苟道流：不轻易冒险，每步必留后路                         │
│ ✓ 黑暗流：修仙界残酷，人心险恶                             │
│ ✓ 智斗流：靠智慧取胜，而非单纯实力碾压                     │
│ ✓ 热血流：关键时刻爆发，热血沸腾                           │
│ ✓ 悬疑流：谜团层层揭开，抽丝剥茧                           │
│ ✓ 治愈流：铁汉柔情，温馨感情                               │
│ ✓ 搞笑流：轻松幽默，调节气氛                               │
└─────────────────────────────────────────────────────────────┘

【叙事特点】
┌─────────────────────────────────────────────────────────────┐
│ ✓ 多线叙事：主线复仇与多条感情线并行                       │
│ ✓ 伏笔回收：前文埋下的伏笔后文回收                         │
│ ✓ 画面感强：远景→中景→特写                                │
│ ✓ 节奏张弛：紧张战斗与温馨日常交替                         │
│ ✓ 人物立体：每个角色都有自己的人格魅力                     │
│ ✓ 情感细腻：爱情、友情、亲情、恩情交织                     │
└─────────────────────────────────────────────────────────────┘
