## 📝 使用说明

### 快速开始飞书集成（推荐）
1. 打开飞书，创建群聊
2. 添加群机器人，获取Webhook URL
3. 填入 `config/feishu_config.json`
4. 双击 `启动飞书集成.bat` 测试

### DeepSeek学习
运行命令（会自动推送到飞书/微信）：
```bash
python deepseek_learning_engine.py --duration 30m --api-key sk-f3246fbd1eef446e9a11d78efefd9bba
```
