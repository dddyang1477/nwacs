## 🔧 配置步骤（5分钟完成）

### 步骤1：创建飞书机器人

1. 打开飞书 → 创建一个群聊
2. 群设置 → 群机器人 → 添加机器人
3. 选择「自定义机器人」
4. 设置机器人名字和头像，点击「添加」
5. **复制Webhook地址**（很重要！）

### 步骤2：配置NWACS

1. 打开 `config/feishu_config.json`
2. 把刚才复制的Webhook地址填入 `webhook_url`
3. 保存配置

### 步骤3：测试

运行飞书测试脚本，看是否能收到消息！
