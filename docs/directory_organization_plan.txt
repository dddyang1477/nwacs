### 3. 迁移脚本 (PowerShell)

将以下内容保存为 `migrate_nwacs.ps1`，并在NWACS项目的根目录下以管理员身份运行。

```powershell
# migrate_nwacs.ps1
# 用途：重构NWACS项目目录结构
# 注意：请在脚本所在目录（项目根目录）运行

Write-Host "开始迁移NWACS项目结构..." -ForegroundColor Green

# 1. 创建新目录结构
Write-Host "创建新目录..." -ForegroundColor Yellow
$newDirs = @(
    "scripts\bat",
    "scripts\ps1",
    "tests",
    "temp_tests",
    "novels\缄默天师"
)
foreach ($dir in $newDirs) {
    if (-not (Test-Path $dir)) {
        New-Item -ItemType Directory -Path $dir -Force | Out-Null
        Write-Host "  创建目录: $dir" -ForegroundColor Gray
    }
}

# 2. 移动配置文件
Write-Host "移动配置文件..." -ForegroundColor Yellow
Move-Item -Path ".\config.json" -Destination ".\core\config.json" -Force -ErrorAction SilentlyContinue

# 3. 移动核心文件到 core/
Write-Host "移动核心文件到 core/..." -ForegroundColor Yellow
$coreFiles = @("main.py", "nwacs_main.py", "server_simple.py")
foreach ($file in $coreFiles) {
    if (Test-Path ".\$file") {
        Move-Item -Path ".\$file" -Destination ".\core\$file" -Force
        Write-Host "  移动: $file -> core/"
    }
}

# 4. 移动技能文件到 skills/
Write-Host "移动技能文件到 skills/..." -ForegroundColor Yellow
$skillFiles = Get-ChildItem -Path ".\" -File | Where-Object { $_.Name -match "^(deepseek|feishu|simple_learning|specialized_learning)\.py$" }
foreach ($file in $skillFiles) {
    Move-Item -Path $file.FullName -Destination ".\skills\$($file.Name)" -Force
    Write-Host "  移动: $($file.Name) -> skills/"
}

# 5. 移动文档到 docs/
Write-Host "移动文档到 docs/..." -ForegroundColor Yellow
$docFiles = Get-ChildItem -Path ".\" -File | Where-Object { $_.Extension -eq ".md" }
foreach ($file in $docFiles) {
    Move-Item -Path $file.FullName -Destination ".\docs\$($file.Name)" -Force
    Write-Host "  移动: $($file.Name) -> docs/"
}

# 6. 移动批处理文件到 scripts/bat/
Write-Host "移动批处理文件到 scripts/bat/..." -ForegroundColor Yellow
$batFiles = Get-ChildItem -Path ".\" -File | Where-Object { $_.Extension -eq ".bat" }
foreach ($file in $batFiles) {
    Move-Item -Path $file.FullName -Destination ".\scripts\bat\$($file.Name)" -Force
    Write-Host "  移动: $($file.Name) -> scripts/bat/"
}

# 7. 移动PowerShell脚本到 scripts/ps1/
Write-Host "移动PowerShell脚本到 scripts/ps1/..." -ForegroundColor Yellow
$ps1Files = Get-ChildItem -Path ".\" -File | Where-Object { $_.Extension -eq ".ps1" }
foreach ($file in $ps1Files) {
    Move-Item -Path $file.FullName -Destination ".\scripts\ps1\$($file.Name)" -Force
    Write-Host "  移动: $($file.Name) -> scripts/ps1/"
}

# 8. 移动测试文件到 tests/
Write-Host "移动测试文件到 tests/..." -ForegroundColor Yellow
$testPattern = @("test_feishu", "test_output", "test_port", "test_python_env", "test_server",
                 "simple_test", "simple_evaluator", "simple_feishu_test",
                 "enterprise_comparison", "manual_feishu_test", "quick_check",
                 "quick_feishu_test", "quick_learning_test", "run_learning_quick",
                 "run_learning_test", "feishu_auto_demo", "feishu_command_server",
                 "feishu_debug", "feishu_final_test", "feishu_simple_test",
                 "feishu_test_send", "setup_feishu")
foreach ($pattern in $testPattern) {
    $testFile = ".\$pattern.py"
    if (Test-Path $testFile) {
        Move-Item -Path $testFile -Destination ".\tests\$pattern.py" -Force
        Write-Host "  移动: $pattern.py -> tests/"
    }
}

# 9. 移动临时测试文件到 temp_tests/
Write-Host "移动临时测试文件到 temp_tests/..." -ForegroundColor Yellow
$tempTestFiles = @("deepseek_classics_learning.py", "start_auto_learn.py")
foreach ($file in $tempTestFiles) {
    if (Test-Path ".\$file") {
        Move-Item -Path ".\$file" -Destination ".\temp_tests\$file" -Force
        Write-Host "  移动: $file -> temp_tests/"
    }
}

# 10. 移动小说相关文件
Write-Host "移动小说相关文件..." -ForegroundColor Yellow
$novelFiles = Get-ChildItem -Path ".\" -File | Where-Object { $_.Name -match "小说|大纲" }
foreach ($file in $novelFiles) {
    $novelName = "缄默天师"  # 根据实际情况调整
    Move-Item -Path $file.FullName -Destination ".\novels\$novelName\$($file.Name)" -Force
    Write-Host "  移动: $($file.Name) -> novels/$novelName/"
}

# 11. 移动工具/报告文件
Write-Host "移动工具文件到 tools/..." -ForegroundColor Yellow
$toolFiles = @("deepseek_optimization_report.txt")
foreach ($file in $toolFiles) {
    if (Test-Path ".\$file") {
        Move-Item -Path ".\$file" -Destination ".\tools\$file" -Force
        Write-Host "  移动: $file -> tools/"
    }
}

Write-Host "迁移完成！" -ForegroundColor Green
Write-Host "请检查是否有遗漏的文件，并更新相关导入路径。" -ForegroundColor Cyan
```

### 4. 注意事项和验证步骤

#### 注意事项
1.  **备份先行**：在执行迁移脚本之前，建议对整个项目进行一次完整备份（可以使用 `git stash` 或复制整个文件夹）。
2.  **路径依赖**：迁移后，所有脚本中硬编码的文件路径（如 `open('config.json')`）都需要更新。例如，`config.json` 现在位于 `core/config.json`。
3.  **模块导入**：如果Python文件之间有 `import` 关系（例如 `from deepseek_learning_engine import ...`），迁移后需要检查导入路径是否正确。可能需要更新为 `from skills.deepseek_learning_engine import ...`。
4.  **`.gitignore` 更新**：检查 `.gitignore` 文件，确保 `temp_tests/`、`novels/` 等新目录的忽略规则是正确的。`novels/` 通常不需要被Git跟踪，除非你想版本控制小说内容。
5.  **脚本执行权限**：PowerShell脚本可能需要先执行 `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass` 才能运行。
6.  **手动检查**：脚本可能无法覆盖所有边界情况（如文件名包含特殊字符）。迁移后，请手动检查根目录，确认所有文件都已正确归类。

#### 验证步骤
1.  **完整性检查**：
    ```powershell
    # 列出根目录下所有非目录文件，确认数量大幅减少
    Get-ChildItem -Path ".\" -File | Select-Object Name
    ```
    理想情况下，根目录应该只剩下 `.gitignore`、`README.md` 等极少数文件。

2.  **路径更新检查**：
    ```bash
    # 在Linux/WSL或Git Bash中，搜索所有Python文件中对旧路径的引用
    grep -rn "
