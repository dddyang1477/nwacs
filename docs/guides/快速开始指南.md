## 🎯 5分钟快速上手

### 第一步：双击启动

找到并双击 `启动NWACS.bat`

你将看到：
```
🎭 NWACS v7.0 统一启动器
小说创作AI协作系统

请选择功能:
  1. 小说创作引擎（交互式）
  2. 快速创作（输入即创作）
  ...
```

### 第二步：开始创作

输入 `1` 按回车，进入小说创作引擎

### 第三步：选择类型

系统会列出8种小说类型：
```
1. 🐉 玄幻仙侠 - 修仙、玄幻、仙侠
2. 🏙️ 都市言情 - 都市、言情、职场
3. 🔍 悬疑推理 - 悬疑、推理、侦探
4. 🚀 科幻未来 - 科幻、未来、末世
5. 📜 历史穿越 - 历史、穿越、架空
6. 👻 恐怖惊悚 - 恐怖、惊悚、灵异
7. 🎮 游戏竞技 - 游戏、竞技、电竞
8. 💖 女频系列 - 总裁、年代、马甲、萌宝
```

输入数字选择，比如 `1`

### 第四步：输入需求

系统会提示：
```
请输入创作需求:
```

输入你的创作想法，比如：
```
写一个废柴逆袭的玄幻开篇，主角被未婚妻退婚
```

### 第五步：生成章节

系统会自动生成3000字左右的章节！

## 📖 常见使用场景

### 场景1：想快速写一个开头

1. 双击 `启动NWACS.bat`
2. 输入 `2` 选择"快速创作"
3. 输入创作需求
4. 选择小说类型
5. 等待生成完成

### 场景2：想精修一篇文章

1. 双击 `启动NWACS.bat`
2. 输入 `5` 选择"AI痕迹去除"
3. 粘贴你的文章
4. 系统自动优化

### 场景3：检查文章质量

1. 双击 `启动NWACS.bat`
2. 输入 `4` 选择"质量检查"
3. 粘贴你的文章
4. 查看评分和优化建议

### 场景4：配置飞书通知

1. 打开飞书，创建一个群
2. 群设置 → 添加群机器人 → 自定义机器人
3. 复制Webhook地址
4. 打开 `config/feishu_config.json`
5. 粘贴Webhook地址，保存
6. 双击 `启动飞书集成.bat` 测试

## 🔧 故障排除

### 问题1：启动后报错

**可能原因**: Python环境问题
**解决方法**:
1. 确保已安装Python 3.8+
2. 确保已安装openai库：`pip install openai`

### 问题2：生成失败

**可能原因**: API Key问题
**解决方法**:
1. 检查API Key是否正确
2. 检查网络连接
3. 查看错误提示信息

### 问题3：飞书收不到消息

**可能原因**: Webhook配置问题
**解决方法**:
1. 确认Webhook地址正确
2. 确认飞书机器人已在群里
3. 测试连接：`启动飞书集成.bat` → 选择测试
