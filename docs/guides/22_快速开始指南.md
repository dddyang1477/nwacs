# ⚡ 快速开始指南
## 5分钟上手小说写作AI协作系统

---

## 第一步：一句话启动

在 Trae CN 中，直接对 **小说总调度官** 说：

> **"我想写一部[类型]小说，题材是[题材]，风格要[风格]，预计[字数]。"**

**示例**：
> "我想写一部玄幻仙侠小说，题材是废柴逆袭，风格要热血激昂，预计100万字。"

---

## 第二步：让系统工作

总调度官会自动：
1. ✅ 解析你的需求
2. ✅ 推荐对应的三级Skill（类型专家）
3. ✅ 拆解任务并调度二级Skill
4. ✅ 监督模块自动审查
5. ✅ 输出完整方案

**你不需要手动调用任何Skill**，总调度官会帮你搞定一切！

---

## 第三步：迭代优化

拿到初稿后，你可以：

### 调整方向
> "世界观太大了，缩小到一个小宗门内部的故事"

### 深化角色
> "@角色塑造师 给主角加一个隐藏身份"

### 优化文笔
> "@写作技巧大师 这段太AI了，帮我改得更有烟火气"

### 检查质量
> "@质量审计师 给这一章打个分"

### 消除AI痕迹
> "@去AI痕迹监督官 全文审查一遍"

## 典型工作流示例

### 场景1：从零开始写小说

```
用户：新建一个都市言情项目，霸道总裁题材，甜宠风格

总调度官：
  → 激活 @都市言情专家
  → 调度 @世界观构造师 → 输出现代都市背景
  → 调度 @角色塑造师 → 输出霸总+独立女主档案
  → 调度 @剧情构造师 → 输出先婚后爱大纲
  → 调度 @场景构造师 → 输出关键场景
  → 调度 @对话设计师 → 输出经典对话
  → 调度 @去AI痕迹监督官 → 审查
  → 调度 @质量审计师 → 评分
  → 输出：完整项目方案
```

### 场景2：修改已有内容

```
用户：第三章的战斗场面太平淡了，帮我改得激烈一点

总调度官：
  → 调取第三章内容
  → 调度 @战斗设计师 → 重新设计战斗流程
  → 调度 @场景构造师 → 增强感官冲击
  → 调度 @写作技巧大师 → 优化文笔节奏
  → 调度 @去AI痕迹监督官 → 审查
  → 输出：修改后的第三章
```

### 场景3：类型咨询

```
用户：我想在悬疑小说里加一个时间循环的元素，可行吗？

总调度官：
  → 调度 @悬疑推理专家 → 分析可行性
  → 调度 @科幻未来专家 → 提供时间循环设定
  → 输出：可行性报告 + 设定建议
```
