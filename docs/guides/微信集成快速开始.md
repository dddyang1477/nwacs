## 🚀 快速使用

### 方式一：交互式演示（推荐先试）
双击运行：`启动微信集成.bat`

选择 `1` 进入交互式演示模式！

### 方式二：配置企业微信机器人（正式使用）

1. **在企业微信群添加机器人**
   - 企业微信群 → 群设置 → 添加群机器人
   - 获取Webhook地址

2. **配置NWACS**
   - 打开 `config/wechat_config.json`
   - 在 `qiyewechat.webhook_url` 中填入你的Webhook

3. **测试推送**
   - 运行DeepSeek学习引擎
   - 完成后会自动推送到微信！

## 🔧 配置说明

### wechat_config.json
```json
{
  "enable": true,
  "mode": "qiyewechat",
  "qiyewechat": {
    "webhook_url": "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=your-key-here",
    "secret": "",
    "chat_id": ""
  },
  "personal": {
    "auto_login": true
  },
  "push_events": [
    "task_start",
    "task_progress",
    "task_complete",
    "learning_complete",
    "quality_warning"
  ]
}
```

### 关键配置项
- **webhook_url**：企业微信机器人Webhook地址（必填）
- **enable**：是否启用微信推送
- **push_events**：需要推送的事件类型

## 🎯 使用示例

### 示例1：DeepSeek学习后自动推送

```bash
python deepseek_learning_engine.py --duration 30m --api-key sk-xxx
```

学习完成后会自动推送到企业微信群！

### 示例2：禁用微信推送
```bash
python deepseek_learning_engine.py --duration 4h --disable-wechat --api-key sk-xxx
```

## 💡 常见问题

### Q：为什么不用个人微信？
A：个人微信有封号风险，企业微信机器人官方支持、稳定、不封号。

### Q：没有企业微信怎么办？
A：可以先使用概念验证模式（交互式演示），或者用钉钉机器人、飞书webhook替代。

### Q：推送失败怎么办？
A：检查 webhook_url 是否正确，网络是否畅通。
