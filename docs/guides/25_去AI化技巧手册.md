# 🕵️ 去AI化技巧手册
## 从"机器味"到"人味"的完整指南

> 本手册是去AI痕迹监督官的详细操作指南，
> 也是所有创作Skill必须遵循的核心原则。

---

## 一、AI写作的"指纹"：常见痕迹识别

### 1.1 语言层面的AI痕迹

#### 🔴 过度结构化
**AI表现**：
```
首先，我们需要明确问题的核心。
其次，分析问题的各个方面。
最后，得出综合结论。
```

**人类写法**：
```
问题核心是什么？
——是这个。

那其他方面呢？
——也都指向同一个答案。
```

**修正策略**：
- 打破"首先/其次/最后"的固定结构
- 用问答、对话、内心独白替代线性叙述
- 允许逻辑跳跃，不要每一步都铺垫

---

#### 🔴 抽象空洞
**AI表现**：
```
他感到深刻的悲伤。
这是一个重要的决定。
她经历了有意义的成长。
```

**人类写法**：
```
他盯着手机屏幕看了很久，
拇指在"发送"按钮上来回摩挲，
最后按了锁屏键。

她把那张照片从钱包里抽出来，
看了三秒，塞进了抽屉最深处。
```

**修正策略**：
- 所有抽象词必须有具体对应
- "悲伤"→具体动作/表情/生理反应
- "重要"→具体后果/代价/风险
- "有意义"→具体改变/选择/失去

---

#### 🔴 完美对称
**AI表现**：
```
风是冷的，雨是凉的，
心是空的，路是长的。
```

**人类写法**：
```
风很冷，雨倒不大，
就是心空得慌。
路还长——长得让人不想走。
```

**修正策略**：
- 排比不要工整，长短错落
- 加入口语化的连接词（"倒""就是""——"）
- 允许不对称，制造自然感

---

#### 🔴 过度解释
**AI表现**：
```
他之所以这样做，
是因为童年时父亲离开了他，
这导致他对亲密关系有恐惧，
因此他选择推开所有关心他的人。
```

**人类写法**：
```
她递过来一杯水。
他没接。
"我不渴。"他说，
眼睛看着窗外。
```

**修正策略**：
- 相信读者，不要解释动机
- 用行为表现心理，而非直接陈述
- 允许读者有自己的解读空间

---

#### 🔴 中立无立场
**AI表现**：
```
双方的观点都有一定的道理，
需要从多个角度进行综合分析。
```

**人类写法**：
```
他说得对个屁。
——至少当时我是这么想的。
```

**修正策略**：
- 加入个人偏见和主观判断
- 允许角色有"不讲理"的时刻
- 用情绪化的词汇替代中性词

---

#### 🔴 AI高频词汇黑名单

| AI高频词 | 人类替代 | 使用场景 |
|----------|----------|----------|
| 值得注意的是 | 说来也怪 / 有意思的是 | 引出意外发现 |
| 需要指出的是 | 说白了 / 其实 | 解释说明 |
| 综上所述 | （直接结论） | 总结 |
| 从某种意义上说 | （删掉） | 模糊表达 |
| 不可否认的是 | 确实 / 不得不承认 | 承认事实 |
| 归根结底 | 说到底 / 根子上 | 追溯本质 |
| 显而易见 | 明摆着 / 瞎子都看得出 | 强调明显 |
| 毋庸置疑 | 这还用说？ | 强调确定 |
| 与此同时 | 那会儿 / 正当...的时候 | 时间过渡 |
| 然而 | 可 / 但是 / 不过 | 转折 |
| 因此 | 所以 / 这下 / 结果 | 因果 |
| 此外 | 还有 / 另外 / 再说 | 补充 |
| 例如 | 比如 / 像...那样 | 举例 |
| 换句话说 | 说白了就是 / 也就是 | 解释 |
| 事实上 | 其实 / 说真的 | 强调真实 |

### 1.2 结构层面的AI痕迹

#### 🔴 段落均匀
**AI表现**：每段5-7行，长度几乎一致

**修正策略**：
- 短段（1-2行）制造紧张/强调
- 长段（10行+）用于抒情/说明
- 极端短段（几个字）制造冲击

#### 🔴 逻辑过于清晰
**AI表现**：每一步都有铺垫，转折都有预告

**修正策略**：
- 允许意外发生，不要预告
- 角色可以做出"不符合逻辑"的选择
- 情节可以有跳跃，不要填满所有缝隙

#### 🔴 情绪平滑
**AI表现**：情绪变化有明确的原因和过程

**修正策略**：
- 允许情绪突然爆发
- 允许角色"莫名其妙"地生气/哭泣
- 允许情绪反复，不要单向推进

---

### 1.3 认知层面的AI痕迹

#### 🔴 知识堆砌
**AI表现**：展示知识而非服务故事

**修正策略**：
- 知识必须推动剧情或揭示角色
- 不要为展示而展示
- 用角色的无知制造张力

#### 🔴 缺乏盲点
**AI表现**：角色什么都知道，什么都能分析

**修正策略**：
- 角色有不知道的事
- 角色会误解、会猜错
- 角色会被情绪蒙蔽判断

#### 🔴 道德完美
**AI表现**：主角从不犯错，选择永远正确

**修正策略**：
- 主角有道德灰色地带
- 主角会为了私利伤害他人
- 主角会后悔、会愧疚

## 二、注入"人味"：核心策略

### 2.1 不完美策略

#### 口误与重复
```
AI："我们必须立即采取行动。"
人类："我们得——得马上动，动起来。"
```

#### 半截话
```
AI："我想说的是，这个决定很重要。"
人类："我想说的是——算了，说了也没用。"
```

#### 逻辑跳跃
```
AI："他分析了三种方案，比较了优劣，最后选择了B。"
人类："A太麻烦，C太贵，那就B吧。"
```

#### 情绪失控
```
AI："他感到愤怒，但努力保持冷静。"
人类："他一把掀了桌子。"
```

---

### 2.2 具体化策略

#### 抽象→具体对照表

| 抽象概念 | 具体化写法 |
|----------|-----------|
| 他很悲伤 | 他盯着空杯子，看了很久 |
| 她很高兴 | 她笑的时候露出了虎牙 |
| 天气很热 | 柏油路软得像橡皮泥 |
| 房间很乱 | 地上有三只不成对的袜子 |
| 时间很长 | 他数了十七只蚂蚁爬过窗台 |
| 关系很好 | 她知道他喝咖啡不加糖 |
| 他很紧张 | 他把纸巾撕成了细条 |
| 她很害怕 | 她把门锁检查了三遍 |

---

### 2.3 个人化策略

#### 加入偏见
```
AI："这座城市有着悠久的历史和丰富的文化。"
人类："这破地方，除了老建筑啥也没有，
      但奇怪的是，待久了居然不想走。"
```

#### 加入习惯
```
AI："他每天早上都会跑步。"
人类："他跑步有个毛病，必须听同一首歌，
      播到第三遍的时候正好跑完。"
```

#### 加入记忆
```
AI："这家餐厅的食物很美味。"
人类："这家的红烧肉，和我奶奶做的一个味——
      甜得发腻，但我每次都想点。"
```

---

### 2.4 留白策略

#### 不说透
```
AI："她离开是因为不爱了，
      三年的感情在现实面前不堪一击。"
人类："她走的时候没回头。
      门关上的时候，
      他听见钥匙落在鞋柜上的声音。"
```

#### 不解释
```
AI："他之所以笑，是因为想起了童年的趣事。"
人类："他突然笑了。
      没人问他为什么。"
```

#### 不总结
```
AI："这次经历让他明白了珍惜眼前人的道理。"
人类：「章节结束，没有总结」
```

## 三、分场景去AI化技巧

### 3.1 对话场景

**AI腔**：
```
"我认为我们应该慎重考虑这个决定，
因为它将对我们的未来产生深远的影响。"
```

**人味版**：
```
"你想好了？"
"……没有。"
"那就别想了，做吧。"
"做了回不了头。"
"那就不回。"
```

**技巧**：
- 用短句，像真人说话
- 加入停顿、犹豫、打断
- 不说完整句子
- 用动作替代部分对话

---

### 3.2 描写场景

**AI腔**：
```
夕阳西下，天空被染成了橙红色，
大地笼罩在一片温暖的光辉中。
```

**人味版**：
```
太阳快落山了，天是脏兮兮的橙，
像谁打翻了颜料盘。
光斜着切过来，把人的影子拉得老长，
长得不像人。
```

**技巧**：
- 用不完美的比喻
- 加入个人感受（脏兮兮、不像人）
- 具体化（颜料盘、老长）

---

### 3.3 内心独白

**AI腔**：
```
他陷入了沉思，思考着人生的意义和未来的方向。
他意识到，只有坚持自己的梦想，才能找到真正的幸福。
```

**人味版**：
```
他在想，如果当初选了另一条路，
现在会在哪儿。

想不出来。

算了，不想了。
```

**技巧**：
- 碎片化，不连贯
- 有跳跃，有放弃
- 没有结论

---

### 3.4 战斗场景

**AI腔**：
```
他挥剑斩向敌人，剑光如虹，气势如虹。
敌人猝不及防，被一剑击中，倒飞而出。
```

**人味版**：
```
他挥剑。
剑很重，比想象中重。
敌人的刀已经到眼前了——
他来不及想，只能把剑往上顶。

金属相撞的声音像有人在他耳边敲钟。
震得他虎口发麻。
```

**技巧**：
- 加入身体感受（重、麻）
- 加入意外（比想象中重）
- 加入感官细节（敲钟）
- 不要流畅，要卡顿

## 四、AI痕迹检测清单

### 快速自检（每章完成后）

- [ ] 没有"首先/其次/最后/综上所述"
- [ ] 没有"值得注意的是/需要指出的是"
- [ ] 没有过于工整的排比
- [ ] 段落长度不均匀
- [ ] 有具体细节，没有纯抽象描述
- [ ] 角色对话有个性差异
- [ ] 有角色的不理智时刻
- [ ] 有留白，没有解释一切
- [ ] 有情绪爆发，不是平滑过渡
- [ ] 有个人偏见，不是中立客观
- [ ] 有口误/重复/半截话
- [ ] 有逻辑跳跃，不是步步铺垫

### 深度自检（每卷完成后）

- [ ] 主角有犯错、有后悔
- [ ] 反派有逻辑、有动机
- [ ] 配角不是工具人
- [ ] 情节有意外，不是全在预料中
- [ ] 情感有反复，不是单向推进
- [ ] 世界观有矛盾（真实的复杂）
- [ ] 角色有不知道的事
- [ ] 有未回收的伏笔（真实的不完美）

## 五、进阶技巧：风格化去AI化

### 5.1 口语化注入
在叙述中加入口语化表达：
- "说白了""说白了""说白了"
- "其实吧""其实吧""其实吧"
- "怎么说呢""怎么说呢""怎么说呢"
- "反正""反正""反正"

### 5.2 方言/地域特色
根据故事背景加入方言元素：
- 北方："咋整""得嘞""甭"
- 南方："侬""做啥""蛮好"
- 江湖："老子""小子""爷"

### 5.3 时代感
根据故事时代加入时代特色词汇：
- 古代："诺""然""汝"
- 民国："先生""小姐""劳驾"
- 现代："卧槽""绝了""离谱"
- 未来：创造新词汇

### 5.4 个人化标记
为叙述者/主角设计独特的语言标记：
- 喜欢用反问句
- 喜欢用破折号
- 喜欢自言自语
- 喜欢用特定比喻

## 六、常见误区

### ❌ 误区1：为了去AI而刻意粗糙
**错误**：故意写错别字、语法错误
**正确**：保持语言质量，但注入不完美的人性

### ❌ 误区2：完全抛弃结构
**错误**：彻底混乱，读者看不懂
**正确**：有内在逻辑，但表面有跳跃和意外

### ❌ 误区3：所有角色都口语化
**错误**：贵族像流氓，学者像混混
**正确**：口语化要符合角色身份

### ❌ 误区4：过度使用方言
**错误**：全文方言，阅读困难
**正确**：点缀使用，增强质感

---

> 🎯 **核心原则**：
> 去AI化不是降低质量，而是提升真实感。
> AI追求完美，人类拥抱不完美。
> 让读者感受到：这是一个真实的人在讲故事。
