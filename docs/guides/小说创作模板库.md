## 🐉 玄幻仙侠模板

### 1. 废柴逆袭模板

**类型**: 玄幻仙侠
**适合**: 喜欢看主角从弱变强的读者

**开篇钩子**:
- 退婚流：未婚妻当众退婚
- 资质测试：被判定为最差资质
- 家族大会：被羞辱嘲讽

**核心爽点**:
- 打脸轻视自己的人
- 获得奇遇/传承
- 修为快速提升
- 越级挑战胜利

**常用元素**:
- 戒指老爷爷
- 神秘传承
- 炼丹/炼器
- 宗门势力

**示例开头**:
```
青云宗大殿，十年一度的资质测试正在进行。

"林逸，金属性，资质...下下品！"

话音落下，全场哗然。曾经的天才，如今的废物，十二岁的年纪，却只有练气一层的修为。

大厅角落，未婚妻苏雪儿的眼神中闪过一丝不易察觉的失望。

"林逸哥哥..."她轻声呢喃，"我们...还是算了吧。"

就在所有人以为林逸会崩溃的时候，他却笑了。

"退婚？"林逸淡淡开口，"正合我意。"
```

## 🏙️ 都市言情模板

### 1. 总裁隐婚模板

**类型**: 都市言情
**适合**: 女频读者

**开篇钩子**:
- 契约结婚
- 醉酒误事

- 女配各种作死被打脸
- 身份揭秘时的震撼

**示例开头**:
```
"这是离婚协议，签了吧。"

顾寒深将一份文件扔在苏念面前，眼神淡漠。

三年的隐婚，他连看她一眼都觉得浪费时间。

苏念低头，协议上的字清清楚楚：三套房产，一千万现金，公司股份5%。

她苦笑，原来三年的婚姻，在他眼里就值这些。

"我不要这些。"她拿起笔，签下自己的名字，"我只要——自由。"

顾寒深皱眉，心中莫名一紧。

"苏念，你——"

话没说完，她已经转身离开，只留下一个决绝的背影。

他不知道的是，三年前那个雨夜，她为什么会出现在他的房间。

更不知道，她肚子里的孩子，现在已经三岁了。
```

## 🔍 悬疑推理模板

### 1. 推理探案模板

**类型**: 悬疑推理

**开篇钩子**:

**核心爽点**:
- 细节推理
- 反转再反转
- 揭开惊人真相
- 凶手身份震撼

**示例开头**:
```
"死者，男，三十五岁，某上市公司董事长。"

"死亡时间：昨晚十点到十二点之间。"

"死因：心脏被锐器刺穿，当场死亡。"

我蹲在尸体旁边，目光扫过整个房间。

门窗紧锁，没有撬动痕迹。

密室杀人。

"监控呢？"我站起身。

"坏了，"警官苦笑，"正好在死者死亡时间段。"

我点点头，目光落在死者的右手上。

他的右手，紧紧攥着一样东西。

那是一张扑克牌。

黑桃A。

"有意思。"我嘴角微微上扬。

"凶手故意留下了线索。"

"他在挑衅。"
```

## 📝 模板使用说明

### 如何使用模板

1. **选择类型**: 根据你想写的题材选择对应模板
2. **理解爽点**: 明确读者想看的核心爽点
3. **调整元素**: 根据需要调整模板中的元素
4. **写出开头**: 按照模板的开篇钩子写出你的开头
5. **持续推进**: 按照核心爽点持续推进剧情

### 模板只是起点

- 模板帮助你快速开始
- 真正的创作在于你自己的故事
- 大胆修改模板，形成你的独特风格
- 最重要的是：开始写！
