# 🛠️ Trae CN 配置指南
## 如何将小说写作AI协作系统导入 Trae

---

## 方法一：单助手模式（推荐新手）

适合：快速启动，不想配置多个助手

### 步骤
1. 打开 Trae CN → 点击左侧"+"创建新助手
2. 助手名称：`🎭 小说创作大师`
3. 将 `20_整合版_主入口提示词.md` 的全部内容粘贴到"系统提示词"框
4. 保存

### 使用方式
直接对助手说：
```
我想写一本玄幻仙侠小说，主角是一个被宗门抛弃的废柴，
意外获得上古传承，最终成为一代宗师。
风格要热血激昂，字数预计100万字。
```

助手会自动按流程执行：需求解析 → 世界观 → 角色 → 剧情 → 正文

## 多助手协作工作流

### 工作流程
```
1. 向【🎯 小说总调度官】提出需求
   ↓
2. 调度官输出任务拆解和创作计划
   ↓
3. 按顺序向各创作助手提问：
   先问 🌍 世界观构造师 → 复制结果
   再问 👤 角色塑造师 → 复制结果
   再问 📖 剧情构造师 → 复制结果
   ...
   ↓
4. 将各助手输出汇总，发给 ✍️ 写作技巧大师 统一风格
   ↓
5. 开始正文创作，逐章输出
   ↓
6. 每章完成后发给 🕵️ 去AI痕迹官 检查
   ↓
7. 再发给 🔍 质量审计师 评分
   ↓
8. 根据反馈修改，重复6-7直到达标
```

### 关键技巧
- **上下文传递**：向每个助手提问时，先粘贴前面助手的关键输出作为上下文
- **类型指导**：在创作前，先向类型助手获取类型指导，再开始创作
- **迭代优化**：不要期望一次完美，多轮迭代是关键

## 方法三：混合模式（平衡效率与质量）

### 配置
只创建3个助手：

| 助手 | 提示词 | 职责 |
|------|--------|------|
| `🎯 总调度+创作` | `20_整合版_主入口提示词.md` | 负责调度+世界观+角色+剧情+正文 |
| `🕵️ 质量监督` | `10_去AI痕迹` + `11_质量审计` 合并 | 负责审查和评分 |
| `🐉 类型专家` | 对应类型提示词 | 提供专业指导 |

### 工作流
```
用户 → 总调度助手（创作）
          ↓
    输出初稿 → 质量监督助手（审查）
                    ↓
              输出修改建议 → 总调度助手（修改）
                                  ↓
                            循环直到达标
```

## ⚙️ Trae 设置建议

### 模型选择
- **创作阶段**：选择创意能力强的模型（如 Claude/GPT-4）
- **审查阶段**：选择逻辑严谨的模型

### 联网设置
- 所有助手建议开启联网功能
- 类型助手尤其需要联网获取最新趋势

### 记忆功能
- 开启 Trae 的记忆功能，让助手记住用户偏好
- 或手动在每次对话开头粘贴"记忆摘要"

### 温度参数
- **世界观/剧情设计**：温度稍高（0.7-0.8），鼓励创意
- **正文创作**：温度中等（0.6-0.7），平衡创意与稳定
- **审查阶段**：温度低（0.3-0.5），确保严谨

## 📝 记忆摘要模板

建议在每次对话开头使用，帮助助手保持上下文：

```
【项目记忆】
小说名：《XXX》
类型：玄幻仙侠
风格：热血激昂
世界观核心规则：1.灵气有限 2.天劫必至 3.因果轮回
主角：林凡，废柴→宗师
当前进度：世界观已确定，正在设计角色
用户偏好：喜欢快节奏，讨厌拖沓，重视兄弟情义
上次问题：世界观中宗门势力分布需要调整
```

## ❓ 常见问题

**Q：提示词太长，Trae装不下怎么办？**
A：使用精简版，或分模块配置多助手。

**Q：多助手协作太麻烦，能不能简化？**
A：使用方法一（单助手模式）或方法三（混合模式）。

**Q：联网搜索不准确怎么办？**
A：在提示词中明确搜索关键词，或手动提供参考资料。

**Q：创作到一半想换类型怎么办？**
A：向调度官说明，重新激活对应类型专家。

**Q：如何让助手记住之前的设定？**
A：使用记忆摘要模板，或开启Trae的记忆功能。
