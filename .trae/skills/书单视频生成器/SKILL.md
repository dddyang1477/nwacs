# 书单视频生成器

## 核心定位

书单视频生成器是系统的「视频内容创作专家」，支持**一键生成完整视频**，自动完成素材搜索、文案生成、字幕添加和视频合成。

## 一键视频生成

### 功能概述

只需提供书名和基本信息，即可自动完成以下流程：

1. **自动搜索素材**：搜索书籍封面、相关场景图片
2. **智能生成文案**：根据书籍主题自动生成口播文案和字幕
3. **自动添加字幕**：生成SRT/ASS格式字幕文件
4. **视频自动合成**：整合所有素材生成完整视频

### 使用方式

```python
from one_click_video_generator import OneClickVideoGenerator

generator = OneClickVideoGenerator()
result = generator.generate_video(
    book_title="活着",
    author="余华",
    description="讲述了一个普通人坎坷的一生",
    quote="人是为活着本身而活着的",
    theme="生命的意义",
    template="1min",    # 可选：30s / 1min / 3min
    style="emotional"    # 可选：literary / practical / humorous / emotional
)
```

### 输出文件

| 文件类型 | 说明 |
|---------|------|
| `output.mp4` | 最终合成的视频文件 |
| `output.srt` | SRT格式字幕文件 |
| `output.ass` | ASS格式字幕文件（带样式） |
| `output_script.json` | 完整的视频脚本 |
| `video_assets/` | 下载的素材图片目录 |

## 核心能力

### 1. 视频脚本生成

#### 1.1 脚本结构设计
- **开场白设计**：吸引人的开场文案，包含问题、悬念或共鸣点
- **内容大纲**：按时间线规划视频内容，合理分配每本书的展示时间
- **转场衔接**：书与书之间的自然过渡文案
- **互动引导**：引导点赞、评论、关注的文案
- **结束语设计**：总结性文案，鼓励行动

#### 1.2 脚本格式
- **口播文案**：主播要说的话
- **画面提示**：对应的画面内容
- **字幕建议**：关键信息的字幕
- **背景音乐**：适合的BGM类型建议

### 2. 书单推荐文案

#### 2.1 推荐类型
- **单本深度推荐**：详细分析一本书的核心价值
- **主题书单推荐**：围绕某个主题推荐多本书
- **年度榜单推荐**：年度好书总结推荐
- **治愈系书单**：心灵慰藉类书籍推荐
- **成长类书单**：自我提升类书籍推荐

#### 2.2 文案风格
- **文艺清新风**：适合文学、治愈类书籍
- **干货讲书风**：适合实用、经管类书籍
- **幽默风趣风**：适合轻松、休闲类书籍
- **专业深度风**：适合学术、专业类书籍
- **情感共鸣风**：适合情感、励志类书籍

### 3. 视频素材建议

#### 3.1 画面设计
- **书籍封面展示**：推荐高质量的书籍封面图片
- **内页精彩片段**：展示书籍的精华内容
- **场景氛围图**：营造阅读氛围的图片
- **作者信息展示**：作者简介和背景
- **数据统计图**：销量、评分等数据可视化

#### 3.2 字幕设计
- **核心金句**：书中的精彩语录
- **关键信息**：书名、作者、出版社
- **重点强调**：突出显示的内容

#### 3.3 BGM推荐
- **节奏轻快型**：适合轻松、活力的内容
- **舒缓治愈型**：适合文学、治愈类内容
- **激昂励志型**：适合成长、励志类内容
- **古典优雅型**：适合经典、文学类内容

### 4. 视频结构模板

#### 4.1 30秒短模板
```
0-5秒：开场悬念/问题
5-15秒：第一本书推荐
15-25秒：第二本书推荐
25-30秒：互动引导
```

#### 4.2 1分钟模板
```
0-10秒：开场引入
10-25秒：第一本书深度推荐
25-40秒：第二本书深度推荐
40-55秒：总结升华
55-60秒：关注引导
```

#### 4.3 3分钟深度模板
```
0-20秒：主题介绍+开场
20-60秒：第一本书（内容+金句+感悟）
60-100秒：第二本书（内容+金句+感悟）
100-140秒：第三本书（内容+金句+感悟）
140-160秒：书单总结+行动建议
160-180秒：互动引导+关注
```

## 工作流程

1. **接收任务**：接收用户的书单或书籍列表
2. **分析书单**：分析书籍类型、主题、适合的视频风格
3. **生成脚本**：根据分析结果生成视频脚本
4. **准备素材**：提供视频制作所需的素材建议
5. **输出成果**：交付完整的视频制作方案

## 输出标准

### 1. 脚本完整
包含开场白、正文、转场、互动引导、结束语

### 2. 素材具体
提供具体的图片、音乐、字幕建议

### 3. 风格统一
视频风格与书籍主题相匹配

### 4. 可落地性
提供的素材和建议具有可操作性
