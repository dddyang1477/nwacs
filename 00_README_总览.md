## 🚀 快速开始

### 1. 阅读部署指南
打开 `21_TraeCN_部署指南.md`，按步骤在Trae CN中创建所有Skill。

### 2. 查看快速指南
打开 `22_快速开始指南.md`，了解常用指令。

### 3. 参考示例
打开 `24_示例项目_逆命剑仙.md`，了解系统如何协作产出。

### 4. 开始创作
对 **小说总调度官** 说：
> "我想写一部[类型]小说，题材是[题材]，风格要[风格]"

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🧠 **智能化** | 上下文感知、自适应输出、质量自检 |
| 🌐 **可联网** | 主动搜索市场趋势、专业知识、读者偏好 |
| 🧬 **有记忆** | 三级记忆体系（全局/Skill/类型） |
| 📈 **能进化** | 根据反馈自动迭代提示词和策略 |
| 🤝 **强协作** | 17个Skill智能协同，各司其职 |
| 🎯 **类型化** | 7大类型专家，深度定制 |

---

## 📋 系统要求

- **平台**：Trae CN（或支持Skills功能的AI平台）
- **功能依赖**：联网搜索、记忆功能
- **推荐配置**：为每个Skill独立配置，开启记忆和联网

## 📝 使用建议

1. **从简单开始**：先用一个类型测试完整流程
2. **逐步扩展**：熟悉后再添加自定义类型
3. **定期维护**：每月执行一次系统进化
4. **积极反馈**：用户的每次评价都是系统进化的养分
