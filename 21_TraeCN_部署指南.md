# 🚀 Trae CN 部署指南
## Novel Writing AI Collaborative System (NWACS)

---

## 一、系统概述

本系统是一套**一主多协**的小说写作AI协作框架，包含：
- **1个一级Skill**（小说总调度官）
- **9个二级Skill**（创作模块 + 监督模块）
- **7个三级Skill**（小说类型专家）

所有Skill均具备：**联网学习能力、记忆功能、持续进化能力**

---

## 二、Trae CN 部署步骤

### Step 1：创建一级 Skill

1. 打开 Trae CN → 点击左侧 **Skills** 图标
2. 点击 **"新建 Skill"**
3. 填写配置：
   - **名称**：`小说总调度官`
   - **描述**：`小说写作AI协作系统的核心大脑，负责任务拆解、资源调度、联网优化、记忆进化`
   - **图标**：选择 🎯 或自定义
4. 在 **System Prompt** 框中，粘贴 `02_一级Skill_小说总调度官.md` 的完整内容
5. 开启 **联网搜索** 和 **记忆** 功能
6. 保存

### Step 5：配置协作关系

在 Trae CN 中，通过以下方式建立协作：

1. **一级Skill调用二级Skill**：
   - 在对话中，一级Skill通过 `@` 提及调用其他Skill
   - 例如：`@世界观构造师 请根据以下需求构建世界观...`

2. **二级Skill间协作**：
   - 创作模块的Skill可以互相 `@` 调用
   - 例如：`@角色塑造师 请为以下剧情设计角色...`

3. **监督模块介入**：
   - 在创作完成后，一级Skill调用监督模块
   - 例如：`@去AI痕迹监督官 请审查以下文本...`

4. **三级Skill提供专业指导**：
   - 在需要类型化知识时，调用对应类型专家
   - 例如：`@玄幻仙侠专家 请提供修炼体系建议...`

## 三、使用流程

### 标准创作流程

```
用户提出需求
    ↓
@小说总调度官 接收需求
    ↓
总调度官拆解任务 → 激活对应三级Skill
    ↓
并行/串行激活二级Skill
    ↓
各Skill输出创作内容
    ↓
@去AI痕迹监督官 审查
    ↓
@质量审计师 评估
    ↓
总调度官汇总 → 输出最终成果
    ↓
记忆固化 → 系统进化
```

### 快速开始示例

**用户输入**：
```
我想写一部玄幻仙侠小说，主角是一个被宗门抛弃的废柴，
意外获得上古传承，最终逆袭成为一代宗师。
风格要热血激昂，字数预计100万字。
```

**系统响应流程**：
1. `@小说总调度官` 解析需求，推荐 `@玄幻仙侠专家`
2. 总调度官拆解任务：
   - `@世界观构造师` → 构建修仙世界
   - `@角色塑造师` → 设计主角及关键角色
   - `@剧情构造师` → 设计逆袭主线
3. 各Skill并行工作，输出专业内容
4. `@去AI痕迹监督官` 审查文笔自然度
5. `@质量审计师` 给出综合评分
6. 总调度官整合输出完整方案

## 四、进阶配置

### 自定义小说类型

如需添加新的三级Skill（如：末日生存、无限流、轻小说等）：

1. 复制 `12_三级Skill_小说类型基类.md`
2. 根据新类型的特点填充内容
3. 参考已有类型专家的结构
4. 在Trae CN中新建Skill并配置

### 调整质量阈值

在 `20_TraeCN_Skills配置.json` 中修改：
```json
"quality_threshold": 75  // 可调整为 70-90
```

### 记忆管理

- 每个Skill独立维护记忆
- 一级Skill维护全局记忆池
- 定期使用 `@小说总调度官 请整理记忆并优化系统` 触发进化

## 五、故障排查

| 问题 | 解决方案 |
|------|----------|
| Skill间信息不连贯 | 检查一级Skill的上下文传递是否完整 |
| 输出质量不稳定 | 调整质量审计师的评分标准 |
| AI痕迹明显 | 强化去AI痕迹监督官的审查力度 |
| 类型知识不足 | 让三级Skill执行联网搜索补充 |
| 记忆混乱 | 使用 `@小说总调度官 清理记忆` |

---

## 六、系统进化

系统具备自我进化能力：

1. **每次任务后**：质量审计师生成报告，总调度官分析
2. **每10次任务**：总调度官联网搜索最新趋势，更新提示词
3. **用户反馈**：用户可直接评价，系统学习偏好
4. **定期维护**：每月执行一次全面系统优化
