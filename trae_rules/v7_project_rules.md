# NWACS v7.0-MangaDrama 项目规则

## 系统身份
本项目使用 NWACS v7.0-MangaDrama 架构。
最终产出为「漫剧」（动态漫/漫画/短剧参考/小说）四栖内容。
小说正文是「源素材」，必须嵌入漫剧标记以供下游视觉/声音/工程层使用。

## 1. 漫剧标记语法（核心规范）
所有创作 Agent 输出正文时，必须在关键位置嵌入以下标记：

### 画面标记
- `[[PANEL:景别,时长]]` — 面板指令
  - 景别: 特写/近景/中景/全景/过肩/主观/鸟瞰/低角度/高角度
  - 时长: 数字+s（如 2s, 3.5s），省略则默认3秒
- `[[CAM:镜头运动]]` — 镜头指令
  - 推镜/拉镜/横摇左/横摇右/纵摇上/纵摇下/快速推拉/手持抖动/环绕/跟拍
- `[[TRANS:转场]]` — 转场指令
  - 硬切/淡入/淡出/闪白/黑场/划像/叠化/ match-cut
- `[[COLOR:色调]]` — 色彩指令
  - 冷青/暖黄/血红滤镜/黑白/高饱和/低饱和/柔焦/复古/赛博霓虹

### 声音标记
- `[[SFX:音效,参数]]` — 音效指令
  - 参数: 渐强/渐弱/瞬时/循环/长音/短促
  - 示例: `[[SFX:心跳声,渐强]]` `[[SFX:雷鸣,瞬时]]`
- `[[BGM:情绪,乐器]]` — BGM指令
  - 情绪: 紧张/温馨/悬疑/史诗/悲伤/欢快/诡异/浪漫
  - 乐器: 弦乐/钢琴/电子/管弦/鼓/民乐/合成器/无伴奏
  - 示例: `[[BGM:紧张,弦乐]]`

### 动态标记
- `[[MOTION:动态类型]]` — 动态层级（由 @动态标注师 细化）
  - 静态/微动/中动/强动/全动
- `[[FX:特效]]` — 视觉特效
  - 粒子/光晕/震动/残影/慢动作/时间静止/水墨/故障艺术

## 2. 文件目录规范
```
project/
├── source/                    # 带漫剧标记的源小说
│   └── chapter_XXX.md
├── visual/
│   ├── storyboards/           # 分镜 JSON
│   ├── characters/            # 角色视觉设定 JSON
│   └── scenes/                # 场景概念描述
├── audio/
│   ├── bgm_cues.json          # BGM 点位表
│   ├── sfx_cues.json          # 音效点位表
│   └── voice_scripts/         # 配音台本
├── drama/
│   ├── episodes/              # 单集工程数据
│   └── export/                # 剪映/PR/漫画导出
└── publish/
    ├── novel/                 # EPUB/干净正文
    ├── comic/                 # 漫画制作包
    └── short_drama/           # 动态漫/短剧制作包
```

## 3. 跨媒介一致性协议
- 角色视觉档案（`visual/characters/*.json`）是**唯一事实来源**
- 任何面板中出现角色外貌描写，必须通过 MCP `check_visual_consistency` 校验
- BGM 情绪曲线必须与视觉情绪报告中的「色彩脚本」同步
- 配音台本中的台词必须与源小说对话 100% 一致（禁止即兴发挥）

## 4. 平台适配规范
- **条漫（抖音/快手）**: 竖屏 9:16，每面板高度 ≤1200px，对话竖排，时长 1-3分钟
- **页漫（腾讯/快看）**: 方屏 1:1 或 4:3，每页 4-6 格，支持长按隐藏内容
- **动态漫（B站/优酷）**: 横屏 16:9，3-5分钟/集，支持简单镜头运动+循环动态
- **短剧参考**: 真人化分镜，1-2分钟/集，强调「黄金3秒」钩子

## 5. 质量红线
- 禁止无漫剧标记的小说正文进入分镜阶段
- 禁止同一角色在不同面板中视觉设定冲突
- 禁止 BGM 与人声频率冲突（工程层必须做 EQ 避让）
- 单集动态漫时长必须控制在 3-5 分钟（超出需拆分）
- 每集必须有至少1个「记忆点面板」（可做封面/切片/推荐位）
