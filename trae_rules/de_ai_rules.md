# 去AI化六诀 (De-AI Rules v3.0)

## 诀一：句式不均衡化
- 禁止连续使用"首先…其次…最后…"三段式结构
- 禁止连续3句以上结构相似度 > 80% 的排比
- 段落开头禁用逻辑连接词：首先/其次/最后/综上所述/因此/一言以蔽之/值得注意的是

## 诀二：逻辑连接弱化
- 减少"因此""所以""由于""鉴于"等因果连接词密度
- 用动作承接代替逻辑连接（"他摔了门。外面下雨了。"代替"因为他很生气，所以摔了门。"）
- 允许省略主语的口语化短句（艺术毛边），但每章 ≤2 处

## 诀三：给句子找主人
- 禁止无主句："确实令人困扰" → "这个问题确实让他感到困扰"
- 禁止空主语开头："毫无疑问/不难发现/众所周知/值得注意的是" → 删除或绑定具体主体
- 每个状态描述必须绑定具体的人、物或环境细节

## 诀四：内容血肉化
- 抽象名词必须落地："意义重大" → "这意味着他再也回不去了"
- 加入具体数据："很多人" → "三百多人"
- 加入实例支撑："他很厉害" → "去年他一个人堵住了东城门"
- 加入个人观点（通过角色之口）："这世道不公" → "他总觉得这世道欠他一条命"

## 诀五：长短句交错
- AI倾向长句均衡，人类写作长短错落
- 长句(>30字)后 70% 概率接短句(<15字)
- 紧张/动作场景：短句为主（平均8-12字）
- 抒情/反思场景：长句可增多，但需用短句破开

## 诀六：感官细节替代情绪标签
- 禁止直接情绪标签："他很愤怒/悲伤/兴奋/紧张"
- 必须转化为感官细节：
  - 愤怒 → 指节发白、呼吸变重、下颌绷紧、摔东西
  - 悲伤 → 盯着某物发呆、声音变轻、眨眼频率降低
  - 兴奋 → 语速加快、坐立不安、瞳孔放大
  - 紧张 → 吞咽口水、摩挲衣角、反复看门口
