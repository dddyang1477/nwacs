## 📞 Skill呼叫指令

### 一级Skill
```
@调度官 请拆解以下创作需求：...
@调度官 当前进度如何？下一步该做什么？
@调度官 请优化[某Skill]的提示词
```

### 二级Skill - 创作模块
```
@世界观师 基于[类型]构建世界观，要求：...
@角色师 设计主角，核心欲望是...，恐惧是...
@剧情师 基于以下世界观和角色，设计三幕式剧情...
@场景师 设计[场景名]，需要营造[氛围]
@对话师 设计[角色A]和[角色B]在[场景]的对话，目标是...
@战斗师 设计[战斗名]，双方是...，赌注是...
@写作大师 请为以下段落统一风格，基调是...
```

### 二级Skill - 监督模块
```
@去AI官 请审查以下段落，标记AI痕迹
@审计师 请对以下章节进行质量审计
```

### 三级Skill - 类型专家
```
@玄幻专家 请提供当前玄幻类型的热门套路
@言情专家 请设计情感发展曲线
@推理专家 请设计一个密室诡计
@科幻专家 请补充[科技]的设定细节
@历史专家 请验证[朝代]的[细节]是否准确
@恐怖专家 请设计一个心理恐惧场景
@游戏专家 请设计一场[游戏类型]的比赛
```

## ⚡ 紧急情况处理

### 创作卡壳
```
问题：想不出剧情
解决：@剧情师 请基于当前设定，提供3个不同的剧情走向
      @类型专家 请提供该类型的经典转折套路
```

### 角色扁平
```
问题：角色没有灵魂
解决：@角色师 请为[角色名]增加一个秘密/创伤/矛盾
      @对话师 请设计一段能展现[角色]性格的独白
```

### 战斗无聊
```
问题：战斗像流水账
解决：@战斗师 请增加环境互动/战术博弈/心理描写
      @场景师 请为战斗场景增加压迫感/空间限制
```

### AI痕迹重
```
问题：读起来像AI写的
解决：@去AI官 请全面审查
      @写作大师 请提供去AI化改写方案
```

### 质量不达标
```
问题：审计评分低于70
解决：@审计师 请指出最严重的前3个问题
      针对性修改后重新审计
```

## 📝 常用Prompt模板

### 启动创作
```
我想写一本[类型]小说，题材是[题材]，
风格要[风格]，预计[字数]。
主角：[简要描述]
核心冲突：[一句话]
特殊要求：[如有]
请按系统流程开始：先解析需求，再构建世界观。
```

### 续写章节
```
请继续创作第[X]章。
前文摘要：[粘贴上一章结尾]
当前角色状态：[粘贴角色当前状态]
本章目标：[需要完成什么剧情]
风格要求：[粘贴风格指南关键词]
```

### 修改优化
```
请修改以下段落：
[粘贴段落]

修改要求：
- 去AI化：增加不完美、具体化、个人偏见
- 风格统一：符合[基调]
- 增强[画面感/对话/节奏]
```

### 类型咨询
```
我是[类型]新手，请告诉我：
1. 该类型的核心爽点是什么？
2. 最常见的3个套路是什么？
3. 最容易踩的雷区是什么？
4. 最近有什么创新趋势？
```

## 🔢 质量评分速查

| 分数 | 等级 | 含义 | 行动 |
|------|------|------|------|
| 90-100 | S | 精品 | 微调即可发布 |
| 80-89 | A | 优秀 | 小修小补 |
| 70-79 | B | 合格 | 有明显问题需返工 |
| 60-69 | C | 不合格 | 需要大改 |
| <60 | D | 废品 | 建议重写 |

## 🎨 风格关键词速查

| 风格 | 句式特点 | 词汇特点 | 情感表达 |
|------|----------|----------|----------|
| 热血激昂 | 短句为主，排比有力 | 动词强烈，色彩浓烈 | 外放直接 |
| 冷峻克制 | 长短交替，留白多 | 名词具体，形容词少 | 内敛深沉 |
| 细腻婉约 | 长句为主，节奏舒缓 | 感官词丰富，比喻多 | 含蓄间接 |
| 幽默诙谐 | 反差句式，意外转折 | 口语化，网络用语 | 轻松调侃 |
| 暗黑沉重 | 破碎句式，压抑节奏 | 冷色调词，死亡意象 | 绝望压抑 |
| 悬疑紧张 | 短句急促，信息碎片化 | 模糊词，暗示性强 | 不安焦虑 |

## 🧩 去AI化检查清单（快速版）

- [ ] 没有"首先...其次...最后..."的过度结构化
- [ ] 没有"值得注意的是""需要指出的是"等AI高频词
- [ ] 没有过于工整的排比和对称
- [ ] 角色对话有个性差异，不是同一个人
- [ ] 有具体的感官细节，不是抽象描述
- [ ] 有角色的不理智、不体面时刻
- [ ] 有留白，没有解释一切
- [ ] 有口误、重复、半截话
- [ ] 有个人偏见和主观判断
- [ ] 段落长度不均匀，有呼吸感
