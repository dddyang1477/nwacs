## 快速开始

### 运行终极引擎

```bash
cd core/v8
python NWACS_ULTIMATE_ENGINE.py
```

### 系统要求

- Python 3.7+
- 无需额外依赖（使用标准库）
- 支持 UTF-8 编码

## 常见问题

### Q: 模块加载失败怎么办？
A: 系统支持降级加载，即使某些模块加载失败，其他功能仍然可以使用。

### Q: 如何保存配置？
A: 在主菜单选择【系统诊断与优化】 → 【保存当前配置】。

### Q: 可以更新内容吗？
A: 是的，定期运行系统优化引擎检查更新，或手动更新模块文件。

### Q: 支持哪些类型的小说？
A: 玄幻/仙侠、都市、悬疑、言情，共四种类型。
