## 一、AI漫剧创作核心条件

### 1.1 创作方向与题材选择

**优选题材（抖音/红果流量倾斜）：**
- 逆袭成长、古风玄幻、都市复仇
- 轻悬疑、家庭爽感、校园甜宠
- 仿真人漫剧、3D动画漫剧

**避雷题材：**
- 小众赌石、涉缅敏感内容
- 日常流水账、价值观擦边内容
- 动态解说漫、静态解说漫

**创作原则：**
- 低场景：单集场景控制在3个以内
- 强节奏：3秒一个小冲突、15秒一个反转
- 高一致：角色形象、画风全程统一

### 1.2 剧本核心规范

| 要素 | 规范要求 |
|------|---------|
| 字数 | 单集700-1000字 |
| 时长 | 1-3分钟（竖屏观剧） |
| 结构 | "3秒钩子、1分钟冲突、结尾留悬念" |
| 场景数 | 每集8-15个场景 |
| 台词长度 | 单句不超过10字 |

### 1.3 关键技术要求

**角色一致性解决方案：**
1. **参考图绑定法**（推荐新手）：上传角色高清参考图，调整权重参数
2. **LoRA训练法**（进阶）：训练专属角色模型
3. **首尾帧控制法**（专业）：用角色定妆照作为视频首帧

**分镜脚本结构：**
```json
{
 "episode": 1,
 "title": "初次相遇",
 "scenes": [
 {
 "scene_id": 1,
 "location": "咖啡店内",
 "characters": ["林小雨", "陈墨"],
 "action": "林小雨推门走进咖啡店",
 "dialogue": "请问，这里有人吗？",
 "camera": "中景，从门口推向座位",
 "mood": "温暖、好奇",
 "duration": 4
 }
 ]
}
```

## 二、AI漫剧制作完整流程

### 2.1 六大核心步骤

```
创意与剧本创作 → 角色与场景设计 → 分镜图像生成 → 图生视频 → 配音音效 → 后期剪辑
```

### 2.2 各环节工具推荐

| 环节 | 推荐工具 | 特点 |
|------|---------|------|
| **剧本生成** | 豆包、Kimi、DeepSeek | 生成结构化分镜脚本 |
| **角色设计** | 即梦AI、Midjourney | 角色一致性控制 |
| **分镜生成** | 即梦AI、ComfyUI | 提示词驱动画面 |
| **图生视频** | 即梦AI、可灵AI、Runway | 让画面动起来 |
| **配音** | 剪映AI、ElevenLabs | 角色语音合成 |
| **剪辑** | 剪映、CapCut | 转场、字幕、BGM |

### 2.3 分镜脚本字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| **scene_id** | 场景编号 | EP1-01 |
| **location** | 场景地点 | 咖啡店内，午后阳光 |
| **characters** | 出场角色 | ["林小雨", "陈墨"] |
| **action** | 动作描述 | 推门走进咖啡店 |
| **dialogue** | 台词 | "请问，这里有人吗？" |
| **camera** | 镜头语言 | 中景，从门口推向座位 |
| **mood** | 情绪氛围 | 温暖、好奇 |
| **duration** | 时长(秒) | 4 |

### 2.4 镜头语言指南

| 景别 | 用途 | 示例 |
|------|------|------|
| 远景 | 场景建立 | 城市全景 |
| 全景 | 人物与环境 | 主角站在屋顶 |
| 中景 | 人物互动 | 两人对话 |
| 近景 | 情绪表达 | 侧脸看向远方 |
| 特写 | 细节刻画 | 眼神特写 |

| 运镜 | 效果 | 适用场景 |
|------|------|---------|
| 推镜 | 情绪加强 | 聚焦角色表情 |
| 拉镜 | 展现环境 | 从人物到场景 |
| 平移 | 展示空间 | 横移展示场景 |
| 跟拍 | 跟随动作 | 角色移动 |

## 三、AI漫剧与小说创作的异同

### 3.1 相同点

| 方面 | 共同点 |
|------|---------|
| 故事核心 | 都需要好的故事、人物、情节 |
| 叙事逻辑 | 因果链驱动，有冲突、有起伏 |
| 情感共鸣 | 都需要打动观众/读者 |

### 3.2 差异点

| 方面 | 小说创作 | AI漫剧创作 |
|------|---------|-----------|
| **呈现方式** | 文字描述 | 视觉画面 |
| **节奏** | 可快可慢 | 必须紧凑（3秒钩子） |
| **场景限制** | 无限想象 | 每集3个场景以内 |
| **台词** | 可长可短 | 单句不超过10字 |
| **读者参与** | 想象空间大 | 画面直接呈现 |
| **制作成本** | 仅文字 | 需要AI工具生成画面 |

### 3.3 转换要点

**小说转漫剧关键步骤：**
1. **剧情精炼**：提取核心冲突，去除冗余描写
2. **场景拆分**：将长段落拆分为多个镜头
3. **动作可视化**：将心理描写转化为动作和表情
4. **节奏压缩**：符合"3秒钩子、1分钟冲突"原则

## 五、新增模块详解

### 5.1 comic_drama_module.py

**核心功能：**

1. **小说转剧本** (`novel_to_comic_script`)
   - 分析小说结构，提取情节点
   - 自动分割为指定集数
   - 转换为结构化漫剧脚本格式

2. **提示词生成** (`generate_prompts_for_ai_tools`)
   - 生成Midjourney/Stable Diffusion/即梦AI格式提示词
   - 确保角色一致性

3. **导出功能**
   - JSON格式（机器可读）
   - 可读脚本格式（人类可读）
   - 制作指南生成

**输出文件结构：**
```
comic_drama/
├── comic_script.json    # 结构化剧本
├── comic_script.txt     # 可读剧本
├── prompts.json         # AI绘图提示词
└── production_guide.md  # 制作指南
```

## 七、实战提示词模板

### 7.1 角色提示词公式
```
[发色] [年龄] [角色类型]，[服饰描述]，[表情]，[场景背景]，[画风风格]，柔光，高清，4K画质
```

**示例：**
```
黑发少年、古风白衣、冷脸、国漫分镜风格、柔光、高清、4K画质
```

### 7.2 分镜提示词公式
```
主体描述 + 环境细节 + 艺术风格 + 镜头语言 + 光影效果
```

**示例：**
```
女生，黑色长直发齐刘海白色连衣裙，站在傍晚的街头，低头落泪，二次元治愈漫剧风格，暖色调，高清
```

### 7.3 AI漫剧剧本生成提示词
```
生成1分钟AI漫剧剧本，题材:[反转搞笑/古风逆袭/校园甜宠]，强冲突开场，单集10个分镜，台词口语化，每镜标注:场景、人物、动作、表情、对话，结尾留钩子，场景控制在3个以内，避免长对白。
```
