## 🎯 推荐启动方式（按优先级）

### 方法1：使用简单启动脚本（首选）
双击 `简单启动NWACS.ps1`
这会使用Windows的py启动器，最可靠！

### 方法2：使用智能启动脚本
双击 `智能启动NWACS.ps1`
会自动查找Python位置

### 方法3：在VS Code中运行（最简单）
1. 在VS Code中打开 `NWACS_FINAL.py`
2. 按 **F5** 或者点击右上角的运行按钮

### 方法4：手动运行命令行
打开PowerShell或命令提示符，输入：
```bash
cd e:\Program Files (x86)\Trae CN\github\NWACS\core\v8
py NWACS_FINAL.py
```

## 🐍 Python查找说明

### 什么是py命令？
`py` 是Windows的Python启动器，可以自动找到系统中的Python版本，是最推荐的方式。

### 如果py命令也不行
请先确保你已安装Python：
- 从 https://www.python.org/downloads/ 下载安装
- 安装时记得勾选 "Add Python to PATH"
