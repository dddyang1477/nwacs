## 🎯 核心功能详解

### 1️⃣ NWACSUnifiedEngine 一体化引擎

```python
from nwacs_unified_engine import NWACSUnifiedEngine

# 初始化引擎
engine = NWACSUnifiedEngine()
```

**功能特点：**

#### ✨ 自动加载核心模块
引擎启动时会自动加载以下V8模块：
- bestseller_analyzer（爆款分析器）
- advanced_writing_techniques（高级写作技巧）
- skill_adapter（Skill适配器）
- professional_prompt_library（专业提示词库）
- sensory_enhancement_engine（感官增强引擎）
- vector_memory_system（向量记忆系统）

#### 🎯 内置技能注册
预注册了 **8个核心技能**：

| 技能名称 | 功能描述 | 适用场景 |
|---------|---------|---------|
| 📊 爆款分析 | 分析10万+爆款小说，提取写作公式 | 了解市场趋势 |
| 📱 公众号爆款 | 生成10万+公众号爆款文章 | 公众号写作 |
| 🔍 悬念设置 | 设置悬念，吸引读者持续阅读 | 悬疑、叙事设计 |
| 💥 反转设计 | 设计惊人反转，震撼读者 | 剧情转折 |
| 👥 群像叙事 | 多视角叙事，像《十日终焉》一样精彩 | 多角色故事 |
| 🛡️ 守护主题 | 打造《斩神》式守护主题 | 情感升华 |
| 🌍 世界观构建 | 构建《诡秘之主》式严谨世界观 | 玄幻、科幻设定 |
| 🎨 感官增强 | 六感描写，让读者身临其境 | 场景描写增强 |

#### 📋 内置工作流
预注册了 **3个完整写作工作流**：

| 工作流名称 | 步骤数 | 适用类型 |
|-----------|--------|---------|
| 玄幻小说创作 | 5步 | 玄幻修仙小说 |
| 悬疑小说创作 | 4步 | 悬疑推理小说 |
| 公众号爆文 | 4步 | 公众号10万+文章 |

### 2️⃣ SmartNovelGenerator 智能小说生成器

```python
from smart_novel_generator_enhanced import SmartNovelGenerator

# 初始化生成器
generator = SmartNovelGenerator()

# 交互模式
generator.interactive_writing_mode()

# 生成单章
generator.generate_chapter_with_skills(31)
```

**写作时的技能调用流程：**

```
生成章节第N章
  ↓
[技能1] 爆款分析 - 参考同类型爆款
  ↓
[技能2] 悬念设置 - 为章节设置钩子
  ↓
[技能3] 反转设计（每5章）- 设计重要反转
  ↓
[技能4] 感官增强 - 增强场景描写
  ↓
生成实际章节内容
```

### 💡 API 使用示例

#### 示例1: 列出所有技能
```python
from nwacs_unified_engine import NWACSUnifiedEngine
engine = NWACSUnifiedEngine()

skills = engine.list_skills()
for skill in skills:
    print(f"{skill['icon']} {skill['name']}")
```

#### 示例2: 执行单个技能
```python
# 执行爆款分析
result = engine.execute_skill("爆款分析", genre="玄幻")
print(result['result'])

# 执行公众号爆款写作
result = engine.execute_skill("公众号爆款", content_type="emotion")
print(result['result'])

# 执行悬念反转
result = engine.execute_skill("反转设计",
    setup="看似正常发展",
    misdirection="所有人都被误导",
    revelation="真相令人震惊！")
print(result['result'])
```

#### 示例3: 运行完整工作流
```python
# 运行玄幻小说创作工作流
result = engine.run_workflow("玄幻小说创作", genre="玄幻")

# 运行公众号爆文工作流
result = engine.run_workflow("公众号爆文")
```
