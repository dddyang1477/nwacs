# NWACS 自动学习系统使用指南

## 系统概述

NWACS 自动学习系统提供了完整的自动学习功能：

| 功能 | 脚本 | 说明 |
|------|------|------|
| **空闲自动学习** | `auto_idle_learning.py` | 检测电脑空闲，10分钟后自动启动联网学习 |
| **全能联网学习** | `nwacs_comprehensive_learning.py` | 联网学习玄幻小说创作技巧、收集词汇等 |
| **小说创作** | `generate_full_novel.py` | 自动生成完整小说 |

---

## 快速启动

### 方式一：批处理文件（推荐）

双击：
```
启动自动学习.bat
```

### 方式二：命令行

```
py auto_idle_learning.py
```

### 方式三：自定义参数

```
# 自定义空闲时间（5分钟）
py auto_idle_learning.py -t 300

# 自定义学习脚本
py auto_idle_learning.py -s nwacs_comprehensive_learning.py
```

---

## 空闲检测说明

| 设置 | 默认值 | 说明 |
|------|--------|------|
| 空闲阈值 | 600秒（10分钟） | 电脑空闲超过此时间触发学习 |
| 启动延迟 | 30秒 | 启动后先等待30秒再开始检测 |
| 学习间隔 | 3600秒（1小时） | 两次学习之间的最小间隔 |
| 检测频率 | 每10秒 | 每10秒检查一次空闲状态 |

## 问题排查

### 问题1：空闲检测不工作

**解决方案**：
1. 确保在Windows上运行
2. 运行测试脚本：
```bash
py test_idle_detection.py
```

### 问题2：脚本无法启动

**解决方案**：
1. 检查Python路径
2. 使用：
```
python 脚本名
```
或
```
py 脚本名
```

### 问题3：学习脚本失败

**解决方案**：
1. 检查config.json中的API key
2. 查看学习报告
