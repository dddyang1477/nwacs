# 文件整理计划

## 目标结构

```
NWACS/
├── docs/                    # 文档说明
│   ├── overview/           # 总览文档
│   ├── guides/             # 使用指南
│   ├── protocols/          # 协议文档
│   └── architecture/       # 架构文档
├── skills/                 # Skill文档
│   ├── level1/             # 一级Skill
│   ├── level2/             # 二级Skill
│   └── level3/             # 三级Skill
├── src/                    # Python源代码
│   ├── core/               # 核心模块
│   ├── mcp/                # MCP服务器相关
│   └── utils/              # 工具模块
├── tests/                  # 测试文件
├── config/                 # 配置文件
├── logs/                   # 日志文件
├── records/                # 学习记录
├── examples/               # 示例项目
│   └── xuanhuan/           # 玄幻修仙示例
└── output/                 # 输出文件（小说等）
```

## 文件归类清单

### docs/
- README文件
- 部署指南、配置指南
- 架构框架文档
- 协议文档

### skills/
- 一级Skill: 小说总调度官
- 二级Skill: 世界观构造师、剧情构造师等
- 三级Skill: 玄幻仙侠、都市言情等

### src/
- 核心模块: novel_writer.py, skill_executor.py等
- MCP模块: mcp_*.py
- 工具模块: logger.py, task_optimizer.py等

### tests/
- 所有test_*.py和test_*.bat文件

### config/
- 所有.json配置文件

### records/
- 学习记录目录

### examples/
- 玄幻修仙相关文件

### output/
- 生成的小说文件
