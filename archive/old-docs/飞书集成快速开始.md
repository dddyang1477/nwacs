## 🚀 5分钟快速配置

### 步骤1：创建飞书机器人（2分钟）

1. 打开飞书，**创建一个群聊**（或者用现有的群）
2. 群设置 → **群机器人** → **添加机器人**
3. 选择 **「自定义机器人」**
4. 设置机器人名字（比如 `NWACS助手`），头像可以用AI生成
5. 点击 **「添加」**
6. **最重要：复制Webhook地址！**（这个要粘贴到配置文件里）

### 步骤2：配置NWACS（2分钟）

1. 打开 `config/feishu_config.json`
2. 把刚才复制的Webhook地址填入 `webhook_url`
3. 保存文件

配置文件示例：
```json
{
  "enable": true,
  "webhook_url": "https://open.feishu.cn/open-apis/bot/v2/hook/你的-webhook-key",
  "secret": "",
  "push_events": [
    "task_start",
    "task_progress",
    "task_complete",
    "learning_complete",
    "quality_warning"
  ],
  "feature_flags": {
    "enable_progress_push": true,
    "enable_learning_report": true,
    "enable_markdown": true
  }
}
```

### 步骤3：测试连接（1分钟）

双击运行 `启动飞书集成.bat`

选择 `1` → 测试飞书连接

如果成功，你的飞书群会收到测试消息！🎉

## 🛠️ 调试说明

### 测试失败怎么办？
1. 检查Webhook URL是否正确
2. 检查网络连接
3. 检查飞书机器人是否在群里
4. 检查群机器人权限

### 没有收到消息？
- 确认Webhook正确
- 看控制台输出
- 测试模式下，即使没配置也会显示模拟发送
