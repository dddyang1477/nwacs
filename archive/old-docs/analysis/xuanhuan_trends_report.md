## 七、可操作写作建议

### 1. 开篇第一句话（如何抓住读者）
- **错误示范**：“我穿越了，成了废柴。”
- **正确示范**：“当我踏入战场时，我已经死了三百次。”

### 2. 人设卡设计（每个角色必须回答）
- 他/她最想要什么？
- 他/她最害怕什么？
- 他/她愿意为什么牺牲？
- 他/她最大的秘密是什么？

### 3. 章节结尾技巧（让读者停不下来）
- **信息钩子**：结尾抛出一个新问题（“突然，墙上的钟响了十三下”）
- **情感钩子**：结尾让读者愤怒/感动/震惊（“他转过身，看到了自己死去的父亲”）
- **悬念钩子**：结尾暗示某个角色要出事（“远方的城池，燃起了冲天大火”）

### 4. 世界观构建清单
- [ ] 世界的基本物理规则（如“没有重力”）
- [ ] 修炼体系的代价是什么
- [ ] 不同文明的冲突点在哪里
- [ ] 历史中隐藏的真相
- [ ] 主角在世界中的特殊地位

## 九、风险预警

1. **避免“设定盖过故事”**：2026年读者更关注角色和情感，世界观再新颖，如果角色立不住，依然会扑街
2. **警惕“过度反套路”**：反套路本身已经成为套路，真正的创新是“在传统框架内做颠覆”
3. **注意平台差异**：起点适合硬核设定，番茄适合快节奏爽文，七猫适合情感向，要根据平台调整写法
4. **控制更新节奏**：长篇玄幻的黄金更新频率是日更4000-6000字，过慢会流失读者，过快容易疲劳
