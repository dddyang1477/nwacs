# NWACS FINAL - 小说自动保存功能演示

## 🎉 新功能说明

### 功能特点
1. **自动创建文件夹** - 以小说名命名文件夹
2. **自动保存文件** - 生成3个核心文件
3. **智能文件管理** - 重名自动加时间戳
4. **中文文件名** - 支持中文小说名

---

## 🚀 使用流程

### 1. 运行程序
```bash
python NWACS_FINAL.py
```

### 2. 选择菜单选项
在主菜单中选择选项 18（快捷生成开局）

### 3. 按提示操作
- 输入小说名
- 选择类型
- 选择模板

### 4. 自动保存
程序会自动在 `novels/` 文件夹下创建以小说名命名的文件夹

---

## 📁 自动生成的文件

### 文件结构
```
core/v8/novels/
└── 你的小说名/
    ├── 小说信息.md        # 模板信息与核心要素
    ├── 第一章_开局.txt     # 开局示例
    └── 小说大纲.md        # 可编辑的大纲模板
```

### 1. 小说信息.md
包含内容：
- 小说基本信息
- 模板来源与爆款公式
- 核心要素列表
- 创建时间

### 2. 第一章_开局.txt
包含内容：
- 模板章节结构
- 完整开局示例
- 可直接编辑的内容

### 3. 小说大纲.md
包含可填写的模板：
- 作品基本信息
- 主角设定
- 开篇设定
- 章节大纲

---

## 💡 功能细节

### 文件夹命名
- 清理非法字符
- 支持中文字符
- 重名自动加时间戳

### 文件编码
- UTF-8 编码
- 完美支持中文

### 模板选择
- 4个大类（玄幻、都市、悬疑、言情）
- 每类10个爆款模板
- 基于2026年真实小说分析

---

## 🎯 示例演示

假设创建小说《修仙：我在宗门躺赢》，类型玄幻：

### 生成的文件
```
novels/
└── 修仙我在宗门躺赢/
    ├── 小说信息.md
    ├── 第一章_开局.txt
    └── 小说大纲.md
```

### 小说信息.md 示例
```markdown
# 修仙：我在宗门躺赢

**类型**: 玄幻/仙侠
**模板**: 家族流开局
**基于**: 《玄鉴仙族》- 季越人
**创建时间**: 2026-05-03 20:00:00

## 爆款公式
困境开局+群像展示+世界观暗示

## 核心要素
- 主角：残魂附身/传承获得
- 核心冲突：家族生存危机
- 金手指：神秘宝物/古镜/传承
- 世界观：残酷修仙世界
- 特色：群像塑造，每个角色有独立成长线
```
