## 🎯 创作者行动指南

### 1. 开局公式（3秒法则）
```
第一句话 → 制造冲突/危机
前300字 → 建立人设反差
章结尾 → 金句钩子
```

### 2. 人设公式
```
表面身份（普通/卑微）
+ 隐藏身份（强大/特殊）
= 人设反差 → 爽点
```

### 3. 平台适配
| 平台 | 侧重点 | 推荐类型 |
|------|-------|---------|
| 起点 | 精品化、长篇、系统严谨 | 玄幻、仙侠、悬疑、军旅 |
| 番茄 | 脑洞、短剧适配、情绪强 | 都市脑洞、重生逆袭、爽文 |
| 晋江 | 情感细腻、人设、纯爱 | 言情、纯爱、古言 |

### 4. 避坑指南
| ❌ 不要 | ✅ 要 |
|--------|------|
| 300字内没有冲突 | 开头就出事/制造危机 |
| 无脑开挂无代价成长 | 有代价的成长 |
| 配角都是工具人 | 每个角色有自己的故事线 |
| 慢热铺垫太久 | 每3章1个小爽点 |
| 反派弱智 | 反派有智商和动机 |
