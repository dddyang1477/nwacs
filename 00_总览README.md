## 🚀 使用方式

### 方式一：Trae CN 分Skill配置（推荐）

在 Trae CN 中，为每个 Skill 创建一个独立的 AI 助手，配置对应的提示词：

1. **创建主助手** → 粘贴 `02_一级Skill_小说总调度官.md` 内容
2. **创建创作助手** → 分别粘贴 `03~09` 的提示词
3. **创建监督助手** → 分别粘贴 `10~11` 的提示词
4. **创建类型助手** → 根据小说类型选择 `13~19` 中的一个粘贴

### 方式二：单助手模式（快速启动）

使用 `20_整合版_主入口提示词.md`，将所有能力整合到一个提示词中，直接复制到 Trae CN 使用。

### 方式三：工作流模式

按照以下流程协作：

```
用户提出需求
    ↓
【一级Skill】需求解析 → 任务拆解 → 激活类型Skill
    ↓
【三级Skill】输出类型指导
    ↓
【二级Skill-创作】并行工作：
    世界观构造师 → 角色塑造师 → 剧情构造师
         ↓              ↓           ↓
    场景构造师 ← 对话设计师 ← 战斗设计师
         ↓              ↓           ↓
    写作技巧大师（统一风格）
         ↓
【二级Skill-监督】串行审查：
    去AI痕迹监督官 → 质量审计师
         ↓
【一级Skill】汇总输出 → 记忆固化 → 学习进化
    ↓
最终作品交付用户
```

## 🔄 记忆与学习进化机制

### 记忆层级
1. **全局记忆**：用户偏好、项目设定、协作历史（一级Skill维护）
2. **Skill记忆**：各Skill的专业经验、成功案例（各Skill维护）
3. **类型记忆**：特定类型的套路与禁忌（三级Skill维护）

### 进化触发
- 用户反馈（好/不好）
- 质量审计低于阈值
- 联网发现新趋势
- 累计完成10个同类型任务

---

## 🌐 联网优化策略

各Skill内置联网指令，可在创作过程中：
- 搜索类型趋势与市场热点
- 补充专业知识（历史、科学、武术等）
- 分析爆款作品的叙事特征
- 获取最新的写作技巧与读者偏好

## ⚙️ 扩展指南

### 添加新小说类型
1. 复制 `12_三级Skill_小说类型基类.md`
2. 填充该类型的套路库、爽点地图、禁忌清单
3. 命名格式：`XX_三级Skill_[类型名].md`

### 添加新二级Skill
1. 参考现有二级Skill格式
2. 定义输入/输出接口
3. 在架构框架中更新协作流程图

### 自定义评估标准
修改 `11_二级Skill_质量审计师.md` 中的评估维度和权重。

---

## 📌 注意事项

1. **提示词长度**：部分提示词较长，Trae CN 可能有长度限制，建议分段粘贴或精简使用
2. **联网功能**：需要 Trae CN 支持联网搜索，否则联网指令部分需手动执行
3. **记忆功能**：建议配合 Trae CN 的记忆/上下文功能使用，或手动维护记忆文档
4. **协作效率**：多Skill协作需要人工调度，建议由用户扮演"一级Skill"的角色进行协调
